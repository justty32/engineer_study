# 核工（Nuclear Engineering）學習計畫

## 本系簡介與學習脈絡

核工（核子工程，Nuclear Engineering）研究的是：如何理解原子核層級的能量釋放與輻射，並把它安全、可控地用在發電、醫療、工業與科學上。它不是單一學科，而是把幾個工程支柱「縫」在一起：

- **物理面**：原子核怎麼分裂／融合、釋放多少能量、放出哪些粒子（中子、γ、β、α）。
- **中子面（反應器物理）**：把大量中子在材料裡的行為當成一個族群來描述，問「鏈鎖反應能不能維持？臨界條件是什麼？功率怎麼分佈？」這是核工最核心、也最獨特的部分。
- **熱流面（核熱流）**：分裂放出的能量是熱，必須靠流體把它帶走，否則燃料會熔毀。這牽涉熱傳、流體力學與兩相沸騰。
- **輻射與防護面**：輻射怎麼跟物質作用、怎麼偵測它、怎麼量化它對人的劑量、怎麼屏蔽防護。
- **系統與安全面**：把上面所有東西組成一座電廠，並回答「最壞情況會怎樣、怎麼防」。

### 給擁有者的脈絡提醒

擁有者背景是電機／通訊，數理工具（微積分、微分方程、線代、機率）熟，但物理（尤其熱力學、流體、近代物理）基礎不扎實。因此本計畫的安排原則是：

1. **先補近代物理與核物理的直覺**，因為整個領域建立在「原子核會放能、會放輻射」這件事上。
2. **反應器物理（中子學）是重頭戲**，它本質上是「中子的輸運與擴散方程」，數學形式上跟擁有者熟悉的偏微分方程、特徵值問題很像——可以用這個類比切入，但物理意義要講清楚。
3. **核熱流需要先有熱傳與流體的基本盤**，所以前面安排了工程熱流基礎。
4. 軟體／數值方法相關（蒙地卡羅、確定論數值解）可講得精簡，因為擁有者程式能力強。

### 深度分層

- **大學層級**：本計畫主體，完整涵蓋標準必修，從基礎講起。
- **碩士層級**：在各科末尾以「延伸」小節簡短介紹子領域。
- **博士層級**：以「研究方向列表」呈現，不展開長篇。

### 共通基礎說明

純數學（微積分、線性代數、工程數學／微分方程、機率與統計）與普通物理屬**共通基礎**，本計畫不另開科目重寫；在各科需要時只薄薄提一句它扮演的角色，必要時標註「詳見 共通基礎/」（該處可能尚未建立）。例如：中子擴散方程是一個含吸收項與源項的擴散型偏微分方程，求解工具屬共通基礎；蒙地卡羅抽樣的機率背景屬共通基礎。

## 各階段與科目清單

進度標記：⬜ 未完成 ／ ✅ 完成

### 階段 01：物理與工程基礎

打底用。先把「原子核會怎樣」與「能量／熱怎麼跑」這兩條主線的最基本觀念建立起來，後面所有東西都靠它。

- ⬜ [01-近代物理與量子概論](01-物理與工程基礎/01-近代物理與量子概論.md)
  範圍：黑體輻射、光電效應、波粒二象性、波函數與能階、原子結構與量子數，建立進入核物理所需的量子直覺。
- ⬜ [02-原子核物理](01-物理與工程基礎/02-原子核物理.md)
  範圍：核子組成、結合能與質量虧損、核力與核模型、放射性衰變（α/β/γ）與衰變律、核分裂與核融合的能量學。
- ⬜ [03-工程熱力學基礎](01-物理與工程基礎/03-工程熱力學基礎.md)
  範圍：熱力學第一、二定律、狀態量、蒸氣動力循環（朗肯循環）與卡諾效率，作為理解核電廠能量轉換的基礎。

### 階段 02：反應器物理與中子學

核工的核心心臟。把中子當成一個族群，用方程描述它能不能維持鏈鎖反應、功率怎麼分佈。

- ⬜ [04-中子與物質的核反應](02-反應器物理與中子學/04-中子與物質的核反應.md)
  範圍：中子截面（微觀／巨觀）、散射與吸收、共振、中子分群與能譜、反應率的物理意義與量綱。
- ⬜ [05-中子擴散理論](02-反應器物理與中子學/05-中子擴散理論.md)
  範圍：中子流與費克定律、單群擴散方程、邊界條件、無限與有限介質、擴散長度與平移臨界，為反應器計算建立解析骨架。
- ⬜ [06-反應器理論與臨界計算](02-反應器物理與中子學/06-反應器理論與臨界計算.md)
  範圍：增殖係數 $k$、四因子與六因子公式、臨界方程與幾何挫率、反射層、單群與多群臨界計算、功率分佈。
- ⬜ [07-反應器動力學與控制](02-反應器物理與中子學/07-反應器動力學與控制.md)
  範圍：緩發中子與其關鍵作用、點動力學方程、反應度與週期、反應度回饋與溫度係數、氙毒與燃耗、控制棒與反應器控制。

### 階段 03：核熱流與反應器系統

分裂能是熱，必須安全帶走。把熱傳、流體、兩相沸騰應用到燃料棒與反應器迴路。

- ⬜ [08-流體力學與熱傳基礎](03-核熱流與反應器系統/08-流體力學與熱傳基礎.md)
  範圍：流體靜力與動力、連續／動量／能量方程、層流與紊流、壓降、傳導／對流／輻射熱傳的工程估算。
- ⬜ [09-核反應器熱流分析](03-核熱流與反應器系統/09-核反應器熱流分析.md)
  範圍：燃料棒內溫度分佈、冷卻劑流道熱平衡、兩相流與沸騰熱傳、臨界熱通量（CHF/DNB）、熱限值與安全裕度。
- ⬜ [10-反應器型式與核能系統](03-核熱流與反應器系統/10-反應器型式與核能系統.md)
  範圍：PWR/BWR/PHWR/氣冷／快中子反應器與第四代爐型概觀、一次與二次迴路、蒸氣產生器、整體電廠系統與運轉概念。

### 階段 04：輻射與物質作用、偵測與防護

把「輻射」這條主線講透：它怎麼跟物質作用、怎麼量到它、怎麼換算成對人的傷害、怎麼防。

- ⬜ [11-輻射與物質的作用](04-輻射偵測與防護/11-輻射與物質的作用.md)
  範圍：帶電粒子的阻擋本領與射程、光子的光電／康普頓／成對效應與衰減係數、中子與物質作用，建立屏蔽與偵測的物理基礎。
- ⬜ [12-輻射偵測與量測](04-輻射偵測與防護/12-輻射偵測與量測.md)
  範圍：氣體偵測器（游離室／正比計數器／GM 管）、閃爍偵測器、半導體偵測器、能譜分析、計數統計與量測誤差。
- ⬜ [13-輻射劑量學與防護](04-輻射偵測與防護/13-輻射劑量學與防護.md)
  範圍：吸收劑量、等效劑量與有效劑量、劑量當量與權重因子、生物效應、ALARA、屏蔽計算、法規限值與輻射安全。

### 階段 05：核燃料、材料與化學

支撐反應器運轉的物質面：燃料從哪來、用完怎麼辦、材料在輻射與高溫下會怎樣。

- ⬜ [14-核燃料循環](05-核燃料材料與化學/14-核燃料循環.md)
  範圍：鈾礦開採與轉化、濃縮、燃料製造、爐內燃耗、用過燃料、再處理與最終處置，前端與後端整體流程及核擴散考量。
- ⬜ [15-核工材料](05-核燃料材料與化學/15-核工材料.md)
  範圍：燃料（UO₂）、護套（鋯合金）、結構材料與冷卻劑相容性、輻射損傷（位移、腫脹、脆化）、高溫腐蝕與材料壽命。
- ⬜ [16-放射化學與核化學](05-核燃料材料與化學/16-放射化學與核化學.md)
  範圍：放射性核種與衰變鏈、活度與比活度、放射性平衡、同位素分離與標誌、放射性廢料化學處理的基本原理。

### 階段 06：核能安全與工程應用

把所有東西組成電廠並回答「最壞情況怎麼防」，再帶到核能以外的應用與政策。

- ⬜ [17-核能安全與系統工程](06-核能安全與應用/17-核能安全與系統工程.md)
  範圍：縱深防禦與多重屏障、安全系統與圍阻體、設計基準事故與嚴重事故（如 LOCA、爐心熔毀）、機率風險評估（PRA）、歷史事故案例與監管。
- ⬜ [18-核工應用與輻射應用](06-核能安全與應用/18-核工應用與輻射應用.md)
  範圍：醫用輻射（放射治療、核醫造影）、工業射源與非破壞檢測、核融合與電漿入門、輻射加工，以及核能經濟與政策概觀。
