# 航太工程（Aerospace Engineering） 學習計畫

> 風格：概論重點式（每科一篇，從基本觀念把脈絡講清楚、給直覺，公式用 LaTeX 表達）。
> 語言：繁體中文。每科核心名詞收錄於 [中英名詞對照表](中英名詞對照表.md)。
> 重點：從流體力學、熱力學與固體力學基礎出發，往上堆出空氣動力學、推進、結構、飛行力學與控制，最後分流到大氣內飛行（飛機）與太空飛行（衛星／火箭）兩條主線。

## 本系簡介與學習脈絡

航太工程處理「物體如何在大氣中或太空中飛行」這件事。它本質上是一門高度整合的系統工程：一架飛機或一枚火箭，同時牽涉到流體（氣動）、熱與燃燒（推進）、固體與材料（結構）、運動與穩定（飛行力學）、感測與回授（航電與控制），以及在太空中還要加上重力場下的軌道運動與嚴苛的太空環境。

學習脈絡建議由下往上堆：

1. **共通基礎**（純數學與普通物理）先就位——這是所有後續課程的語言。
2. **工程力學與熱流基礎**：靜力、動力、材料力學、熱力學、流體力學。這些是航太四大支柱（氣動、推進、結構、飛行力學）的共同地基。
3. **氣動與推進**：流體進一步發展成空氣動力學與可壓縮氣體動力學，熱流發展成噴射與火箭推進。
4. **結構與材料**：固體力學發展成航太結構分析與輕量化材料。
5. **飛行力學、控制與軌道**：把前面的力與運動整合成「飛行器整體怎麼動、怎麼穩、怎麼被控制」，並延伸到太空中的軌道力學與太空環境。
6. **整合與系統**：航電、飛行器設計與專題實驗，把所有學科綁在一起（CDIO：構思—設計—實作—操作）。

對擁有者（電機／通訊背景、軟體強）的提醒：

- 航電、飛行控制中的「訊號、回授、狀態空間」概念與你熟悉的控制系統、通訊系統高度相通，這部分可以講得精簡、直接類比。
- 但「為什麼機翼會產生升力」「可壓縮流的震波」「結構的應力分布」「軌道為什麼是橢圓」這些屬於你較不熟的力學／流體領域，會從基本觀念與直覺講起。

### 深度分層

- **大學層級**：主要重點，完整涵蓋標準必修課綱，從基礎講起。
- **碩士層級**：各子領域（如黏性流 CFD、複合材料破壞力學、最佳控制、電推進、再入熱防護）只做簡短介紹。
- **博士層級**：以研究方向列表呈現，不寫長篇。

## 共通基礎（不在本系重寫）

以下屬「共通基礎」，本系筆記只在用到時薄薄帶一段「它在航太扮演的角色」，不整章重寫；詳見 [共通基礎/](../共通基礎/)。

- **微積分、線性代數、工程數學（常微分方程、偏微分方程、向量分析、複變）**：氣動方程、結構有限元、控制系統、軌道計算的共同語言。
- **機率與統計**：可靠度、量測誤差、結構安全係數。
- **普通物理（力學、熱、電磁）**：所有力學與熱流課程的物理出發點。

## 進度表

### 第一階段 — 工程力學與熱流基礎（01-工程力學與熱流基礎）

> 航太四大支柱的共同地基：把物體的受力、運動、變形、能量與流動講清楚。

| 課程 | 範圍說明 | 狀態 |
| :--- | :--- | :--- |
| **[01-工程靜力學與動力學](01-工程力學與熱流基礎/01-工程靜力學與動力學.md)** | 受力平衡、自由體圖、桁架與力矩；質點與剛體運動學、牛頓第二定律、功能與動量。 | ⬜ |
| **[02-材料力學](01-工程力學與熱流基礎/02-材料力學.md)** | 應力應變、軸向／扭轉／彎曲、樑的撓度、應力轉換與莫爾圓、挫屈。 | ⬜ |
| **[03-熱力學](01-工程力學與熱流基礎/03-熱力學.md)** | 系統與性質、第一與第二定律、熵、理想氣體、動力循環（布雷頓循環為推進核心）。 | ⬜ |
| **[04-流體力學](01-工程力學與熱流基礎/04-流體力學.md)** | 流體靜力、連續／動量／能量方程、白努利、黏性與雷諾數、邊界層概念。 | ⬜ |

### 第二階段 — 空氣動力學（02-空氣動力學）

> 把流體力學專門化到「空氣繞過飛行器」的問題：升力、阻力與震波從哪裡來。

| 課程 | 範圍說明 | 狀態 |
| :--- | :--- | :--- |
| **[05-空氣動力學（不可壓縮）](02-空氣動力學/05-空氣動力學（不可壓縮）.md)** | 翼型與機翼理論、環流與升力（Kutta-Joukowski）、薄翼理論、升力線理論、誘導阻力。 | ⬜ |
| **[06-可壓縮流與氣體動力學](02-空氣動力學/06-可壓縮流與氣體動力學.md)** | 馬赫數、等熵流、正震波與斜震波、膨脹波、噴嘴流、穿音速與超音速氣動。 | ⬜ |

### 第三階段 — 航太推進（03-航太推進）

> 飛行器靠什麼往前／往上：吸氣式噴射引擎與火箭。

| 課程 | 範圍說明 | 狀態 |
| :--- | :--- | :--- |
| **[07-噴射推進與燃氣渦輪](03-航太推進/07-噴射推進與燃氣渦輪.md)** | 推力方程、進氣／壓縮／燃燒／渦輪／噴嘴各部件、渦輪噴射與渦輪扇、比推力與耗油率。 | ⬜ |
| **[08-火箭推進](03-航太推進/08-火箭推進.md)** | 火箭方程（齊奧爾科夫斯基）、比衝、噴嘴設計與膨脹比、化學／固體／液體推進、電推進簡介。 | ⬜ |

### 第四階段 — 航太結構與材料（04-航太結構與材料）

> 飛行器又要輕又要承受大載荷：結構怎麼算、用什麼材料。

| 課程 | 範圍說明 | 狀態 |
| :--- | :--- | :--- |
| **[09-航太結構分析](04-航太結構與材料/09-航太結構分析.md)** | 薄壁結構、剪流與扭轉、肋與桁、能量法、有限元素法概念、結構振動與顫振入門。 | ⬜ |
| **[10-航太材料與複合材料](04-航太結構與材料/10-航太材料與複合材料.md)** | 鋁／鈦合金、複合材料疊層、疲勞與破壞、比強度與比剛度、選材取捨。 | ⬜ |

### 第五階段 — 飛行力學、控制與太空（05-飛行力學控制與太空）

> 把力與運動整合成飛行器整體行為：性能、穩定、控制，並延伸到太空軌道與環境。

| 課程 | 範圍說明 | 狀態 |
| :--- | :--- | :--- |
| **[11-飛行力學與性能](05-飛行力學控制與太空/11-飛行力學與性能.md)** | 力平衡、爬升／巡航／滑翔、航程與續航、起降性能、飛行包絡線、轉彎與負荷因數。 | ⬜ |
| **[12-飛行器穩定與控制](05-飛行力學控制與太空/12-飛行器穩定與控制.md)** | 縱向與橫向穩定、操縱面、運動方程線性化、短週期／長週期模態、自動駕駛概念。 | ⬜ |
| **[13-自動控制系統](05-飛行力學控制與太空/13-自動控制系統.md)** | 回授控制、轉移函數、根軌跡、頻率響應、狀態空間（與電機控制相通，從航太應用切入）。 | ⬜ |
| **[14-軌道力學](05-飛行力學控制與太空/14-軌道力學.md)** | 二體問題、克卜勒軌道要素、霍曼轉移、軌道攝動、發射與再入、星際轉移簡介。 | ⬜ |
| **[15-太空環境與太空飛行器](05-飛行力學控制與太空/15-太空環境與太空飛行器.md)** | 真空、輻射、原子氧、熱控、姿態判定與控制、太空船子系統概觀、再入熱防護。 | ⬜ |

### 第六階段 — 整合與系統（06-整合與系統）

> 把所有學科綁在一起：感測與航電、整機設計與專題。

| 課程 | 範圍說明 | 狀態 |
| :--- | :--- | :--- |
| **[16-航電與導航](06-整合與系統/16-航電與導航.md)** | 慣性導航（IMU）、GNSS、感測器融合、飛控電腦、通訊與資料匯流排（與電機／通訊背景高度相通）。 | ⬜ |
| **[17-航太系統設計與專題](06-整合與系統/17-航太系統設計與專題.md)** | 概念設計流程、需求與權衡、重量與配置、多領域整合（CDIO）、飛機／太空船設計案例。 | ⬜ |
