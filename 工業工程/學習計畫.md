# 工業工程與管理 學習計畫（Industrial Engineering and Management）

## 本系簡介與學習脈絡

工業工程（Industrial Engineering, IE）是「把系統做得更好」的工程：它的研究對象不是單一機械或電路，而是由**人、機器、物料、資訊、資金**共同組成的**生產與服務系統**。傳統工程問「這個東西能不能做出來、能不能運作」，工業工程則接著問「如何用最少的資源、最低的成本、最短的時間、最穩定的品質，把它做出來並持續運作」。因此 IE 同時橫跨**工程**與**管理**兩端，常被概括為四個支柱（交大稱之為 M-COS）：

- **M（Management，管理）**：生產管理、品質管理、供應鏈、工程經濟。
- **C（Computer，電腦／資訊）**：資料分析、系統模擬、製造資訊系統、自動化。
- **O（Operations Research，作業研究）**：以數學最佳化（線性規劃、整數規劃、網路流、排程）做決策。
- **S（Statistics，統計）**：工程統計、品質管制、實驗設計、可靠度。

對擁有者（電機／通訊背景、軟體強）的學習脈絡建議：

- 你最熟悉的「最佳化、機率、模擬、演算法」其實正是 IE 的數學核心（作業研究、排隊論、模擬），這部分可以學得很快，重點放在**問題建模**而非工具實作。
- 你比較陌生的會是**管理面與實體面**：生產排程的實務邏輯、品質管制的統計慣例、人因工程的人體與認知限制、設施規劃與物料搬運的空間思維、供應鏈的庫存與長鞭效應。這些章節會從基本觀念、直覺與典型決策講起。
- IE 的精神是**量化決策**：幾乎每個主題最後都會落到「定義目標函數與限制式，再求解或評估」。學習時請刻意把每個情境翻譯成數學問題。

**深度分層**：大學層級為主，完整涵蓋標準必修；碩士層級於各科末尾「進階方向」簡述；博士層級僅以研究方向列表呈現。

**共通基礎**：微積分、線性代數、工程數學、機率與統計、普通物理屬於跨領域共通基礎，本計畫不重寫；遇到時只說明它在 IE 中扮演的角色（例如線性代數是線性規劃的骨架、機率是排隊與品質的語言），必要時標註「詳見 共通基礎/」。其中**機率與統計**對 IE 特別關鍵，本計畫的「工程統計」一科會把 IE 真正會用到的統計觀念重新組織、銜接共通基礎。

## 學習階段與科目清單

進度標記：⬜ 未完成　✅ 已完成

### 01-基礎與導論
建立對「系統觀」與「量化決策」的基本認識，並補上 IE 特有的工程經濟語言。

- ⬜ [01-工業工程概論](01-基礎與導論/01-工業工程概論.md)　範圍：IE 的定義、歷史（泰勒科學管理到現代）、與管理科學的關係、IE 工程師的工作面向與四大支柱全貌。
- ⬜ [02-工程經濟](01-基礎與導論/02-工程經濟.md)　範圍：金錢的時間價值、現值/年金/利率、方案評估（NPV、IRR、回收期）、折舊與稅、汰換分析與敏感度分析。

### 02-機率統計與作業研究核心
IE 的兩大數學引擎：用統計描述不確定、用最佳化做決策。

- ⬜ [03-工程統計](02-機率統計與作業研究核心/03-工程統計.md)　範圍：機率分布回顧、抽樣與估計、信賴區間、假設檢定、迴歸與變異數分析在工程資料上的應用（銜接共通基礎機率統計）。
- ⬜ [04-作業研究：確定性模型](02-機率統計與作業研究核心/04-作業研究-確定性模型.md)　範圍：線性規劃建模與單體法、對偶與敏感度分析、整數規劃、運輸與指派、網路最佳化、動態規劃概念。
- ⬜ [05-作業研究：隨機模型](02-機率統計與作業研究核心/05-作業研究-隨機模型.md)　範圍：隨機過程基礎、卜瓦松過程、馬可夫鏈、排隊理論（M/M/1 等）、存貨隨機模型與決策分析。
- ⬜ [06-系統模擬](02-機率統計與作業研究核心/06-系統模擬.md)　範圍：離散事件模擬原理、亂數與分布抽樣、模型驗證與確認、輸出分析與變異縮減；對軟體實作可精簡，重點在建模與統計解讀。

### 03-品質與工作研究
把統計工具落實到品質保證與作業改善，是 IE 最具代表性的傳統強項。

- ⬜ [07-品質管制與工程統計應用](03-品質與工作研究/07-品質管制.md)　範圍：統計製程管制（SPC）、管制圖（X-bar/R、p、c 等）、製程能力指標（Cp、Cpk）、驗收抽樣、六標準差與 DMAIC、TQM 概念。
- ⬜ [08-實驗設計](03-品質與工作研究/08-實驗設計.md)　範圍：單因子與多因子 ANOVA、區集與隨機化、全因子與部分因子設計、田口方法、反應曲面法，與品質改善的連結。
- ⬜ [09-工作研究與方法工程](03-品質與工作研究/09-工作研究與方法工程.md)　範圍：方法研究、動作經濟原則、時間研究與碼錶法、預定時間標準（MTM/MOST）、工作抽樣、標準工時與生產力衡量。

### 04-人因與設施
讓系統適配人、並把廠房空間與物料流動規劃好。

- ⬜ [10-人因工程與工效學](04-人因與設施/10-人因工程.md)　範圍：人機系統、人體測計與生理工作負荷、顯示與控制設計、認知人因與人為失誤、職業安全與作業環境（噪音/照明/姿勢/抬舉）。
- ⬜ [11-設施規劃與物料搬運](04-人因與設施/11-設施規劃與物料搬運.md)　範圍：設施需求與選址、廠房佈置（製程別/產品別/群組）、SLP 系統化佈置規劃、物料搬運系統設計、流程分析與線平衡。

### 05-製造與生產系統
從製程本身到整廠自動化與資訊系統。

- ⬜ [12-製造程序概論](05-製造與生產系統/12-製造程序概論.md)　範圍：主要製造程序（鑄造、成形、切削加工、銲接、塑膠成形、積層製造）的原理、能力與成本觀念，供 IE 做製程選擇與成本估算。
- ⬜ [13-製造系統與自動化](05-製造與生產系統/13-製造系統與自動化.md)　範圍：數值控制（NC/CNC）、群組技術與彈性製造系統（FMS）、工業機器人、可程式控制器（PLC）概念、自動化導入決策、CIM 與工業 4.0 概念。
- ⬜ [14-生產與作業管理](05-製造與生產系統/14-生產與作業管理.md)　範圍：需求預測、總體規劃、主生產排程（MPS）、物料需求規劃（MRP）、產能規劃、生產排程與派工、精實生產與 JIT、TOC。

### 06-營運、供應鏈與整合
把單廠的決策放大到跨企業的供應網路，並收尾資訊與專題能力。

- ⬜ [15-存貨管理](06-營運供應鏈與整合/15-存貨管理.md)　範圍：EOQ 與其延伸、再訂購點與安全存量、服務水準、ABC 分析、報童模型、(s,S) 等補貨政策。
- ⬜ [16-供應鏈與物流管理](06-營運供應鏈與整合/16-供應鏈與物流管理.md)　範圍：供應鏈結構與決策層級、長鞭效應、配銷與運輸網路、採購與供應商管理、SCOR 與績效指標、風險與韌性。
- ⬜ [17-製造資訊系統與資料分析](06-營運供應鏈與整合/17-製造資訊系統與資料分析.md)　範圍：ERP/MES 概念、生產資料蒐集、IE 中的資料分析與最佳化應用；因擁有者軟體強，本科從精，聚焦資料如何支援決策。
- ⬜ [18-專案管理](06-營運供應鏈與整合/18-專案管理.md)　範圍：工作分解結構（WBS）、CPM 與 PERT 網路、要徑與時程、資源配置與趕工取捨、成本控制與實獲值管理（EVM）。
