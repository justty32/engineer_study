# A2 微波工程（Microwave Engineering）概論

> 天線特化第 2 科。
> 名詞對照見 [中英名詞對照表 → 天線](../中英名詞對照表-4.md#天線)。

## 這科在學什麼

頻率高到「波長與電路尺寸相當」時，集總元件（lumped R/L/C）的近似失效，電壓 / 電流隨位置改變，必須用**分布參數**與**場 / 波**的觀點分析。微波工程處理約 **300 MHz – 300 GHz**（波長 1 m – 1 mm）的電路與系統：

1. 微波網路與 **S 參數**
2. 阻抗匹配與 Smith 圖
3. 共振器、濾波器
4. 被動元件（耦合器、功率分配 / 合成、隔離器）
5. 主動元件（放大器、混頻器、振盪器）
6. 系統指標（雜訊、線性度）

> 傳輸線 / 波導的場與模態理論放在 [A3](A3-傳輸線與波導.md)，本科把它們當「已知元件」使用。

## 第 1 章　為什麼微波要用「波」的觀點

### 1.1 電氣長度
判準是**電氣長度** $\ell/\lambda$：
- $\ell \ll \lambda$：集總近似成立（一般電路學）。
- $\ell \sim \lambda$：相位沿線變化不可忽略 → 分布參數 / 傳輸線。

1 GHz 在自由空間波長 30 cm；PCB 上更短。幾公分的走線就要當傳輸線看。

### 1.2 從電路到場
- 低頻：用 $V$、$I$、KVL / KCL。
- 微波：用 $\mathbf{E}$、$\mathbf{H}$、馬克士威方程；但仍盡量化為「等效電壓 / 電流 / 阻抗」以沿用電路直覺。

## 第 2 章　傳輸線基本量（A3 詳述，這裡夠用）

- **特性阻抗** $Z_0$：行波電壓 / 電流比；常見 50 Ω（功率 / 雜訊折衷）、75 Ω（影音、低損耗）。
- **傳播常數** $\gamma=\alpha+j\beta$：$\alpha$ 衰減、$\beta=2\pi/\lambda$ 相位。
- **反射係數**：
$$\Gamma=\dfrac{Z_L-Z_0}{Z_L+Z_0}$$
- **駐波比** $\text{VSWR}=\dfrac{1+|\Gamma|}{1-|\Gamma|}$。
- **輸入阻抗**（無損耗線，長 $\ell$）：
$$Z_\text{in}=Z_0\dfrac{Z_L+jZ_0\tan\beta\ell}{Z_0+jZ_L\tan\beta\ell}$$
  - $\lambda/4$ 線：$Z_\text{in}=Z_0^2/Z_L$（阻抗轉換器）。
  - $\lambda/2$ 線：$Z_\text{in}=Z_L$（重現）。

## 第 3 章　Smith 圖（Smith Chart）

把複數反射係數平面 $|\Gamma|\le 1$ 上畫**等電阻圓 / 等電抗弧**的歸一化阻抗圖。
- 順時針繞圓 = 往電源方向移動（電氣長度）。
- 阻抗 ↔ 導納：旋轉 180°。
- 用途：可視化匹配、設計 stub / L-match、讀 VSWR、由量測 $\Gamma$ 反推負載。
- 現代雖用 CAD（ADS、Microwave Office），但 Smith 圖仍是工程師的「心智模型」。

## 第 4 章　微波網路與 S 參數

### 4.1 為什麼不用 Z / Y 參數
高頻**開路 / 短路難實現**（寄生效應），而且器件可能不穩。S 參數用**匹配負載端接 + 入射 / 反射波**定義，量測穩定、物理直觀。

### 4.2 散射矩陣
以歸一化入射波 $a_i$、反射波 $b_i$：
$$b_i=\sum_j S_{ij}a_j$$
- $S_{11}$：埠 1 輸入反射（return loss、匹配）。
- $S_{21}$：埠 1→2 傳輸（增益 / 插入損耗）。
- $S_{12}$：反向傳輸（隔離 / 回饋）。
- $S_{22}$：埠 2 輸出反射。

常用 dB：$\text{RL}=-20\log|S_{11}|$、$\text{IL}=-20\log|S_{21}|$。

### 4.3 網路性質
- **互易（reciprocal）**：$S=S^\top$（無鐵氧體 / 主動元件時成立）。
- **無損耗**：$S$ 為么正矩陣（$S^\dagger S=I$）。
- **對稱**。

### 4.4 其他矩陣與用途
- **ABCD 矩陣**：串接網路相乘，方便級聯分析。
- **T（傳輸）參數**：也用於級聯。
- S ↔ ABCD ↔ Z/Y 可互轉。

## 第 5 章　阻抗匹配

目標：把負載變到 $Z_0$，使反射最小、功率轉移最大、保護主動元件。

### 5.1 集總匹配
- **L 網路**：兩個電抗元件，匹配單頻；頻寬有限。
- **π / T 網路**：多一自由度，可控 Q / 頻寬。

### 5.2 分布匹配
- **單 / 雙短截線（stub）**：開路 / 短路 stub 並聯，調長度與位置。
- **$\lambda/4$ 轉換器**：匹配實阻抗；多段 $\lambda/4$（二項式 / 切比雪夫）做寬頻。
- **漸變線（taper）**：連續阻抗變化做超寬頻。

### 5.3 頻寬與 Bode–Fano 極限
任何被動匹配對某負載都有「匹配品質 × 頻寬」的理論上限（Bode–Fano）。不能既完美又寬頻。

## 第 6 章　共振器與濾波器

### 6.1 共振器
- 集總 LC、傳輸線（$\lambda/4$、$\lambda/2$）、介質共振器、腔體（cavity）。
- 品質因數 $Q$：儲能 / 每週期損耗。腔體 $Q$ 可達上萬。

### 6.2 濾波器設計流程
1. 選響應：Butterworth（平坦）、Chebyshev（等漣波、陡）、橢圓（最陡、有零點）、Bessel（線性相位）。
2. 由**低通原型**（歸一化 g 值）做頻率 / 阻抗縮放。
3. 低通 → 高通 / 帶通 / 帶阻轉換。
4. 實作：耦合線、stub、SIR、腔體、SAW / BAW（射頻前端）。

### 6.3 實作技術
- 微帶耦合線濾波器、interdigital、combline。
- 腔體濾波器（基地台高功率、低損耗）。
- 介質 / 陶瓷濾波器（手機小型化）。
- **BAW / FBAR、SAW**：手機射頻前端的主力（多頻共存、抗干擾）。

## 第 7 章　被動元件

- **功率分配 / 合成器**：Wilkinson（等分、埠間隔離）、T 接點。
- **定向耦合器**：取樣部分功率，量測 / 回授；參數：耦合度、方向性、隔離度。
- **混成接頭（hybrid）**：90°（branch-line）、180°（rat-race / magic-T）。
- **隔離器 / 環行器（isolator / circulator）**：鐵氧體非互易元件，保護源、收發共用天線。
- **衰減器、相移器、開關**：相位陣列與量測系統的關鍵（PIN diode、MEMS、半導體）。

## 第 8 章　主動元件

### 8.1 放大器
- **低雜訊放大器（LNA）**：接收前端第一級，決定整體雜訊指數（Friis 雜訊公式）。
- **功率放大器（PA）**：發射端，重效率與線性度；類別 A/AB/B/C/D/E/F、Doherty（基地台高效）。
- 設計：用 S 參數做**穩定性（K-Δ 判據）**、增益圓、雜訊圓、負載牽引（load-pull）找最佳匹配。

### 8.2 混頻器（Mixer）
利用非線性把訊號搬頻：$f_\text{IF}=|f_\text{RF}\pm f_\text{LO}|$。
- 指標：轉換增益 / 損耗、隔離、線性度（IP3）、鏡像抑制。
- 結構：單平衡、雙平衡、Gilbert cell、image-reject。

### 8.3 振盪器（Oscillator）
產生載波 / 本振。
- 條件：迴路增益 ≥ 1 且相位 0（Barkhausen）/ 負阻。
- 指標：頻率穩定度、**相位雜訊**（決定通訊鄰道、雷達雜波）。
- VCO + PLL 合成所需頻率。

## 第 9 章　系統層級指標

- **雜訊指數 NF**：訊號通過後 SNR 劣化；級聯用 Friis 公式（前級增益越高、後級雜訊影響越小）。
- **線性度**：1 dB 壓縮點（P1dB）、三階交越點（IP3）；決定大訊號 / 互調失真。
- **動態範圍（SFDR）**：可用訊號最大 / 最小的範圍。
- **效率**：PA 的 PAE 影響功耗與散熱。

## 第 10 章　微波技術與封裝

- **傳輸介質**：微帶、帶狀線、共面波導（CPW）、波導、SIW（基板整合波導）。
- **製程**：PCB、LTCC、GaAs / GaN / SiGe MMIC、CMOS RFIC。
- **mmWave / 5G / 6G**：相位陣列模組、封裝天線（AiP）。
- **量測**：VNA（S 參數）、頻譜儀、雜訊指數儀、訊號源、相位雜訊量測。
