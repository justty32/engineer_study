# A4 雷達系統與射頻前端（Radar Systems & RF Front-Ends）概論

> 天線特化第 4 科。
> 名詞對照見 [中英名詞對照表 → 天線](../中英名詞對照表-4.md#天線)。

## 這科在學什麼

把前三科（天線、微波元件、傳輸線）組裝成**完整的射頻收發系統**，並以**雷達**為旗艦應用。內容分兩半：

1. **射頻前端（RF Front-End）**：從天線到基頻之間的鏈路——LNA、混頻、濾波、PA、頻率合成、收發機架構、鏈路與雜訊預算。
2. **雷達系統**：用電磁波偵測目標的距離、速度、角度——雷達方程、波形、脈衝 / FMCW、都卜勒、相位陣列、訊號處理。

## 第一部分：射頻前端

## 第 1 章　收發機架構

### 1.1 超外差（Superheterodyne）
RF → （LNA）→ 混頻到 **IF（中頻）** → IF 濾波 / 放大 → 再降頻到基頻。
- 優點：選擇性好、增益分散、鏡像可濾。
- 缺點：需鏡像抑制濾波器、零件多。

### 1.2 直接轉換（Zero-IF / Homodyne）
RF 直接混到 DC（$f_\text{LO}=f_\text{RF}$）。
- 優點：省去 IF、易整合（手機 RFIC 主流）。
- 缺點：DC offset、I/Q 不平衡、閃爍雜訊、LO 洩漏。

### 1.3 低中頻、數位中頻、直接取樣
- **Low-IF**：避開 DC 問題又易整合。
- **直接 RF 取樣**：高速 ADC 直接取樣 RF（軟體定義無線電 SDR、現代雷達），把更多工作丟給數位。

### 1.4 I/Q 架構
用兩路正交（0° / 90°）本振做複數降頻，保留完整振幅 + 相位資訊，支援任意數位調變解調。

## 第 2 章　射頻鏈路指標（複習 A2 + 通訊）

### 2.1 雜訊指數與級聯
Friis 級聯雜訊：
$$F_\text{total}=F_1+\dfrac{F_2-1}{G_1}+\dfrac{F_3-1}{G_1G_2}+\cdots$$
- 第一級（LNA）增益高、雜訊低 → 主導整體雜訊。
- 接收靈敏度 $\approx -174\,\text{dBm/Hz}+10\log B+\text{NF}+\text{SNR}_\text{min}$。

### 2.2 線性度
- **P1dB、IP3**：大訊號失真、互調。
- **阻塞（blocker）**：強鄰道訊號使前端飽和 → 需高線性度 + 濾波。

### 2.3 動態範圍
最大可處理訊號（線性度上限）到最小可偵測訊號（雜訊下限）的範圍。

### 2.4 增益控制
AGC、可變增益放大器（VGA）讓接收機在強弱訊號間維持 ADC 滿刻度。

## 第 3 章　頻率合成（Frequency Synthesis）

### 3.1 鎖相迴路（PLL）
參考晶振 + 相位偵測器 + 迴路濾波器 + VCO + 除頻器，鎖定到 $N\times f_\text{ref}$。
- **整數 N**、**分數 N（fractional-N）**：細頻率解析。
- 指標：鎖定時間、相位雜訊、雜散（spur）。

### 3.2 相位雜訊的影響
- 通訊：鄰道干擾、EVM 劣化。
- 雷達：限制慢速目標偵測（雜波旁的微弱都卜勒被相位雜訊淹沒）。

### 3.3 直接數位合成（DDS）
數位累加相位 + 查表 + DAC 產生波形；快速跳頻、精準波形（FMCW chirp 線性度佳）。

## 第二部分：雷達系統

## 第 4 章　雷達基本原理

### 4.1 偵測什麼
- **距離 R**：回波延遲 $t=2R/c$。
- **徑向速度 v**：都卜勒頻移 $f_d=\dfrac{2v}{\lambda}$。
- **角度（方位 / 仰角）**：天線指向 / 陣列波束 / 單脈衝。
- **雷達散射截面（RCS, $\sigma$）**：目標「電磁可見度」。

### 4.2 雷達方程
$$P_r=\dfrac{P_t G_t G_r \lambda^2 \sigma}{(4\pi)^3 R^4 L}$$
- 回波功率 $\propto R^{-4}$（往返各一次 $R^{-2}$）。
- 最大偵測距離由最小可偵測訊號決定。
- $L$：系統損耗。

### 4.3 雷達頻段
- HF / VHF / UHF（超視距、穿透）。
- L / S（搜索、空管）。
- C / X（多功能、氣象、成像）。
- Ku / Ka / W（汽車雷達 77 GHz、高解析成像）。

## 第 5 章　雷達波形與量測

### 5.1 脈衝雷達（Pulsed）
發射短脈衝、收聽回波。
- **距離解析度** $\Delta R=\dfrac{c\tau}{2}=\dfrac{c}{2B}$（脈寬越短 / 頻寬越大越精細）。
- **最大不模糊距離** $R_\text{max}=\dfrac{c}{2\,\text{PRF}}$（脈衝重複頻率 PRF 越低越遠）。
- **脈衝壓縮**：發射長脈衝（高能量）+ 線性調頻（chirp）/ 相位編碼，接收端匹配濾波 → 兼得能量與解析度。

### 5.2 FMCW（連續波調頻）
連續發射線性掃頻；回波與發射混頻得**拍頻**，拍頻 ∝ 距離。
- 汽車雷達、近距高解析、低峰值功率。
- 距離 + 都卜勒可由 2D FFT（距離-都卜勒圖）同時取得。

### 5.3 都卜勒處理與 MTI / MTD
- **MTI（動目標顯示）**：濾掉零都卜勒雜波（地物）。
- **MTD / 脈衝都卜勒**：對多個脈衝做 FFT，分辨速度、抑制雜波。
- **盲速（blind speed）**：都卜勒折疊到雜波頻率時失效，用參差 PRF 解。

### 5.4 模糊與解模糊
距離模糊（PRF 高）與速度模糊（PRF 低）此消彼長；多 PRF / 波形設計折衷。

## 第 6 章　雷達訊號處理鏈

```
天線 → RF 前端 → ADC → 脈衝壓縮 → 都卜勒處理（FFT）→
       CFAR 偵測 → 角度估測 / 測向 → 追蹤（Kalman）→ 分類
```

- **匹配濾波**：最大化 SNR 的最佳線性接收（複習數位通訊）。
- **CFAR（恆虛警率）**：依鄰近雜訊自適應設門檻，控制虛警率。
- **追蹤**：Kalman / 卡爾曼濾波、資料關聯（多目標 JPDA、MHT）。
- **分類**：微都卜勒、高解析距離像、深度學習。

## 第 7 章　相位陣列雷達

### 7.1 電子掃描
用移相器 / 時間延遲改變各單元相位，**不動機械**即可掃描波束（A5 詳述陣列理論）。
- 掃描快、可多波束、可同時搜索 + 追蹤。

### 7.2 PESA vs AESA
- **PESA（被動）**：單一高功率源 + 移相器分配。
- **AESA（主動）**：每單元各有 T/R 模組（含 PA、LNA、移相器、衰減器）。
  - 可靠（單元失效退化柔和）、頻寬大、波形敏捷、低截獲（LPI）。
  - 現代戰機、艦載、汽車（MIMO 雷達）主流。

### 7.3 T/R 模組
GaN / GaAs MMIC：PA + LNA + 環行器 / 開關 + 移相器 + 衰減器，整合在每個天線單元背後。

## 第 8 章　雷達種類與應用

- **搜索 / 監視雷達**：空管、防空。
- **追蹤雷達**：單脈衝測角、精密追蹤。
- **氣象雷達**：都卜勒測風雨、雙偏極。
- **合成孔徑雷達（SAR）**：用平台運動合成大孔徑 → 高解析地表成像；ISAR 對動目標。
- **汽車雷達**：77 GHz FMCW MIMO，ADAS / 自駕測距測速。
- **穿牆 / 生命偵測**：UWB 雷達量呼吸心跳。
- **被動雷達**：借第三方訊號（廣播、5G）偵測，無自身發射。

## 第 9 章　電子戰與抗干擾（簡介）

- **ECM（電子干擾）**：壓制干擾、欺騙干擾。
- **ECCM（抗干擾）**：跳頻、波形捷變、旁瓣對消、空間濾波。
- **LPI（低截獲機率）**：低峰值功率、寬頻擴頻波形，難被截獲。

## 第 10 章　工程整合與其他射頻系統

- **通訊收發機**：手機 / Wi-Fi / 衛星，架構與前端與雷達同源。
- **電子設備相容性（EMC）**：高功率發射對自身與周邊系統的干擾。
- **封裝與散熱**：AESA / PA 高功率密度，熱管理是關鍵。
- **校正**：陣列各通道幅相校正決定波束品質。
- **雷達-通訊一體化（JCAS / ISAC）**：6G 趨勢，同一波形兼做感測與通訊。
