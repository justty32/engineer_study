# A5 天線陣列與波束成形（Antenna Arrays & Beamforming）概論

> 天線特化第 5 科（天線特化完結篇）。
> 名詞對照見 [中英名詞對照表 → 天線](../中英名詞對照表-4.md#天線)。

## 這科在學什麼

單一天線的方向圖由其結構固定；把多個天線排成**陣列**並控制各單元的**振幅與相位**，就能「電子化」地塑形與導向波束。本科深入：

1. 陣列因子與方向圖合成
2. 相位掃描與柵瓣
3. 振幅錐削與副瓣控制
4. 平面 / 大規模陣列
5. 波束成形（類比 / 數位 / 混合）
6. 自適應陣列與 MIMO（接回通訊）

這科是 A4 相位陣列雷達、5G 大規模 MIMO、衛星多波束的理論核心。

## 第 1 章　為什麼要陣列

- **高增益**：N 個單元同相相加，主瓣峰值功率 $\propto N^2$（場相加），增益約提升 $10\log N$。
- **電子掃描**：改相位即可移動波束，毫秒級、無機械慣性。
- **波束塑形**：壓低特定方向（抗干擾）、產生多波束。
- **空間多工（MIMO）**：同頻同時服務多用戶 / 多資料流。
- **分集**：對抗多徑衰落。

## 第 2 章　陣列因子（Array Factor）

### 2.1 均勻線陣（ULA）
N 個單元沿一線、間距 $d$、單元激發 $a_n=|a_n|e^{j\beta_n}$。遠場各單元路程差造成相位差 $kd\cos\theta$：
$$AF(\theta)=\sum_{n=0}^{N-1}a_n\,e^{jn(kd\cos\theta+\beta)}$$
**方向圖相乘原理**：總方向圖 = 單元方向圖 × 陣列因子。

### 2.2 均勻激發的 AF
$a_n=1$、線性遞增相位 $\beta$：
$$|AF(\theta)|=\left|\dfrac{\sin(N\psi/2)}{\sin(\psi/2)}\right|,\quad \psi=kd\cos\theta+\beta$$
- 主瓣在 $\psi=0$。
- 主瓣寬 $\propto 1/(Nd)$：陣列越大（孔徑越大）波束越窄。
- 第一副瓣約 −13.2 dB（均勻激發固有）。

## 第 3 章　波束導向與柵瓣

### 3.1 相位掃描
要把主瓣指向 $\theta_0$，令各單元漸進相位 $\beta=-kd\cos\theta_0$。
- 改 $\beta$（移相器）即可掃描，這就是相位陣列。

### 3.2 柵瓣（Grating Lobes）
當 $d$ 太大，$\psi$ 在可見區（$|\cos\theta|\le 1$）出現多個 $2\pi$ 整數倍 → 主瓣外又冒出同強度「柵瓣」。
- 避免條件：$d<\dfrac{\lambda}{1+|\cos\theta_0|}$；要掃到 ±90° 則 $d<\lambda/2$。
- 這就是「**半波長間距**」的由來。

### 3.3 波束展寬與掃描損失
波束掃離正前方時，等效孔徑投影變小 → 主瓣變寬、增益下降（約 $\cos\theta_0$）。

## 第 4 章　振幅錐削與副瓣控制

均勻激發副瓣只有 −13 dB，許多應用（雷達抗雜波、低旁瓣通訊）要更低：
- **加權（taper / windowing）**：中間單元權重大、邊緣小。
- 典型分布：
  - **二項式**：無副瓣，但主瓣很寬、增益低。
  - **Dolph–Chebyshev**：給定副瓣電平下主瓣最窄（最佳折衷）。
  - **Taylor**：可控的近副瓣 + 平滑遠副瓣，實務常用。
- 取捨：副瓣越低 → 主瓣越寬、增益略降、孔徑效率下降。

## 第 5 章　平面與大規模陣列

### 5.1 平面陣列
單元排成 2D（矩形 / 三角格）→ 可在方位 + 仰角兩維掃描，產生筆形波束（pencil beam）。
- 矩形陣 AF = 兩方向 AF 之積（可分離）。
- 三角格在相同柵瓣限制下用更少單元。

### 5.2 大規模 MIMO（Massive MIMO）
基地台數十～數百天線：
- 窄波束對準用戶 → 高增益、低干擾。
- 同頻空間多工多用戶。
- 5G sub-6 GHz 與 mmWave 的核心技術。

### 5.3 共形陣列（Conformal）
貼合曲面（機身、飛彈、車體）的陣列，需逐單元補償幾何相位。

## 第 6 章　波束成形架構

### 6.1 類比波束成形（Analog BF）
在 RF（或 IF）以移相器 / 衰減器加權，後接單一收發鏈。
- 優點：硬體少、功耗低。
- 缺點：一次一個波束、頻寬受限（移相器非真時延 → beam squint）。
- mmWave 前端常用。

### 6.2 數位波束成形（Digital BF）
每單元各有獨立收發鏈 + ADC/DAC，在基頻數位加權。
- 優點：同時多波束、彈性最大、適合 MIMO。
- 缺點：鏈路與功耗、資料量巨大。

### 6.3 混合波束成形（Hybrid BF）
少數 RF 鏈（類比）+ 數位預編碼，子陣列分群。
- mmWave 大規模 MIMO 的實務折衷：兼顧多流與功耗。

### 6.4 真時延（True-Time-Delay）
寬頻 / 大孔徑用時延取代移相，避免 beam squint（寬頻訊號各頻率指向不同方向）。

## 第 7 章　自適應陣列與測向

### 7.1 自適應波束成形
依環境即時調整權重：
- **MVDR / Capon**：在維持目標方向增益下最小化輸出功率（壓制干擾）。
- **LCMV**：多重線性約束。
- **零陷導向（null steering）**：把方向圖零點對準干擾源。
- **LMS / RLS / SMI**：自適應演算法更新權重。

### 7.2 到達方向估測（DOA）
- **波束掃描 / Capon 譜**。
- **子空間法：MUSIC、ESPRIT**——超解析，能分辨比波束寬更近的訊號源。
- 應用：測向、定位、雷達測角、智慧天線。

### 7.3 校正
各通道幅相不一致會破壞波束 / 零陷品質 → 內建校正網路、線上校正。

## 第 8 章　接回通訊：MIMO 與預編碼

### 8.1 從波束成形到 MIMO
- **波束成形**：增益 / 方向（單流）。
- **空間多工**：多天線送多獨立資料流，容量隨 $\min(N_t,N_r)$ 線性增長。
- **分集**：空時編碼（STBC）對抗衰落。

### 8.2 預編碼（Precoding）
- 已知通道 H → SVD 分解出平行子通道 + 注水功率分配（複習資訊理論）。
- 多用戶 MIMO：ZF / MMSE 預編碼、髒紙編碼（DPC 理論上限）。

### 8.3 通道資訊
- TDD 用互易性、FDD 需回授 CSI。
- 大規模 MIMO 的導頻汙染問題。

## 第 9 章　典型系統

- **相位陣列雷達 / AESA**（接 A4）：搜索 + 追蹤 + 抗干擾。
- **5G 基地台**：64T64R 大規模 MIMO、波束管理。
- **衛星多波束 / 相位陣列地面站**：Starlink 等 LEO 用平板電子掃描天線。
- **Wi-Fi 6/7**：多天線波束成形 + MU-MIMO。
- **汽車 MIMO 雷達**：虛擬陣列（TDM-MIMO）擴增角解析。

## 第 10 章　趨勢

- **可重構智慧表面（RIS / IRS）**：被動反射面調控傳播環境，6G 熱門。
- **全息 MIMO / 超大孔徑陣列**。
- **太赫茲超大規模陣列**。
- **天線-RF-基頻聯合設計（co-design）**、**AI 輔助波束管理**。
- **雷達-通訊一體化（ISAC）** 的共用陣列。
