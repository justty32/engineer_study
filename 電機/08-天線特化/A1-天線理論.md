# A1 天線理論（Antenna Theory）概論

> 天線特化第 1 科。
> 名詞對照見 [中英名詞對照表 → 天線](../中英名詞對照表-4.md#天線)。

## 這科在學什麼

天線是電磁波與電路的「介面元件」。本科把電磁學裡的輻射、傳輸線、波傳，整合到天線的設計與分析：輻射機制、各種天線拓樸、性能參數、極化、阻抗匹配、陣列基礎。

## 第 1 章　輻射的物理本質

### 1.1 加速電荷會輻射
靜止電荷不輻射；等速直線運動也不輻射（在慣性參考系內）；**加速**電荷才會輻射電磁波。

### 1.2 從馬克士威方程式到輻射
時變源（$\rho,\mathbf{J}$）產生時變場 $\mathbf{E}, \mathbf{B}$；場以波動方程傳播出去。
近場（reactive、Fresnel）/ 遠場（Fraunhofer）：
- **遠場條件**：$r>\dfrac{2D^2}{\lambda}$，$D$ 為天線最大尺寸。
- 在遠場，$\mathbf{E}$、$\mathbf{H}$ 互相垂直、與傳播方向垂直，$|E|/|H|=\eta_0\approx 377\,\Omega$。

### 1.3 向量位（Vector Potential）視角
$$\mathbf{A}(\mathbf{r})=\dfrac{\mu_0}{4\pi}\int\dfrac{\mathbf{J}(\mathbf{r}')e^{-jk|\mathbf{r}-\mathbf{r}'|}}{|\mathbf{r}-\mathbf{r}'|}\,dv'$$
從電流分布算 $\mathbf{A}$，再得 $\mathbf{B}=\nabla\times\mathbf{A}$、$\mathbf{E}=\dots$。
這條積分是所有天線輻射分析的根本。

## 第 2 章　天線性能參數

### 2.1 輻射圖（Radiation Pattern）
天線輻射強度（功率密度）隨方向 $(\theta,\phi)$ 的分布。常用 dB 表示。
- **主瓣（main lobe）**、**副瓣（side lobe）**、**後瓣（back lobe）**。
- **半功率波束寬度（HPBW）**：主瓣 −3 dB 兩側角寬。

### 2.2 方向性與增益
- **方向性 D**：最大輻射強度 / 全向均勻分配的強度。
- **增益 G**：含天線效率，$G=\eta_a D$，$\eta_a$ 為輻射效率。
- 單位 dBi（相對等向天線）、dBd（相對半波偶極）。

### 2.3 有效面積（Effective Aperture）
接收觀點下，天線從入射波擷取功率的能力：
$$A_e=\dfrac{G\lambda^2}{4\pi}$$

### 2.4 輸入阻抗與匹配
- 輸入阻抗 $Z_\text{in}=R_\text{rad}+R_\text{loss}+jX$。
- **輻射電阻**：把功率輻射出去的等效電阻。
- 與饋線（多數 50 Ω）匹配以減反射。
- **回波損失（return loss）**、**VSWR**。

### 2.5 極化（Polarization）
- 線性、圓、橢圓。
- 收 / 發極化不匹配 → 極化損失（兩線性垂直 → 完全失配）。
- 圓極化對抗 multipath、Faraday rotation。

### 2.6 頻寬
天線在某些指標（VSWR < 2、增益不掉太多）下的可用頻率範圍。
- 窄頻：< 5%。
- 寬頻 / 多頻：覆蓋多個服務（手機 4-5 個頻段）。
- 超寬頻（UWB）：> 25%。

## 第 3 章　基本天線

### 3.1 偶極天線（Dipole）
- **半波偶極（λ/2）**：最經典，$Z_\text{in}\approx 73+j42.5\,\Omega$。圓對稱 8 字輻射圖。
- **短偶極**（$\ell<\lambda/10$）：低 $R_\text{rad}$、難匹配，AM 廣播塔有用。

### 3.2 單極天線（Monopole）
偶極一半 + 地平面，常見於汽車收音機、無線網卡。半空間輻射，$Z_\text{in}\approx 36.5\,\Omega$。

### 3.3 環圈天線（Loop Antenna）
- **小環圈**：磁偶極輻射，弱輻射阻抗。AM 收音用。
- **大環圈（周長 ~ λ）**：較高效率，UHF 區廣用。

### 3.4 喇叭天線（Horn）
- 矩形 / 圓錐喇叭。
- 從波導開口逐漸放大，匹配自由空間。
- 衛星地面站、雷達校準。

### 3.5 拋物面（Parabolic Reflector）
- 反射面把饋源輻射聚焦或反向擴展為平行波。
- 大型衛星接收、雷達、射電天文。
- 高增益（30–60 dB）。

### 3.6 螺旋天線（Helical）
- 軸向模：圓極化，衛星通訊、GPS。

### 3.7 八木宇田（Yagi-Uda）
- 一驅動 + 多寄生（反射、引向）的線陣。
- 增益 10–20 dBi，廉價，老式電視天線。

## 第 4 章　平面與印刷天線

### 4.1 微帶（patch）天線
- 印在 PCB 上一片金屬貼片 + 地平面。
- 低剖面、易整合，但頻寬窄、增益有限。
- 變形：堆疊式、寄生 patch、L-probe、aperture coupled。

### 4.2 平面倒 F 天線（PIFA）
手機 / IoT 裝置常用。緊湊、多頻容易。

### 4.3 環形 / 槽（slot）天線
互補偶極（巴比涅原理）。

### 4.4 開放波導 / 開口（aperture）
波導開口直接輻射。

### 4.5 寬頻天線
- **對數週期（log-periodic）**：跨多倍頻仍工作。
- **錐 / 對數螺旋**：寬頻 + 圓極化。

## 第 5 章　天線陣列基礎

### 5.1 為什麼用陣列
- 提高增益、塑造方向圖。
- **波束掃描** 不用機械旋轉。
- 分集 / 多工。

### 5.2 陣列因子（Array Factor, AF）
N 個相同元件、間距 $d$、激發 $a_n$、相位 $\beta_n$：
$$AF(\theta)=\sum_{n=0}^{N-1}a_n e^{j(n k d\cos\theta+\beta_n)}$$
整體輻射圖 = 元件圖 × AF。

### 5.3 均勻線陣（ULA）波束
- 主瓣方向 $\cos\theta_0=-\beta/(kd)$ → 用相位控制波束掃描。
- 副瓣電平 ≈ −13 dB（均勻幅度）。
- 加幅度權重（Chebyshev、Taylor）降低副瓣，代價是主瓣變寬。

### 5.4 平面 / 立體陣列
- 矩形格、三角格、圓形、隨機。
- 大規模 MIMO 用平面陣列做 2D 波束成形。

## 第 6 章　天線量測與設計流程

### 6.1 模擬工具
- **方法**：方法矩陣（MoM、HFSS、CST、FEKO）、有限元素（FEM、HFSS）、時域有限差分（FDTD）。
- 設計多用全波模擬器迭代。

### 6.2 量測設施
- **暗室（anechoic chamber）**：自由空間環境。
- **近場測量 + 遠場轉換**。
- **球面 / 平面掃描**。
- VNA（向量網路分析儀）量 $S_{11}$、$Z_\text{in}$。

### 6.3 主要量測量
- 輻射圖（方位 / 仰角）
- 增益（替代法、絕對法）
- 極化、軸比
- 輸入阻抗 / VSWR
- 效率

### 6.4 設計流程
1. 規格：頻段、增益、極化、尺寸、成本。
2. 選拓樸（patch / horn / Yagi …）。
3. 全波模擬粗調 → 微調幾何 / 饋入。
4. 製造原型 → 量測 → 修正。
5. EMC、機械、環境測試。

## 第 7 章　應用速覽（先導，A2–A5 細談）

- 手機天線：多頻 PIFA / IFA、5G mmWave 模組。
- Wi-Fi：偶極、PCB pattern、MIMO 多支。
- GPS / GNSS：圓極化 patch 或 quadrifilar helix。
- 衛星地面站：拋物面 + 饋源。
- 5G 基地台：大規模平面陣列 + beamforming。
- 雷達：相位陣列、AESA。
- RFID：環形 / 偶極 + 整流。
- 醫療：植入式天線、無線供電。
- 太空 / 火箭 / 雷達：高增益反射面、phased array。

## 第 8 章　Friis 傳輸與鏈路預算（複習通訊系統）

$$P_R=P_T G_T G_R\left(\dfrac{\lambda}{4\pi d}\right)^2$$
- 自由空間下接收功率與距離平方反比、與波長平方成正比。
- 對特定 SNR 要求 → 算需要的 $P_T G_T G_R$。

雙路徑 / 多徑、繞射、大氣損耗都讓實際 < 自由空間。

## 第 9 章　天線與其他系統的整合

- 與**前端（LNA、PA、Filter）**：阻抗匹配、共設計、整合在 RFIC 中。
- 與**Mechanical**：高頻天線對結構件相對位置極敏感。
- 與**EMC**：天線 = 主動輻射源；其他電路也有天線效應（要 EMC）。
