# C3 強健控制（Robust Control）概論

> 控制特化第 3 科。
> 名詞對照見 [中英名詞對照表 → 強健控制](../中英名詞對照表-4.md#強健控制)。

## 這科在學什麼

真實系統與模型總有差距：參數誤差、未建模動態、雜訊。**強健控制**要求設計出的控制器在「**所有可能的不確定下**」都仍能穩定且滿足性能。

工具：$H_\infty$ 範數、結構奇異值 $\mu$、LMI、Kharitonov 定理。
應用：航空、汽車、製造、半導體製程——任何「模型不準也得安全運轉」的場合。

## 第 1 章　不確定性建模

### 1.1 兩種主要不確定性
- **參數（structured）**：物理參數有範圍（例：質量在 ±10%）。
- **未建模動態（unstructured）**：模型沒抓到的高頻動態、感測 / 致動延遲。

### 1.2 表示法
- **加性（additive）**：$G_p=G+\Delta_a$。
- **乘性（multiplicative）**：$G_p=G(1+\Delta_m W_m)$，$\|\Delta_m\|_\infty\le 1$。
- **回授扰動**、輸入 / 輸出乘性等。
- 結構化：每個 $\Delta_i$ 對應一個物理參數。

### 1.3 性能與不確定性的「標準形式」
把控制問題重組為 $\mathbf{P}-\mathbf{K}$ 結構，加上 $\Delta$ 包住所有不確定：
```
        [w]            [z]
   ──→  P  ──→
        ▲ ▲ ▲          ▲
        │ │ │          │
        Δ─┘ K───────────┘
```
分析：給定 K 求 worst-case 性能。
設計：找 K 使 worst-case 性能在所有 $\|\Delta\|\le 1$ 下達標。

## 第 2 章　範數工具

### 2.1 訊號範數
- **$\mathcal{L}_2$**：能量 $\|x\|_2=\sqrt{\int |x|^2}$。
- **$\mathcal{L}_\infty$**：最大幅度（峰值）。

### 2.2 系統範數
- **$\mathcal{H}_2$**：$\|G\|_2=\sqrt{\dfrac{1}{2\pi}\int\|G(j\omega)\|_F^2 d\omega}$，能量增益。
- **$\mathcal{H}_\infty$**：$\|G\|_\infty=\sup_\omega \bar\sigma(G(j\omega))$，最壞情況增益（最大奇異值上界）。

$H_\infty$ 範數 = 從輸入到輸出最差能量放大率 → 強健性設計的天然語言。

### 2.3 小增益定理（Small Gain Theorem）
若 $\|M\|_\infty\cdot\|\Delta\|_\infty < 1$，則 $M$-$\Delta$ 回授系統穩定。
是強健穩定性的核心判據。

## 第 3 章　強健穩定性與性能

### 3.1 強健穩定性（Robust Stability, RS）
對所有 $\|\Delta\|_\infty\le 1$ 系統仍穩定。
用小增益：對乘性扰動，**強健穩定條件**：
$$\|W_m T\|_\infty<1$$
$T=GK/(1+GK)$ 為閉迴路互補敏感度，$W_m$ 為扰動的權重。

### 3.2 標稱性能（Nominal Performance, NP）
標稱系統下滿足要求，例：
$$\|W_p S\|_\infty<1$$
$S=1/(1+GK)$ 為敏感度，$W_p$ 性能權重。

### 3.3 強健性能（Robust Performance, RP）
任意 $\|\Delta\|\le 1$ 下都滿足性能：對 SISO 乘性扰動：
$$\||W_p S|+|W_m T|\|_\infty<1$$

對 MIMO / 結構化扰動 → 用 $\mu$（結構奇異值）。

## 第 4 章　$H_\infty$ 控制設計

### 4.1 目標
給定 $G$ 與權重 $W_p, W_m, W_u$ 等，求 $K$ 使閉迴路 $\|\mathcal{T}_{zw}\|_\infty < \gamma$，$\gamma$ 越小越好。

### 4.2 解法（標準）
- **Riccati 形式**：解兩個耦合 Riccati 方程（Glover–Doyle, 1988）。
- **LMI 形式**：可用內點法解。線性矩陣不等式天然能加結構化約束、多目標。

### 4.3 權重函數選擇（最關鍵）
- $W_p$：低頻大、高頻小 → 強制低頻追蹤好。
- $W_m$：高頻大 → 抑制高頻不確定。
- $W_u$：限制控制能量。
- 經驗：先設「目標頻譜形狀」再對 $K$ 求解。

### 4.4 經典應用
- 飛機 / 直升機 / 無人機強健控制。
- 主動懸吊。
- 高精度伺服（HDD、半導體曝光機台）。
- 製程控制。

## 第 5 章　結構奇異值 $\mu$

### 5.1 為什麼需要
小增益對「無結構」扰動保守；如果 $\Delta=\text{diag}(\delta_1,\delta_2,\dots)$，可能太悲觀。
$\mu$ 把「$\Delta$ 結構」納入，得到更精確的強健穩定 / 性能判據。

### 5.2 定義
$$\mu_\Delta(M)=\dfrac{1}{\min\{\bar\sigma(\Delta):\det(I-M\Delta)=0,\ \Delta\in\boldsymbol{\Delta}\}}$$

### 5.3 強健性能定理
RP ⇔ $\sup_\omega \mu_{\hat\Delta}(M(j\omega))<1$，$\hat\Delta=\mathrm{diag}(\Delta_\text{扰動},\Delta_\text{性能})$。

### 5.4 $\mu$ 合成（DK 迭代）
$\mu$ 一般 NP-hard，工程上用「DK 迭代」：
1. 給定 K 算 $\mu$ 上界（**D-scale**）。
2. 給定 D 解 $H_\infty$ 問題求 K。
3. 重複到收斂。

工具：MATLAB Robust Control Toolbox。

## 第 6 章　Kharitonov 與參數強健性

### 6.1 Kharitonov 定理
特徵多項式係數有區間：$[a_i^-, a_i^+]$。只要檢查 **4 個特殊多項式**穩定 → 整個區間穩定。
- 適用於 SISO 線性系統的純參數扰動。
- 直觀、無需高深工具。

### 6.2 邊界 / 邊緣定理
推廣到多項式族、含某些結構。

### 6.3 與 $H_\infty$ / $\mu$ 互補
- Kharitonov 處理「參數有界」最強。
- $H_\infty$ / $\mu$ 同時處理參數 + 動態 + 性能，能量化、可設計。

## 第 7 章　LMI 框架

### 7.1 為什麼用 LMI
- 大量控制問題可寫成**凸**最佳化（LMI、SDP）。
- 多目標、多約束、結構限制都好加。
- 工具：MOSEK、SeDuMi、SDPT3、CVX、YALMIP。

### 7.2 經典 LMI 表示
- **Lyapunov 穩定性**：$A^\top P+PA\prec 0$、$P\succ 0$。
- **$H_\infty$ 範數約束**：Bounded Real Lemma。
- **狀態回授 LQR / $H_\infty$**：聯立 LMI 求 $P, K$。
- **多目標 / 多 cell**：把所有條件堆在一起求解。

### 7.3 應用
- 強健 MPC：把不確定 + 約束放進 LMI。
- 增益排程（Gain-scheduled LQG / $H_\infty$）：LPV 系統。
- Network Control（時變延遲）。

## 第 8 章　強健性與性能權衡

### 8.1 經典 trade-off
$$S+T=I$$
（敏感度 + 互補敏感度 = 1）
所以**低頻追蹤好** ↔ **高頻干擾抑制好** ↔ **強健不確定** 三者之間要分配「頻寬」。

工程實務：性能頻寬 < 不確定起點。

### 8.2 雜訊與性能
量測雜訊越大 → 控制頻寬要降 → 追蹤性能下降。

### 8.3 控制能量
飽和（saturation）會破壞線性假設 → 加 anti-windup。

## 第 9 章　LPV / 增益排程（Gain Scheduling）

### 9.1 動機
非線性系統 / 工作點變化大 → 在多個工作點線性化 → 各設計線性控制器 → 平滑切換。

### 9.2 LPV（Linear Parameter Varying）
系統矩陣依參數 $\theta(t)$ 變動。設計考慮整個 $\theta$ 範圍。
- 用 LMI / $H_\infty$ 多面體模型。
- 應用：飛機高度 / 速度大範圍變化、馬達磁飽和。

## 第 10 章　工程應用案例

### 10.1 飛機飛控（fly-by-wire）
- 模型隨高度 / 速度 / 載重變化。
- LPV / $H_\infty$ 增益排程是業界標準。

### 10.2 硬碟讀寫頭伺服
- 高頻機械諧振 + 低頻擾動。
- $H_\infty$ 控制器設計頻譜整形。

### 10.3 半導體微影機台
- 奈米級精度 + 強烈振動 + 熱漂移。
- 強健控制 + 前饋 + 學習控制（ILC）混合。

### 10.4 自駕車轉向 / 制動
- 摩擦、輪胎、載重不確定。
- LPV + 強健 MPC 趨勢。

### 10.5 醫療呼吸器
- 病人病肺特性多變 + 安全嚴格 → 強健控制 + Safety Mechanism。

## 第 11 章　從強健到 學習結合

### 11.1 強健 + 自適應
保證強健穩定的同時，慢慢學參數調適（[C4 自適應控制](C4-非線性與自適應控制.md)）。

### 11.2 強健 + RL
- Safe RL：強化學習過程中始終滿足強健穩定。
- Robust Adversarial RL：訓練時加入扰動，提升 policy 在實機的容錯。

### 11.3 數據驅動強健控制（Data-Driven）
- 直接從資料估「不確定球」與系統，跳過顯式建模。
- Willems 基本引理 → 數據驅動 MPC。
- 近年熱門研究方向。
