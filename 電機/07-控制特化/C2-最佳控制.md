# C2 最佳控制（Optimal Control）概論

> 控制特化第 2 科。
> 名詞對照見 [中英名詞對照表 → 最佳控制](../中英名詞對照表-4.md#最佳控制)。

## 這科在學什麼

把控制設計變成一個**最佳化問題**：找控制律 $\mathbf{u}(\cdot)$ 在約束下最小化代價函式。
工具：變分法、Pontryagin 最大原理、Hamilton–Jacobi–Bellman 方程、Riccati 方程、動態規劃。
對控制工程師最常用的「LQR / LQG / Kalman / MPC」都從這個框架而來。

## 第 1 章　最佳控制問題的標準形式

### 1.1 連續時間
$$\min_{\mathbf{u}(\cdot)}\ J=\Phi(\mathbf{x}(t_f),t_f)+\int_{t_0}^{t_f}L(\mathbf{x},\mathbf{u},t)\,dt$$
$$\text{s.t.}\quad \dot{\mathbf{x}}=f(\mathbf{x},\mathbf{u},t),\quad \mathbf{x}(t_0)=\mathbf{x}_0,\quad \mathbf{u}(t)\in\mathcal{U}$$

- $\Phi$：終端代價。
- $L$：行進代價（running cost）。
- $f$：動態（可能非線性）。
- $\mathcal{U}$：控制限制（飽和、安全）。

### 1.2 離散時間
$$\min_{\mathbf{u}_k}\ J=\Phi(\mathbf{x}_N)+\sum_{k=0}^{N-1}L(\mathbf{x}_k,\mathbf{u}_k)$$
$$\text{s.t.}\quad \mathbf{x}_{k+1}=f(\mathbf{x}_k,\mathbf{u}_k)$$

工程上的 MPC、強化學習 Q-learning 都以這形式起跳。

## 第 2 章　變分法（Calculus of Variations）

### 2.1 Euler–Lagrange 方程
無約束 functional 的極值條件：
$$\dfrac{\partial L}{\partial \mathbf{x}}-\dfrac{d}{dt}\dfrac{\partial L}{\partial \dot{\mathbf{x}}}=0$$

### 2.2 加上動態約束（拉氏乘子）
引入 $\boldsymbol{\lambda}(t)$（協態 / co-state）：
$$\tilde{J}=\int [L+\boldsymbol{\lambda}^\top(f-\dot{\mathbf{x}})]\,dt+\Phi$$
對 $\mathbf{x},\mathbf{u},\boldsymbol{\lambda}$ 取變分得到必要條件。

### 2.3 Hamiltonian
$$H(\mathbf{x},\mathbf{u},\boldsymbol{\lambda},t)=L+\boldsymbol{\lambda}^\top f$$
最佳化必要條件變成：
$$\dot{\mathbf{x}}=\dfrac{\partial H}{\partial\boldsymbol{\lambda}},\quad \dot{\boldsymbol{\lambda}}=-\dfrac{\partial H}{\partial\mathbf{x}},\quad \dfrac{\partial H}{\partial\mathbf{u}}=0$$
（最佳控制是 $\mathbf{u}^*=\arg\min_{\mathbf{u}\in\mathcal{U}} H$）

加上邊界條件（$\mathbf{x}(t_0)=\mathbf{x}_0$、$\boldsymbol{\lambda}(t_f)=\partial\Phi/\partial\mathbf{x}$）。

## 第 3 章　Pontryagin 最大原理（PMP）

當控制有飽和（$\mathbf{u}\in\mathcal{U}$）時，$\dfrac{\partial H}{\partial\mathbf{u}}=0$ 不一定有解；改成：
$$\mathbf{u}^*(t)=\arg\min_{\mathbf{u}\in\mathcal{U}} H(\mathbf{x}^*,\mathbf{u},\boldsymbol{\lambda}^*,t)$$

**典型結果**：
- **時間最短控制**：bang-bang，控制總在邊界。
- **最少燃料**：bang-zero-bang。
- 開關時刻由「切換函數（switching function）」決定。

PMP 把動態最佳化變成「兩點邊界值問題（TPBVP）」，解法通常需要數值（shooting、collocation）。

## 第 4 章　動態規劃與 HJB 方程

### 4.1 動態規劃原理（Bellman）
最佳子問題：以「現在狀態」為起點的最佳代價函式 $V(\mathbf{x},t)$ 滿足：
$$V(\mathbf{x},t)=\min_{\mathbf{u}}\{L(\mathbf{x},\mathbf{u})\Delta t+V(\mathbf{x}+f\Delta t,t+\Delta t)\}$$

### 4.2 HJB（Hamilton–Jacobi–Bellman）
取 $\Delta t\to 0$：
$$-\dfrac{\partial V}{\partial t}=\min_{\mathbf{u}}\left[L+\dfrac{\partial V}{\partial\mathbf{x}}^\top f\right]$$
- 終端條件：$V(\mathbf{x},t_f)=\Phi(\mathbf{x})$。
- 解 HJB 後，最佳控制律 $\mathbf{u}^*(\mathbf{x},t)=\arg\min_{\mathbf{u}}[L+V_x^\top f]$，是**狀態回授形式**（強大！）。

### 4.3 計算困難（curse of dimensionality）
- 連續狀態 → 偏微分方程，幾乎無解析解。
- 離散狀態：狀態數隨維度指數爆炸。
- 對策：函數逼近（神經網路 → 深度強化學習）、模型預測（MPC）。

## 第 5 章　LQR（Linear Quadratic Regulator）

### 5.1 設定
連續 LTI + 二次代價（無限期）：
$$\dot{\mathbf{x}}=A\mathbf{x}+B\mathbf{u}$$
$$J=\int_0^\infty(\mathbf{x}^\top Q\mathbf{x}+\mathbf{u}^\top R\mathbf{u})\,dt,\quad Q\succeq 0,\ R\succ 0$$

### 5.2 解：代數 Riccati 方程（CARE）
$$A^\top P+PA-PBR^{-1}B^\top P+Q=0$$
求 $P\succeq 0$ 的穩定解。

### 5.3 最佳控制律
$$\mathbf{u}^*(t)=-K\mathbf{x}(t),\quad K=R^{-1}B^\top P$$
是**狀態回授**——一個簡單矩陣乘法。

### 5.4 性質
- 閉迴路 $A-BK$ 必定穩定（若 $(A,B)$ 可穩）。
- 相位裕度 ≥ 60°、增益裕度無限（很強的內建強健性）—— 連續時間 LQR 的經典結果。
- 對雜訊強健性可疑（一旦加觀測器就不一定）。

### 5.5 設計直覺
- $Q$ 大 → 重視狀態追蹤 → 增益高、控制激進。
- $R$ 大 → 重視能量 → 增益低、控制保守。
- 對角 $Q,R$ 用 Bryson rule 初始化：對角項取「最大可接受值」的倒數平方。

### 5.6 離散 LQR
類似 Riccati 形式，工程上 MCU / DSP 用此實作。

## 第 6 章　LQG（Linear Quadratic Gaussian）

### 6.1 問題
受高斯雜訊干擾的 LTI 系統 + 二次代價 + 不完全狀態量測：
$$\dot{\mathbf{x}}=A\mathbf{x}+B\mathbf{u}+G\mathbf{w},\quad \mathbf{y}=C\mathbf{x}+\mathbf{v}$$
$\mathbf{w}\sim N(0,Q_w),\ \mathbf{v}\sim N(0,R_v)$。

### 6.2 解（分離原理）
1. 設計 LQR 求 $K$（用 $Q,R$ 代價）。
2. 設計 Kalman 濾波器求 $L$（用 $Q_w,R_v$ 統計）。
3. 控制律：$\mathbf{u}^*=-K\hat{\mathbf{x}}$，$\hat{\mathbf{x}}$ 來自 Kalman。

### 6.3 強健性陷阱
LQG 沒有 LQR 的內建強健性（Doyle 1978 famous「LQG has no guaranteed margins」）。
→ 衍生 LQG/LTR（loop transfer recovery）、$H_\infty$ 控制。

## 第 7 章　Kalman 濾波器

### 7.1 連續時間
$$\dot{\hat{\mathbf{x}}}=A\hat{\mathbf{x}}+B\mathbf{u}+L(\mathbf{y}-C\hat{\mathbf{x}})$$
最佳 $L=PC^\top R_v^{-1}$，$P$ 解：
$$AP+PA^\top-PC^\top R_v^{-1}CP+GQ_wG^\top=0$$

### 7.2 離散時間 Kalman（標準工程版）
**預測**：
$$\hat{\mathbf{x}}^-_k=A\hat{\mathbf{x}}_{k-1}+B\mathbf{u}_{k-1}$$
$$P^-_k=A P_{k-1} A^\top+Q_w$$

**更新**：
$$K_k=P^-_k C^\top(CP^-_k C^\top+R_v)^{-1}$$
$$\hat{\mathbf{x}}_k=\hat{\mathbf{x}}^-_k+K_k(\mathbf{y}_k-C\hat{\mathbf{x}}^-_k)$$
$$P_k=(I-K_k C)P^-_k$$

### 7.3 延伸
- **EKF（Extended）**：把非線性 $f,h$ 線性化後做 Kalman。
- **UKF（Unscented）**：用 Sigma 點傳遞均值與協方差，對強非線性更穩。
- **粒子濾波器（Particle Filter）**：粒子採樣表示後驗，處理非高斯。

### 7.4 應用
- GNSS / INS 融合定位。
- 機器人 SLAM。
- 雷達 / 聲納目標追蹤。
- 馬達無感測估角。
- 金融追蹤、感測器融合。

## 第 8 章　模型預測控制（MPC）

### 8.1 概念
每個時刻：
1. 根據目前狀態 $\mathbf{x}_k$，**在 $N$ 步預測視窗內**解最佳化問題。
2. 取得最佳控制序列 $\mathbf{u}_k,\dots,\mathbf{u}_{k+N-1}$。
3. **只施加第一步** $\mathbf{u}_k$。
4. 下一時刻重新解最佳化（receding horizon）。

### 8.2 線性 MPC
動態線性 + 二次代價 + 線性約束：
$$\min\ \sum_{i=0}^{N-1}(\mathbf{x}_i^\top Q\mathbf{x}_i+\mathbf{u}_i^\top R\mathbf{u}_i)+\mathbf{x}_N^\top P\mathbf{x}_N$$
$$\text{s.t.}\quad \mathbf{x}_{i+1}=A\mathbf{x}_i+B\mathbf{u}_i,\ \mathbf{u}_i\in\mathcal{U},\ \mathbf{x}_i\in\mathcal{X}$$
→ 二次規劃（QP）問題，可即時解。

### 8.3 非線性 MPC（NMPC）
動態 / 代價 / 約束都可以非線性 → 非線性規劃，計算量大。常用直接 collocation、shooting + IPOPT。

### 8.4 穩定性與可行性
- **終端代價 + 終端集合** 條件保證穩定（依 Mayne et al. 經典理論）。
- 兼顧可行性（infeasibility recovery、軟約束）。

### 8.5 工程實務
- 工業 MPC（化工、煉油）30 年以上歷史。
- 自駕車軌跡追蹤、無人機飛控、機器人運動規劃。
- 嵌入式 MPC：用 active set 或 ADMM 在 MCU 上即時跑。

## 第 9 章　強化學習（RL）視角的最佳控制

### 9.1 與最佳控制的對應
| 控制 | RL |
| --- | --- |
| 狀態 | state |
| 控制 | action |
| 動態 | environment |
| 代價函數 | reward（負代價） |
| 價值函數 $V$ | value function |
| Q 函數 $Q(\mathbf{x},\mathbf{u})$ | action-value |

RL 是 model-free / model-based 都包括的解最佳控制方法。

### 9.2 經典演算法
- **Q-learning / DQN**
- **Policy gradient / REINFORCE**
- **Actor-Critic、A3C**
- **PPO、SAC、TD3**：現代主流。
- **Model-based RL**：學習模型 + 用 MPC / shooting 決策。

### 9.3 與工程控制結合
- **Iterative LQR（iLQR）/ Differential Dynamic Programming（DDP）**：把 RL 風格用在 model-based 控制。
- **Sim-to-real**：模擬中訓練 RL，部署到實機。
- **Safe RL**：用控制理論證明 RL 不違反安全限制。

## 第 10 章　應用速覽

### 10.1 太空 / 航空
- 火箭與衛星姿態控制（LQR / MPC）。

- SpaceX 著陸：凸最佳化即時求解。

### 10.2 汽車
- 自動駕駛軌跡規劃 + 跟隨（MPC）。

### 10.3 機器人
- 機械手臂軌跡規劃（DDP / iLQR）。
- 雙足機器人步態控制（MPC + WBC）。
- 無人機（PX4 用 PID + 部分 LQR / MPC）。

### 10.4 工業流程
- 化工 / 煉油 / 半導體製程：DMC、QDMC 至今仍是工業 MPC 標準。

### 10.5 能源
- 智慧電網最佳調度。
- 電池儲能管理。
- 建築物 HVAC 預測控制。
