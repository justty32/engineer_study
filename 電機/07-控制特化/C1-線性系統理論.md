# C1 線性系統理論（Linear Systems Theory）概論

> 控制特化第 1 科。
> 名詞對照見 [中英名詞對照表 → 線性系統](../中英名詞對照表-4.md#線性系統)。

## 這科在學什麼

把控制系統（[13](../02-電機核心/13-控制系統.md)）裡的「狀態空間 + 特徵值」這條線推到完整，補齊**現代控制**的數學底層：可控可觀性、Lyapunov 穩定性、實現理論、穩定鑑別、極點配置、觀測器設計。
這是後續最佳 / 強健 / 非線性 / 自適應 / MPC 等所有現代控制的「線性架構」。

## 第 1 章　狀態空間（State Space）

### 1.1 連續時間 LTI
$$\dot{\mathbf{x}}(t)=A\mathbf{x}(t)+B\mathbf{u}(t)$$
$$\mathbf{y}(t)=C\mathbf{x}(t)+D\mathbf{u}(t)$$

- $\mathbf{x}\in\mathbb{R}^n$：狀態。
- $\mathbf{u}\in\mathbb{R}^m$：輸入。
- $\mathbf{y}\in\mathbb{R}^p$：輸出。
- $A,B,C,D$：固定矩陣。

### 1.2 離散時間 LTI
$$\mathbf{x}[k+1]=A_d\mathbf{x}[k]+B_d\mathbf{u}[k],\quad \mathbf{y}[k]=C\mathbf{x}[k]+D\mathbf{u}[k]$$
由連續系統用 ZOH 離散化得 $A_d=e^{AT},\ B_d=\int_0^T e^{A\tau}d\tau\cdot B$。

### 1.3 狀態方程的解
$$\mathbf{x}(t)=e^{At}\mathbf{x}(0)+\int_0^t e^{A(t-\tau)}B\mathbf{u}(\tau)\,d\tau$$
**矩陣指數 $e^{At}$** 由特徵值 / 特徵向量決定：
$$e^{At}=Te^{Dt}T^{-1}\quad(\text{若 }A=TDT^{-1}\text{ 可對角化})$$

特徵值在左半平面 → $e^{At}$ 衰減 → 系統內在穩定。

### 1.4 轉移函數與狀態空間互換
$$H(s)=C(sI-A)^{-1}B+D$$
反向：給定 $H(s)$，可造出狀態空間實現（controllable canonical、observable canonical、Jordan）。

## 第 2 章　穩定性

### 2.1 內部穩定 vs BIBO
- **內部（asymptotic / Lyapunov）**：$\mathbf{x}(t)\to 0$ 對任意 $\mathbf{x}(0)$。
- **BIBO**：有界輸入 → 有界輸出。
線性 LTI 中兩者等價於 $A$ 特徵值都在左半平面（連續）或單位圓內（離散）。

### 2.2 Lyapunov 方程
$$A^\top P+PA=-Q,\quad Q>0\Rightarrow P>0\text{ 唯一解，系統穩定}$$
（離散版：$A^\top PA-P=-Q$）

意義：找到 $V(\mathbf{x})=\mathbf{x}^\top P\mathbf{x}$ 為 Lyapunov 函數，沿軌跡單調下降 → 穩定。

### 2.3 Routh–Hurwitz / Nyquist
經典工具見 [13 控制系統](../02-電機核心/13-控制系統.md)。
線性系統視角下，Routh 等價於檢查特徵多項式根都在左半平面。

## 第 3 章　可控性（Controllability）

### 3.1 定義
是否能用適當的 $\mathbf{u}(\cdot)$ 把 $\mathbf{x}(0)$ 引到任意 $\mathbf{x}_f$。

### 3.2 可控矩陣
$$\mathcal{C}=[B,\ AB,\ A^2 B,\ \dots,\ A^{n-1}B]$$
**系統可控 ⇔ $\mathrm{rank}(\mathcal{C})=n$**。

### 3.3 PBH 檢驗
對每個特徵值 $\lambda_i$，矩陣 $[\lambda_i I-A,\ B]$ 滿秩。
與 $\mathcal{C}$ 等價、但對每個模態可分別判斷「能控 / 不能控」。

### 3.4 不可控時的結構
可分解 $\mathbf{x}=\begin{pmatrix}\mathbf{x}_c\\\mathbf{x}_{nc}\end{pmatrix}$，$\mathbf{x}_{nc}$ 不受 $\mathbf{u}$ 控制。整體穩定要求 $\mathbf{x}_{nc}$ 自然穩定（**可穩 stabilizable**）。

## 第 4 章　可觀性（Observability）

### 4.1 定義
是否能用 $\mathbf{u}(\cdot),\mathbf{y}(\cdot)$ 唯一決定 $\mathbf{x}(0)$。

### 4.2 可觀矩陣
$$\mathcal{O}=\begin{bmatrix}C\\CA\\CA^2\\\vdots\\CA^{n-1}\end{bmatrix}$$
**可觀 ⇔ $\mathrm{rank}(\mathcal{O})=n$**。

### 4.3 對偶性（Duality）
$(A,B,C)$ 可控可觀 ↔ $(A^\top,C^\top,B^\top)$ 可觀可控。
許多控制定理「對偶」於觀測器定理；證明常用 dual。

### 4.4 不可觀時的結構
類似：分解出**可偵測（detectable）**子系統，其餘自然穩定才行。

## 第 5 章　典型實現（Canonical Forms）

### 5.1 可控標準型（Controllable Canonical Form）
$A,B$ 變成「companion」形式，$B$ 為 $[0,\dots,0,1]^\top$。
- 直接從 $H(s)$ 分子 / 分母係數寫出。
- 適合做極點配置控制器設計。

### 5.2 可觀標準型
對偶版本，$C$ 為 $[1,0,\dots,0]$。

### 5.3 Jordan 形式
$A=TJT^{-1}$，$J$ 為 Jordan 塊。
適合分析特徵值多重度與廣義特徵向量；對重根與不可對角化系統很重要。

### 5.4 平衡實現（Balanced Realization）
讓可控 Gramian = 可觀 Gramian = 對角且相等。是**模型降階（model reduction）**的基礎（Hankel singular values）。

## 第 6 章　Gramian 與能量視角

### 6.1 定義
$$W_c=\int_0^\infty e^{At}BB^\top e^{A^\top t}\,dt$$
$$W_o=\int_0^\infty e^{A^\top t}C^\top C e^{At}\,dt$$
（穩定系統下存在）

### 6.2 解 Lyapunov
$AW_c+W_c A^\top=-BB^\top$、$A^\top W_o+W_o A=-C^\top C$。

### 6.3 意義
- $W_c$：把狀態從 0 開到某點所需最小能量的反向量化。
- $W_o$：從某狀態出發產生的輸出能量。
- 兩者結合定義 Hankel singular values，描述每個內部模態對輸入輸出的「重要性」。

## 第 7 章　狀態回授與極點配置

### 7.1 狀態回授
$$\mathbf{u}=-K\mathbf{x}+v$$
閉迴路：$\dot{\mathbf{x}}=(A-BK)\mathbf{x}+Bv$。

**極點配置定理**：若 $(A,B)$ 可控，則可選 $K$ 使 $A-BK$ 任意特徵值（受複數成對限制）。

### 7.2 設計方法
- **Ackermann 公式**（單輸入）：
$$K=[0,\dots,0,1]\mathcal{C}^{-1}\alpha_d(A)$$
$\alpha_d(s)$ 為期望特徵多項式。
- **多輸入**：自由度多，通常用最佳化（如 LQR）求 $K$。

### 7.3 限制
- 不可控部分 $\mathbf{x}_{nc}$ 的特徵值無法改變。
- 實際限制：$K$ 大 → 控制量飽和；$K$ 過於激進 → 雜訊放大。

## 第 8 章　觀測器（Observer / State Estimator）

### 8.1 Luenberger 觀測器
$$\dot{\hat{\mathbf{x}}}=A\hat{\mathbf{x}}+B\mathbf{u}+L(\mathbf{y}-C\hat{\mathbf{x}})$$
估測誤差 $\tilde{\mathbf{x}}=\mathbf{x}-\hat{\mathbf{x}}$ 滿足：
$$\dot{\tilde{\mathbf{x}}}=(A-LC)\tilde{\mathbf{x}}$$

**對偶極點配置**：若 $(A,C)$ 可觀，可選 $L$ 使 $A-LC$ 特徵值任意 → 估測誤差任意快收斂。

### 8.2 分離原理（Separation Principle）
用 $\hat{\mathbf{x}}$ 取代 $\mathbf{x}$ 做狀態回授 $\mathbf{u}=-K\hat{\mathbf{x}}$，整體閉迴路特徵值 = $\{A-BK\}\cup\{A-LC\}$。
所以控制器與觀測器可**獨立設計**。

### 8.3 簡化 / 降階觀測器
若部分狀態可直接量測 → 只估其餘部分，省結構與雜訊敏感度。

### 8.4 Kalman 濾波器（最佳線性觀測器）
有雜訊統計時：
- 過程雜訊 $\sim N(0,Q)$，量測雜訊 $\sim N(0,R)$。
- 解 Riccati 方程得**最佳** $L=PC^\top R^{-1}$。
- 詳細留給 C2 最佳控制 / [16 DSP](../03-通訊特化/16-數位信號處理-下.md)。

## 第 9 章　系統實現與模型降階

### 9.1 最小實現
從 $H(s)$ 找出可控 + 可觀的最低階狀態空間實現。
不必要的內部狀態（既不可控也不可觀）可剔除。

### 9.2 模型降階
- **Truncation（截斷）**：留下 Hankel 大值對應狀態。
- **Balanced truncation**：先平衡實現再截。
- **Krylov 子空間法**：對大型系統適用。
- **POD / 模態截斷**：適合 PDE 模型（流體、結構）。

降階用於控制器設計、模擬加速、嵌入式實作。

## 第 10 章　線性系統設計範式

### 10.1 SISO 經典控制（古典）
頻率域 + 根軌跡 + Bode + Nyquist。

### 10.2 SISO 現代控制
狀態空間 + 極點配置 + Luenberger。

### 10.3 MIMO 現代控制
- 狀態回授 / 觀測器。
- LQR / LQG。
- $H_\infty$、$\mu$ 合成（強健控制）。
- 詳細見後續 C2、C3。

### 10.4 數位實作
- 離散化（ZOH、Tustin）。
- 數值穩定的演算法（square-root Kalman、Schur 形式）。
- MCU / DSP 的定點化與 anti-windup。

## 第 11 章　幾個必懂的小工具

### 11.1 Schur 補（Schur Complement）
矩陣不等式（LMI）裡常用：
$$\begin{bmatrix}A&B\\B^\top&C\end{bmatrix}\succ 0\quad\Leftrightarrow\quad A\succ 0,\ C-B^\top A^{-1}B\succ 0$$

LMI 是 $H_\infty$、強健控制、MPC 的求解框架基礎。

### 11.2 SVD 在系統的角色
- 矩陣條件數、可控 / 可觀 Gramian 的奇異值 → Hankel 奇異值。
- 多變數頻率響應的「奇異值繪圖」(σ-plot)：MIMO Bode 圖。

### 11.3 Riccati 方程
- 連續：$A^\top P+PA-PBR^{-1}B^\top P+Q=0$。
- 離散：相似形式。
- 是 LQR、LQE、$H_\infty$ 控制的核心方程，解法多採 Schur / Hamiltonian 矩陣特徵分解。
