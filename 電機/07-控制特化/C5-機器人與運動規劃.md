# C5 機器人與運動規劃（Robotics & Motion Planning）概論

> 控制特化第 5 科。
> 名詞對照見 [中英名詞對照表 → 機器人](../中英名詞對照表-4.md#機器人)。

## 這科在學什麼

機器人 = 機構（mechanism）+ 感測（sensing）+ 計算（compute）+ 控制（control）+ 規劃（planning）。本科從電機 / 控制視角整合：
1. 運動學與動力學（建模）
2. 軌跡 / 路徑規劃
3. 控制（位置、力、阻抗、視覺）
4. 感測與狀態估測（IMU、視覺、LiDAR）
5. SLAM 與環境感知
6. 機器人系統整合

## 第 1 章　機器人的分類

- **工業手臂（manipulator）**：6 軸 / 7 軸，重複定位、組裝、焊接。
- **協作手臂（cobot）**：UR、TM、Franka，可與人共處。
- **行動機器人（mobile）**：輪式、履帶、四足、雙足、飛行。
- **無人機（UAV / drone）**：四旋翼、固定翼、混合翼。
- **水下 / 太空機器人**。
- **微 / 奈米機器人**（生醫植入）。
- **手術機器人**（da Vinci）。

## 第 2 章　運動學（Kinematics）

### 2.1 座標、姿態、轉換
- 位置 $\mathbf{p}\in\mathbb{R}^3$、姿態 $R\in SO(3)$。
- 表示：旋轉矩陣、軸角、四元數、歐拉角。
- 齊次轉換 $T\in SE(3)$：
$$T=\begin{bmatrix}R&\mathbf{p}\\0&1\end{bmatrix}$$

### 2.2 正運動學（Forward Kinematics）
給定關節角 $\mathbf{q}$ 計算末端執行器位置 / 姿態。
- **DH 參數**（Denavit–Hartenberg）。
- **Product of Exponentials** (POE) 形式：用旋量（screw）+ 指數映射。

### 2.3 逆運動學（Inverse Kinematics）
給定目標位置 / 姿態求 $\mathbf{q}$。
- 解析解：6R 手臂滿足 Pieper 條件可閉式。
- 數值解：Newton / damped least squares、Jacobian 迭代。
- 多解性、奇異點處理。

### 2.4 Jacobian
$$\dot{\mathbf{x}}=J(\mathbf{q})\dot{\mathbf{q}}$$
- $J$ 把關節速度映到末端速度。
- 奇異性 $\det J=0$：失去某方向自由度。
- 微分逆運動學：$\dot{\mathbf{q}}=J^+\dot{\mathbf{x}}$ 用 Moore–Penrose 偽逆。

## 第 3 章　動力學（Dynamics）

### 3.1 Lagrange 形式
$$M(\mathbf{q})\ddot{\mathbf{q}}+C(\mathbf{q},\dot{\mathbf{q}})\dot{\mathbf{q}}+G(\mathbf{q})=\boldsymbol{\tau}+J^\top\mathbf{F}_\text{ext}$$
- $M$：慣性矩陣（對稱正定）。
- $C$：科氏力 + 離心力。
- $G$：重力。
- $\boldsymbol{\tau}$：關節扭矩。
- $\mathbf{F}_\text{ext}$：外力。

### 3.2 Newton–Euler 形式
逐節遞迴計算：適合即時計算動力學項（RNEA 演算法）。

### 3.3 接觸動力學
- 接觸力 = 互補約束（complementarity）。
- 對應四足 / 雙足機器人。
- 模擬：MuJoCo、Drake、Bullet、Isaac Gym。

## 第 4 章　軌跡規劃（Trajectory Planning）

### 4.1 關節空間插值
- 線性、立方多項式、五次多項式（保證連續加速度）。
- B-spline 平滑軌跡。

### 4.2 操作空間軌跡
- 直線 / 圓弧。
- 螺線（SLERP for 旋轉）。

### 4.3 帶限制最佳軌跡
- 加速度、速度、扭矩、能耗、時間最短。
- 用 SQP、IPOPT、CASADI 解。
- TOPP、TOTG 演算法。

### 4.4 動態可行軌跡
- 機器人本身動力學 + 接觸限制。
- 雙足機器人：ZMP、Capture Point、Centroidal Dynamics。

## 第 5 章　路徑規劃（Path Planning）

### 5.1 全域規劃
- **柵格 + Dijkstra / A\***：移動機器人常用。
- **取樣：PRM、RRT、RRT\*、PRM\***：高維、複雜障礙。
- **混合 A\***：結合運動學限制（自駕、機器人）。

### 5.2 局部規劃
- **動態視窗法（DWA）**：在速度空間搜尋下一步。
- **局部勢場（potential field）**：吸引目標 + 排斥障礙。注意局部極小陷阱。
- **MPC 軌跡跟隨**：兼顧追蹤與避障。

### 5.3 約束規劃
- 機械手臂的關節限制、奇異性、自我碰撞。
- CHOMP、STOMP、TrajOpt 等基於最佳化的路徑規劃。

## 第 6 章　控制方法

### 6.1 PID + 重力補償
最基本機械手臂控制。$\boldsymbol{\tau}=K_p(\mathbf{q}_d-\mathbf{q})+K_d(\dot{\mathbf{q}}_d-\dot{\mathbf{q}})+G(\mathbf{q})$。

### 6.2 計算扭矩控制（Computed Torque）
反饋線性化機械臂動力學：
$$\boldsymbol{\tau}=M(\mathbf{q})\mathbf{u}+C\dot{\mathbf{q}}+G$$
其中 $\mathbf{u}=\ddot{\mathbf{q}}_d+K_d(\dot{\mathbf{q}}_d-\dot{\mathbf{q}})+K_p(\mathbf{q}_d-\mathbf{q})$。

### 6.3 阻抗 / 順應控制（Impedance / Admittance）
要機器人對外力有「彈簧 + 阻尼」般的行為：
$$M_d\ddot{\mathbf{e}}+B_d\dot{\mathbf{e}}+K_d\mathbf{e}=-\mathbf{F}_\text{ext}$$
協作機器人、組裝、人機互動關鍵。

### 6.4 力控（Force Control）
直接控制接觸力：混合位置 / 力控、direct force control。

### 6.5 視覺伺服（Visual Servoing）
用相機影像作為回授，控制機器人朝目標移動。
- Image-based（IBVS）vs Position-based（PBVS）。
- 結合深度學習做「pose estimation + 跟隨」。

### 6.6 全身控制（Whole-Body Control, WBC）
高自由度（雙足、四足）：把所有任務（重心、足底、平衡）變成優先順序最佳化（QP），同時滿足。

### 6.7 步態與 MPC
雙足 / 四足機器人：模型預測接觸序列 + 軌跡（如 Boston Dynamics 用 MPC + WBC、ANYmal 用 RL）。

## 第 7 章　感測與狀態估測

### 7.1 編碼器 / 解角器
關節位置、速度量測。

### 7.2 IMU（複習）
加速度 + 陀螺 + 磁力計。
姿態估測：Madgwick、Mahony 濾波器；複雜的：EKF / UKF。

### 7.3 力 / 扭矩感測器
六軸 F/T sensor 裝在腕關節，回授接觸力。

### 7.4 視覺
- 相機（單目、雙目、深度 RGB-D）。
- 演算法：特徵點（SIFT、ORB）、深度學習物件偵測 / 分割。

### 7.5 LiDAR
旋轉 / 固態雷射雷達。產生點雲。
- 機械式（Velodyne）、固態（Livox、Hesai）。

### 7.6 感測融合
- KF / EKF / UKF 結合多感測。
- Factor Graph（GTSAM、Ceres）：現代 SLAM 主流框架。

## 第 8 章　SLAM（Simultaneous Localization and Mapping）

### 8.1 任務
邊建地圖邊定位自己。

### 8.2 分類
- **Visual SLAM**：ORB-SLAM、DSO、Direct Sparse Odometry。
- **LiDAR SLAM**：LOAM、A-LOAM、FAST-LIO。
- **Visual-Inertial（VIO）**：VINS-Mono、Kimera。
- **多感測器**：相機 + IMU + LiDAR + GPS 融合。

### 8.3 核心數學
- 非線性最小平方法 + Sparse Bundle Adjustment。
- 用 Factor Graph 表達與最佳化。

### 8.4 應用
- 自駕車、無人機、AR/VR、家用掃地機器人、配送機器人。

## 第 9 章　軟體與框架

### 9.1 ROS / ROS 2
- 機器人作業系統 / 中介軟體。
- 節點 + 主題 + 服務 + 動作 + 參數。
- ROS 2 用 DDS、提供即時 + 分散式。
- 主流工業 / 學術機器人開發平台。

### 9.2 模擬器
- **Gazebo / Ignition**：通用機器人模擬。
- **PyBullet、MuJoCo、Drake、Isaac Sim / Lab**：物理引擎 + 深度學習。
- 提供 sim-to-real 訓練平台。

### 9.3 學習框架
- 強化學習：Stable Baselines、Tianshou、RLlib。
- 模仿學習、行為克隆、Diffusion Policy。
- LLM-powered 機器人（PaLM-E、RT-2、Open VLA）。

## 第 10 章　工程整合議題

### 10.1 即時性
- 控制迴路（1 kHz）：硬即時，RTOS 或 Linux RT。
- 規劃 / 感知（10–100 Hz）：可放在通用 OS。

### 10.2 安全（Safety）
- ISO 10218（工業機器人）、ISO/TS 15066（協作）。
- 速度限制、力量限制、碰撞偵測、緊急停止。

### 10.3 校正（Calibration）
- 機構參數、相機內外參、IMU 偏置、手眼校正。

### 10.4 嵌入式平台
- MCU（STM32、ARM Cortex-R）做低階控制。
- 應用處理器（Jetson Orin、x86 / Ryzen Embedded）跑 SLAM / 感知 / 規劃。
- FPGA 做客製 I/O 與時序敏感邏輯。

## 第 11 章　典型機器人系統的工作流

```
感測 → 狀態估測 / SLAM → 場景理解（物件偵測 / 分割）→
任務規劃（高階）→ 運動規劃 → 軌跡 / WBC → 控制（即時）→ 致動
                ↑___ feedback ___________|
```

每一階段都對應到上面的章節。本科是「橫向整合」的科目。

## 第 12 章　趨勢

- **學習與規劃融合**：基於模型 RL、Diffusion-based policy、視覺-語言-動作（VLA）模型。
- **大模型（Foundation Models）作機器人控制**：RT-2、Open X-Embodiment、PaLM-E、Helix。
- **觸覺感測陣列**：GelSight、DIGIT、軟質皮膚。
- **可變剛性 / 軟體機器人**：充氣、形狀記憶合金。
- **群體機器人（swarm）**：分散式協同。
- **人機協作 / 服務機器人量產化**：物流、長照、醫療。
