# C4 非線性與自適應控制（Nonlinear & Adaptive Control）概論

> 控制特化第 4 科。
> 名詞對照見 [中英名詞對照表 → 非線性控制](../中英名詞對照表-4.md#非線性控制)。

## 這科在學什麼

線性系統理論 + LQR + $H_\infty$ 處理「線性世界」。實際系統很多是非線性（飽和、摩擦、機械手臂、化工反應器）；模型有時還會慢慢漂（質量變了、磨損了）。本科覆蓋兩件事：
1. **非線性控制**：在非線性下證穩定 / 設計控制器。
2. **自適應控制**：參數未知或變動下，邊運作邊學。

## 第 1 章　為什麼需要非線性控制

線性化技巧（在某點做小擾動分析）有限：
- **行為距工作點較遠**就失效（劇烈動作的機器人）。
- **本質非線性**（飽和、繼電、滑動、滯後）無法線性化。
- **多平衡點 / 極限環**（振盪）只有非線性才能描述。
- **非線性現象**：分叉（bifurcation）、混沌（chaos）。

## 第 2 章　Lyapunov 穩定性

### 2.1 平衡點與穩定性定義
非線性 $\dot{\mathbf{x}}=f(\mathbf{x})$ 有平衡點 $\mathbf{x}_e$ 滿足 $f(\mathbf{x}_e)=0$。

三個穩定性等級：
- **穩定（stable）**：軌跡保持在原點附近。
- **漸近穩定（asymptotic）**：軌跡會趨近於平衡。
- **指數穩定**：以指數速率趨近。

### 2.2 直接法（Lyapunov 第二法）
找一個能量函式 $V(\mathbf{x})$：
1. $V(\mathbf{0})=0,\ V(\mathbf{x})>0\ (\mathbf{x}\ne 0)$。
2. $\dot V(\mathbf{x})\le 0$（沿軌跡單調或不增）。
則平衡點穩定。若 $\dot V<0$ 則漸近穩定。

直覺：能量守恆系統不發散；耗散系統最終回到平衡。

### 2.3 LaSalle 不變原理
$\dot V\le 0$（不嚴格 < 0）時，若軌跡的極限集合是「$\dot V=0$ 的不變子集合」中只有原點，則漸近穩定。
適合處理 $\dot V$ 只 semi-定符號的情況。

### 2.4 ISS（Input-to-State Stability）
有外擾 $\mathbf{u}$ 時，狀態界由初始狀態 + 擾動界決定：
$$\|\mathbf{x}(t)\|\le \beta(\|\mathbf{x}(0)\|,t)+\gamma(\|\mathbf{u}\|_\infty)$$
對串接 / 回授系統穩定性分析很實用。

## 第 3 章　主要非線性控制技術

### 3.1 反饋線性化（Feedback Linearization）
若可寫成「**仿射控制系統**」 $\dot{\mathbf{x}}=f(\mathbf{x})+g(\mathbf{x})\mathbf{u}$，找控制律：
$$\mathbf{u}=\alpha(\mathbf{x})+\beta(\mathbf{x})v$$
使閉迴路在新輸入 $v$ 下變線性。再對線性系統用 LQR / 極點配置。

要求：相對階（relative degree）= $n$（系統可全線性化），否則只能部分線性化（input-output linearization）+ 內動態（zero dynamics）要穩定。

### 3.2 滑動模式控制（Sliding Mode Control, SMC）
1. 設計**滑動面** $s(\mathbf{x})=0$，使在面上時動態自動穩定。
2. 設計控制律驅使軌跡到面上 → 用大增益 + 切換項。
3. 軌跡達面後沿面滑動到原點。

**特性**：對匹配扰動極強健，但**會 chattering（高頻切換）**。
**改良**：邊界層、Super-twisting、Higher-order SMC 平滑切換。

### 3.3 反推法（Backstepping）
逐步把高階非線性系統往內推：先設計虛擬控制 + Lyapunov，再把虛擬控制當作下一階目標。
適合**嚴格回授（strict feedback）** 形式的系統，機器人 / 飛行控制常用。

### 3.4 被動性（Passivity）控制
系統輸入 $u$、輸出 $y$ 滿足 $\int u^\top y\,dt\ge V(\mathbf{x})-V(\mathbf{x}_0)$（能量增加不超過外部輸入功）。
被動性 + 負回授 → 穩定。
**Port-Hamiltonian 系統**：能量化建模 + 被動性控制，機械 / 電路 / 流體統一描述。

### 3.5 描述函數法
非線性元件（飽和、繼電、滯後）替換為等效「振幅相依」的傳遞函數。
用於分析 / 預測極限環（自激振盪）。

## 第 4 章　非線性系統分析工具

### 4.1 相平面（Phase Plane）
2D 系統的軌跡圖。視覺判斷穩定性、極限環、分叉。

### 4.2 線性化
在平衡點 $\mathbf{x}_e$ 線性化得 $\dot{\boldsymbol{\delta}}=A\boldsymbol{\delta}$，$A=\partial f/\partial\mathbf{x}|_{\mathbf{x}_e}$。
- $A$ 特徵值全在左半平面 → 平衡漸近穩定（局部）。
- 有右半平面特徵 → 不穩。
- 純虛軸 → 線性化不能定論，要其他方法。

### 4.3 分叉（Bifurcation）
參數變動時系統的穩定結構發生質變：
- Saddle-node、pitchfork、Hopf（產生極限環）、period-doubling、homoclinic。
- 工程上重要：避免無意中跨越分叉點。

### 4.4 混沌
確定性系統但對初始極敏感，長期不可預測（如雙擺、Lorenz 系統）。
- 控制混沌：OGY 法、延遲反饋。

## 第 5 章　自適應控制

### 5.1 為什麼需要
- 參數變化大（飛機載重變、馬達溫度變）。
- 模型不完整、不易精確得知。
- 想隨環境改善性能。

### 5.2 兩大流派
**(a) 模型參考自適應（MRAC, Model-Reference Adaptive Control）**
給定參考模型 $G_m(s)$，調控制器參數使閉迴路追隨。
- MIT rule：梯度方向更新 $\dot\theta=-\gamma e\dfrac{\partial e}{\partial\theta}$。
- Lyapunov 法：用 Lyapunov 函數證明穩定的同時導出參數律。

**(b) 自我整定（Self-Tuning Regulator, STR）**
線上估參數（用 RLS / LMS）+ 用估值即時設計控制器（如 LQR / pole placement）。

### 5.3 穩定性
自適應控制非線性 → 需要 Lyapunov 證明：
- 在某些假設（持續激勵、SPR 條件）下參數估計收斂、追蹤誤差收斂。

### 5.4 強健自適應
加入 $\sigma$-修改、dead-zone、投影算子防止參數漂移。

### 5.5 適用情境
- 飛行器、衛星姿態：載重隨任務變。
- 馬達 + 機械臂：負載慣量未知。
- 化工製程：催化劑老化。
- 醫療：胰島素泵根據病人反應調整。

## 第 6 章　學習控制（Learning Control）

### 6.1 迭代學習控制（ILC）
針對**重複動作**（如機械手臂裝配、晶圓步進），每一次嘗試從上次誤差學：
$$\mathbf{u}_{k+1}(t)=\mathbf{u}_k(t)+L\mathbf{e}_k(t)$$
經過多次後追蹤精度逼近極限。

### 6.2 重複控制（Repetitive Control, RC）
類似 ILC，但用於連續週期擾動。內模原理：控制器內含「週期訊號的模型」（積分週期內）→ 完美抑制週期擾動。

硬碟伺服、雷射印表機、CNC 加工常用。

## 第 7 章　強化學習與最佳控制的橋

### 7.1 ADP（Approximate Dynamic Programming）
逼近 HJB 解 → 利用神經網路 / 線性表徵。
連接控制工程與機器學習。

### 7.2 Safe RL
- 用控制 barrier function（CBF）強制動作不越界。
- 結合 Lyapunov 穩定性保證。
- 工程應用：人機協作、自駕、機器手臂。

## 第 8 章　常見應用

### 8.1 機械手臂
- 動力學含 Coriolis、慣性矩陣依關節變化 → 強非線性。
- 控制：computed-torque（部分反饋線性化）、impedance control、自適應。

### 8.2 雙足 / 四足機器人
- 接觸動態切換（觸地 / 騰空）→ 混合系統。
- 控制：whole-body control + MPC + 學習。

### 8.3 飛機 / 火箭
- 大攻角 / 大姿態非線性效應。
- 反饋線性化 + 增益排程 + 強健設計組合。

### 8.4 製程 / 化工
- 反應速率含指數（Arrhenius）→ 強非線性。
- MPC / 自適應 + 經驗模型。

### 8.5 電機 / 電力電子
- 馬達磁飽和、雙折變壓器。
- FOC 是反饋線性化的特例。
- 自適應 / 滑動模式無感測控制。

### 8.6 醫療
- 胰島素 / 麻醉劑量：個體差異大、安全嚴格 → 自適應 + 強健 + Safe-RL。

## 第 9 章　非線性 / 自適應 vs 線性 / 強健的取捨

- 線性 + 強健：簡單、可分析、可驗證；不確定範圍大時保守。
- 非線性 / 自適應：性能更佳、能處理本質非線性；但證明難、產業化 / 認證困難。
- 工程實務：80% 用 PID + 增益排程；複雜系統再加 MPC / 強健 / 自適應。

## 與其他科目的銜接

- 工程數學 / 線性代數：Lyapunov、Riccati、LMI。
- 機率 / 統計：自適應參數估計、Kalman、RL。
- 機器學習：神經網路控制、Safe RL。
- 電機機械、電力電子：實際非線性、飽和。
- 機器人（C5）：應用層。
