# 電機機械（Electric Machinery）概論

> 電機核心第 7 科。風格：概論重點式。
> 名詞對照見 [中英名詞對照表 → 電機機械](../中英名詞對照表-2.md#電機機械)。

## 這科在學什麼

把電能與機械能互相轉換的元件：**發電機**（機械 → 電）、**電動機**（電 → 機械）、**變壓器**（電 → 電；不轉但同屬電機機械範疇）。
原理就是電磁學裡的法拉第感應、安培力、馬克士威方程式——但這科把焦點放在「實際做出來的機器怎麼運作、怎麼分析」。

電力工程、馬達控制、再生能源、電動車的核心。

## 第 1 章　基本電磁能量轉換

### 1.1 兩個基本原理
1. **法拉第定律**：磁通變化產生電動勢 → 發電原理。
2. **羅倫茲力 / 安培力**：載流導體在磁場中受力 → 電動原理。

### 1.2 磁路（複習電磁學中）
$$\mathcal{F}=NI,\quad \Phi=\dfrac{\mathcal{F}}{\mathcal{R}},\quad \mathcal{R}=\dfrac{\ell}{\mu A}$$
鐵心高 $\mu_r$ → 集中磁通；空氣間隙磁阻是主導項。

### 1.3 能量平衡
能量 = 電氣輸入 − 機械輸出 − 損耗（銅損 + 鐵損 + 摩擦 + 風損）。
**效率**：實際機器約 85%–98%。

## 第 2 章　變壓器（Transformer）

### 2.1 理想變壓器
- 鐵心 + 兩組線圈（$N_1, N_2$）。
- $V_2/V_1=N_2/N_1$、$I_2/I_1=N_1/N_2$。
- 阻抗折算 $Z_\text{ref}=Z_2(N_1/N_2)^2$。

### 2.2 實際變壓器等效電路
含：
- 一次 / 二次線圈電阻 $R_1, R_2$
- 漏感 $L_{l1}, L_{l2}$
- 激磁分支：$R_c$（鐵損）並聯 $X_m$（激磁電抗）

### 2.3 短路試驗與開路試驗
- **開路試驗**：低壓邊開路、額定電壓激磁高壓邊 → 量鐵損與激磁參數。
- **短路試驗**：把低壓邊短路、由高壓邊加低電壓使電流達額定 → 量銅損與漏阻抗。

### 2.4 效率與電壓調節
- **效率** = 輸出功率 / 輸入功率，最大效率發生在鐵損 = 銅損時。
- **電壓調節率**：滿載 vs 空載端電壓差比。

### 2.5 三相變壓器與接法
- Y–Y、Y–Δ、Δ–Y、Δ–Δ；各種接法有相位差與諧波處理差異。
- 電力系統高壓配電變壓器多用 Δ–Y。

### 2.6 自耦變壓器、儀器用變壓器
- **自耦變壓器**：一二次線圈共用一段繞組，省銅但隔離性差。
- **電流互感器（CT）、電壓互感器（PT）**：量測 / 保護用。

## 第 3 章　旋轉機械的共通觀念

### 3.1 結構
- **定子（stator）**：不動，提供磁場或感應繞組。
- **轉子（rotor）**：旋轉部分。
- **氣隙（air gap）**：定子與轉子間的空氣間隙——磁阻主要來源。
- **整流子 / 換向器（commutator）**：直流機特有，把外部 DC 與內部 AC 互換。
- **電刷（brush）**：滑動接點。

### 3.2 同步速與滑差
同步速：
$$n_s=\frac{60 f}{p}\,\mathrm{rpm}$$
$f$ 是電源頻率，$p$ 是極對數。
**滑差**：$s=\dfrac{n_s-n}{n_s}$，感應馬達特有，描述轉子相對於同步轉場的滯後。

### 3.3 損耗
- **銅損**（$I^2R$）
- **鐵損**（磁滯 + 渦流）
- **機械損**（摩擦、風扇損）
- **雜散損**

## 第 4 章　直流機（DC Machine）

### 4.1 結構
定子產生固定磁場（永磁或激磁繞組），轉子有換向器把感應的 AC 轉成外部 DC。

### 4.2 基本方程
- 反電動勢：$E_a=K\Phi\omega$
- 轉矩：$T=K\Phi I_a$

### 4.3 激磁方式
- **他激（separately excited）**
- **並激（shunt）**：磁場繞組並聯。
- **串激（series）**：磁場繞組串聯，起動轉矩大，速度隨負載變動大。
- **複激（compound）**：兩者組合。

### 4.4 啟動與速控
- 啟動電流大 → 加啟動電阻或軟啟動。
- 速度控制：改電樞電壓（最常用）或改磁通。

DC 馬達已被永磁同步馬達 + 變頻器取代不少，但概念是後面所有馬達的基礎。

## 第 5 章　感應馬達（Induction Motor）

### 5.1 三相感應馬達原理
三相電流產生**旋轉磁場**（以 $n_s$ 旋轉）；轉子（鼠籠或繞線型）感應出電流 → 與磁場互動產生轉矩。轉子必須比同步速慢一點（有滑差）才有感應 → 「**非同步機（asynchronous machine）**」。

### 5.2 等效電路
類似變壓器：定子側 + 轉子側，但轉子的阻抗要除以滑差 $s$：$R_2/s$。

### 5.3 轉矩—速度特性
- 啟動轉矩 → 最大轉矩（pull-out）→ 額定點 → 同步速。
- 設計目標：啟動有足夠扭力、額定點靠近最大效率。

### 5.4 啟動方式
- 直接啟動（DOL）
- 星—三角啟動（Y–Δ）
- 自耦變壓器啟動
- **變頻啟動（VFD）**：現代主流，配合驅動器做平滑啟動。

### 5.5 單相感應馬達
家用電器（風扇、洗衣機）用。需要啟動繞組或電容才能產生旋轉磁場。

感應馬達是**工業界用量最大**的馬達——便宜、堅固、免換向器。

## 第 6 章　同步機（Synchronous Machine）

### 6.1 同步發電機
工業界發電的主力（火力、水力、核能、風力）。轉子由直流激磁（或永磁），轉子轉速 = 同步速，產生三相電。

### 6.2 等效電路與相量圖
- 內部感應電動勢 $E_f$
- 同步電抗 $X_s$
- 外部端電壓 $V_t$
$V_t=E_f-jX_s I_a$

相量圖上的角度（功率角 $\delta$）決定主動功率：
$$P=\frac{V_t E_f}{X_s}\sin\delta$$

### 6.3 同步電動機
- 轉子由直流激磁，鎖在同步轉場上跑（無滑差）。
- 可調功率因數（過激→電容性，欠激→電感性）。常用作功率因數修正。

### 6.4 永磁同步馬達（PMSM）
- 轉子改為永久磁鐵。
- 高效率、高功率密度 → **電動車主流**馬達。
- 需要位置感測 + 變頻器驅動。

## 第 7 章　特殊馬達

- **無刷直流馬達（BLDC）**：永磁 + 電子換向，類似 PMSM 但驅動策略不同。常見於無人機、CPU 風扇、伺服。
- **步進馬達（stepper）**：每脈衝固定角度。開迴路定位用（印表機、3D 印表機）。
- **伺服馬達（servo）**：閉迴路位置控制，工業自動化用。
- **線性馬達**：把旋轉拉直；高速鐵路、磁浮、半導體製程平台。

## 第 8 章　馬達驅動與電力電子（一窺）

- **整流器**：AC → DC。
- **逆變器（inverter）**：DC → AC，用 PWM 產生三相可變頻率 AC。
- **變頻器（VFD）**：整流 + 逆變的組合，用於感應 / 同步馬達速度控制。
- **磁場導向控制（FOC, Field-Oriented Control）**：把三相電流變換到旋轉座標（$d, q$ 軸），可像 DC 馬達一樣獨立控制磁通與轉矩——**電動車、機器手臂的核心控制法**。

## 第 9 章　電力系統概念（一窺）

- 發電 → 升壓變壓器 → 高壓輸電 → 降壓變壓器 → 配電 → 用戶。
- 三相平衡與單相負載、無功補償、保護電驛、短路分析、潮流計算（power flow）。
- 再生能源（太陽能、風電）並網：電力電子是關鍵。
- 智慧電網、微電網。
