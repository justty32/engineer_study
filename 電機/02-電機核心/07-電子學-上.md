## 這科在學什麼

電路學把元件當「黑箱」，給你 R、C、L、源就讓你算。電子學則是要打開盒子：半導體元件（二極體、BJT、MOSFET）內部物理 → 等效電路模型 → 用它們組合出整流器、放大器、振盪器、邏輯閘 → 整個現代電子的根基。

## 第 1 章　半導體物理基礎

### 1.1 能帶論與導體分類
- 固體中電子的能量形成**能帶（band）**。
- 最高被占滿的能帶叫**價帶（valence band）**，最低未填滿（或部分填滿）的叫**導帶（conduction band）**。
- 兩者之間禁止電子停留的區間叫**能隙（band gap）** $E_g$。

| 材料類別 | 能隙 | 特性 |
| --- | --- | --- |
| 導體 | 沒有 / 重疊 | 自由電子多 |
| 絕緣體 | 大（$>3$ eV） | 電子幾乎跳不過 |
| 半導體 | 小（Si: 1.12 eV、Ge: 0.66 eV、GaAs: 1.42 eV） | 介於兩者 |

### 1.2 本徵半導體（intrinsic）
純的 Si、Ge：電子被熱激發越過能隙進入導帶，留下一個**電洞（hole）**。
電子（負）與電洞（正）的數目相等：$n=p=n_i$。
室溫下 Si 的 $n_i\approx 1.5\times 10^{10}\,\mathrm{cm^{-3}}$。

### 1.3 雜質摻雜（doping）—— n 型與 p 型
- **n 型**：摻 V 族（P、As）—— 多一個價電子 → 多了「自由電子」。多數載子是電子。
- **p 型**：摻 III 族（B、Ga）—— 少一個價電子 → 多了「電洞」。多數載子是電洞。

質量作用定律（mass-action law）：$np=n_i^2$（熱平衡下恆成立）。

### 1.4 電流機制
- **漂移電流（drift）**：電場驅動，$J=\sigma E$。電子遷移率 $\mu_n$、電洞遷移率 $\mu_p$。
- **擴散電流（diffusion）**：濃度梯度驅動，$J_n=qD_n\,dn/dx$。

兩者並存，是 PN 接面行為的根本。

## 第 2 章　PN 接面（PN Junction）

### 2.1 形成
把 n 型與 p 型半導體接在一起：
- 電子從 n 側擴散到 p 側、電洞反之 → 接合處留下不可動的施體 / 受體離子。
- 這些離子形成**空乏區（depletion region）**，產生內建電場 → 阻止繼續擴散。
- 平衡時內建電位 $V_0\approx 0.7$ V（Si）。

### 2.2 順向偏壓（forward bias）
$V_D>0$（p 接正、n 接負）：
- 外電場抵消部分內建電場 → 空乏區變窄 → 載子大量擴散過去 → 電流大幅增加。

### 2.3 反向偏壓（reverse bias）
$V_D<0$：
- 空乏區變寬 → 幾乎沒有電流（只有極小的飽和電流 $I_S$）。
- 反向電壓太大會**崩潰（breakdown）**：齊納崩潰或雪崩崩潰。

### 2.4 理想二極體方程
$$I_D=I_S\bigl(e^{V_D/(nV_T)}-1\bigr)$$
- $V_T=kT/q\approx 25.85$ mV（室溫熱電壓）。
- $n$ 是理想因子（1～2）。

直覺：順偏指數成長，反偏接近 $-I_S$。

### 2.5 二極體的模型
| 模型 | 用途 |
| --- | --- |
| 理想開關 | 大致估算 |
| 恆壓降（$V_D=0.7$ V 順偏） | 手算電路 |
| 分段線性 | 含順向電阻 |
| 完整指數 | 精準分析 |
| 小訊號 $r_d=nV_T/I_D$ | 動態行為 |

## 第 3 章　二極體應用電路

### 3.1 整流器（Rectifier）
把 AC 變 DC。
- **半波整流**：只取單邊正電壓。
- **全波整流**（橋式）：四個二極體把兩邊都翻成正。
- 加**濾波電容**：把整流後的脈動電壓平滑成接近 DC，剩餘的**漣波（ripple）**正比於 $1/(RC)$。
- 加穩壓器或穩壓二極體 → 穩定 DC 電源供應。

### 3.2 限幅器（Clipper）／箝位器（Clamper）
- **限幅器**：把超過某電壓的部分切掉。
- **箝位器**：把訊號 DC 準位平移到某固定點（用電容 + 二極體）。

### 3.3 倍壓器（Voltage Multiplier）
電容 + 二極體疊加，輸出可達輸入弦波振幅的 2、3、4 倍。

### 3.4 特殊二極體
- **齊納二極體（Zener）**：故意在反向崩潰區工作 → 穩壓參考。
- **發光二極體（LED）**：順偏發光，能隙決定波長。
- **光二極體（Photodiode）**：光生載子產生電流，光纖通訊接收端的主角。
- **變容二極體（Varactor）**：反向偏壓改變空乏區寬度 → 可變電容；VCO 用。
- **蕭特基二極體（Schottky）**：金屬—半導體接面，順偏壓降低、切換快，常見於高速整流。

### 3.5 邏輯閘（簡介）
最早的二極體—電阻邏輯：用二極體的單向導通做 AND、OR。現代 IC 用電晶體實作，但邏輯函數概念由此而來。

## 第 4 章　半導體製程概覽（一窺）

- 矽晶圓 → 氧化 → 微影（lithography）→ 蝕刻 → 摻雜（離子佈植）→ 沉積 → 金屬連線。
- **製程節點**（process node，例如 7 nm、3 nm）標稱最小特徵尺寸，與**摩爾定律**：每 18–24 個月電晶體密度翻倍。
- 現代 IC 包含數十億個 MOSFET，這套製程把所有原理「白菜化」。

## 與後續的連結

二極體只是開場，真正的主角是電晶體（[電子學中](07-電子學-中.md)）。但 PN 接面是兩個電晶體（BJT、MOSFET）的核心構件——理解 PN 後，BJT 與 MOSFET 都會自然。
