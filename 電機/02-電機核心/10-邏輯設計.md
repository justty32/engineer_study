# 邏輯設計（Logic Design）概論

> 電機核心第 5 科。風格：概論重點式（軟體本科可較精簡）。
> 名詞對照見 [中英名詞對照表 → 邏輯設計](../中英名詞對照表-2.md#邏輯設計)。

## 這科在學什麼

把「0/1」的訊號用閘級電路實作邏輯運算與儲存狀態。組合邏輯（無記憶）+ 序向邏輯（有狀態）構成所有數位系統。內容軟體寫過 Verilog/VHDL 的人會很熟，重點放在硬體層的特性。

## 第 1 章　數值系統

- **二進位（base 2）**：硬體基本。
- **十六進位（base 16）**：壓縮二進位顯示。
- **二進位補數（two's complement）**：負數表示。減法 = 取補後加。
- **BCD**：每個十進位數字用 4 位元。
- **格雷碼（Gray Code）**：相鄰碼只差 1 位元；旋轉編碼器、有限狀態機編碼用。
- **ASCII / Unicode**：字元編碼（軟體已熟）。

## 第 2 章　布林代數（Boolean Algebra）

### 2.1 基本運算
AND、OR、NOT、XOR、NAND、NOR、XNOR。
真值表決定函數。

### 2.2 重要恆等式
- 交換律、結合律、分配律。
- **狄摩根定律**：$\overline{A+B}=\bar A\cdot\bar B$、$\overline{A\cdot B}=\bar A+\bar B$。
- 吸收：$A+AB=A$、$A(A+B)=A$。
- $A+\bar A B=A+B$（共識）。

### 2.3 標準型
- **SOP（積之和）**：$\sum$ minterms。
- **POS（和之積）**：$\prod$ maxterms。
- 任何布林函數都可寫成這兩種。

## 第 3 章　邏輯化簡

### 3.1 卡諾圖（Karnaugh Map）
用幾何排列把相鄰 minterm 圈起來，視覺化化簡。適合 ≤ 5 變數。

### 3.2 Quine–McCluskey 演算法
表格化、可程式化。電子設計自動化（EDA）軟體背後的算法之一。

### 3.3 不關心項（Don't Care）
某些輸入組合不可能發生時可任意指定，幫助化簡。

## 第 4 章　組合邏輯電路

### 4.1 基本閘
AND、OR、NOT、NAND、NOR、XOR；**NAND / NOR 是「通用閘」**，光用其中一種就能組出任何邏輯函數。

### 4.2 重要組合元件
- **多工器（MUX）**：$2^n$ 對 1 選擇，由 $n$ 條選擇線決定。
- **解多工器（DEMUX）**：1 對 $2^n$。
- **編碼器 / 解碼器**：$n$ 對 $2^n$ / $2^n$ 對 $n$。
- **比較器**：判斷 $A>B$、$A=B$、$A<B$。
- **加法器**：
  - 半加器（HA）：兩位元加。
  - 全加器（FA）：含進位。
  - 漣波進位加法器（ripple-carry）：簡單但慢。
  - **超前進位（carry look-ahead）**：以面積換速度。
- **ALU**：算術邏輯單元，CPU 的核心區塊。

### 4.3 競賽與險境（Hazards）
- **靜態險境**：輸入變化時輸出短暫毛刺。
- **動態險境**：輸出多次跳變。
- 解法：加冗餘項或同步電路設計。

## 第 5 章　序向邏輯：基本元件

### 5.1 鎖存器與正反器（Latch vs Flip-Flop）
- **鎖存器（latch）**：位準觸發（level-sensitive）。
- **正反器（flip-flop）**：邊緣觸發（edge-triggered）。同步設計幾乎都用 D flip-flop。

常見類型：
- **SR / SR'**
- **JK**
- **D**（資料）：$Q^+=D$；最常用。
- **T**（toggle）：時脈邊緣翻轉。

### 5.2 時脈與時序
- 時脈是同步電路的「心跳」。
- **建立時間（setup time）**、**保持時間（hold time）**：D 在時脈邊緣前 / 後必須穩定的時間。
- **時脈偏斜（clock skew）**、**時脈抖動（jitter）**：實際時序設計的兩個挑戰。
- **最大時脈頻率** 受最長組合路徑延遲限制。

## 第 6 章　序向邏輯：有限狀態機（FSM）

### 6.1 Moore vs Mealy
- **Moore**：輸出只依當前狀態。
- **Mealy**：輸出依當前狀態 + 輸入；通常較少狀態但對時序更敏感。

### 6.2 設計流程
1. 狀態圖（state diagram）。
2. 狀態表。
3. 狀態編碼（二進位 / 格雷 / one-hot）。
4. 下一狀態邏輯（組合邏輯）+ 狀態暫存器。
5. 輸出邏輯。

### 6.3 計數器與暫存器
- **二進位計數器**、BCD 計數器、模 N 計數器。
- 同步 / 非同步計數器。
- **移位暫存器（shift register）**：串並轉換、序列偵測、PRBS（偽隨機序列）。

## 第 7 章　記憶體（Memory）

- **SRAM**：用正反器，快、不需更新、密度低。
- **DRAM**：用電容儲存，需週期 refresh，密度高，主記憶體用。
- **ROM / PROM / EPROM / EEPROM / Flash**：非揮發儲存。
- **CAM**（內容定址）：用內容找位址，常見於 cache TLB、路由表查詢。

設計要點：位址解碼、字元線 / 位元線、感測放大器、refresh、ECC。

## 第 8 章　可程式邏輯與 HDL

### 8.1 可程式邏輯元件
- **PLD / CPLD**
- **FPGA**（Field Programmable Gate Array）：可重組的查找表（LUT）+ 暫存器矩陣 + 路由 + 嵌入塊（RAM、DSP、I/O）。
- **ASIC**：應用特定積體電路，效能 / 功耗最佳但成本高。

### 8.2 硬體描述語言（HDL）
**Verilog** 與 **VHDL**：用程式語法描述硬體。
- **行為層（behavioral）**：抽象描述功能。
- **暫存器轉移層（RTL）**：每個時脈邊緣的暫存器與運算流動。
- **閘級（gate-level）**：直接列出閘與連線。

設計流程：HDL → 合成（synthesis）→ 佈局繞線 → 時序驗證 → 燒錄 / 製造。

### 8.3 軟體角度的提示
寫過 RTL 與用 testbench 跑模擬 (cocotb、UVM 之類) 對軟體背景人並不難；麻煩在**時序收斂**與**併發思維**——所有邏輯同時動作，不是一行一行跑。

## 第 9 章　數位電路的物理層

- **CMOS 邏輯**：詳見 [電子學-下](07-電子學-下.md)。
- **驅動能力、扇出（fan-out）**、**雜訊裕度（noise margin）**。
- **傳播延遲**（$t_{pd}$）、**轉換時間**（$t_r,t_f$）。
- **功耗**：靜態（漏電）+ 動態（$P=\alpha CV^2 f$）。
- **訊號完整性**：高速時走線變傳輸線——回到電磁學 / 傳輸線。
