## 第 6 章　磁共振成像（MRI）

### 6.1 物理基礎（量子加上經典）
- 質子（特別是水分子的 ¹H）有自旋，置於強磁場 $B_0$ 中：能階分裂，產生**淨磁化** $M$ 沿 $B_0$ 方向（z 軸）。
- 拉莫頻率 $\omega_0=\gamma B_0$，1.5T → 64 MHz、3T → 128 MHz。
- 用 RF 脈衝（在拉莫頻率上）把 $M$ 翻倒進 xy 平面 → 產生可量到的橫向磁化。
- 橫向磁化會以兩種時間常數鬆弛：
  - **T1**：縱向恢復（自旋與晶格交換能量）。
  - **T2**：橫向衰減（自旋之間互相去相干）。
  - **T2\***：含磁場不均勻的有效衰減（更快）。
- 不同組織 T1、T2 不同 → 對比。

### 6.2 空間編碼（核心黑魔法）
- **梯度線圈** $G_x, G_y, G_z$：在 $B_0$ 上加線性梯度 → 不同位置不同拉莫頻率 / 相位。
- **頻率編碼（frequency encoding）**：讀訊號時開 $G_x$，不同 x 位置對應不同頻率。
- **相位編碼（phase encoding）**：在訊號前短暫開 $G_y$，給不同 y 加不同相位。
- **層面選擇（slice selection）**：開 $G_z$ + RF 脈衝（窄頻寬）→ 只激發某一層。

最後得到一張頻率—相位平面上的訊號 = **k 空間（k-space）**。
**重建 = 2D 反傅立葉轉換 k 空間 → 影像**。

### 6.3 脈衝序列（Pulse Sequence）
組合 RF 脈衝 + 梯度開關的時序，產生不同對比：
- **自旋回波（SE）**：對 T1 / T2 加權影像基本款。
- **梯度回波（GRE）**：快速影像；常用 fMRI、MR 血管。
- **EPI（echo-planar imaging）**：一次激發拿完整 k 空間切片，快但偽影多。fMRI、擴散影像。
- **FLAIR**：壓水脈衝，看腦白質病變。
- **STIR**：壓脂訊號。
- **平衡式穩態自由進動（bSSFP）**：心臟電影成像首選。

### 6.4 進階應用
- **fMRI（功能性 MRI）**：BOLD 對比量大腦活動引起的血氧變化。
- **DTI（擴散張量影像）**：量水分子在組織中擴散方向 → 神經纖維走向（tractography）。
- **MRA / MRV**：血管 / 靜脈攝影。
- **MRS（光譜）**：量代謝物（NAA、Cho、Lac）。
- **MR 彈性成像（MRE）**：用外加振動估組織硬度。
- **超低場 MRI（< 0.1T）**：可攜式、便宜，AI 補償解析度不足。

### 6.5 系統工程
- **主磁鐵**：超導（NbTi 線浸液氦）1.5–7T；新一代用無液氦 cryo-cooler。
- **梯度系統**：每軸幾百安培、kV 級驅動，毫秒級切換 → 巨大噪音與機械應力。
- **RF 系統**：發射機 + 多通道接收（8–64 通道線圈陣列）。
- **重建電腦**：GPU 叢集。

### 6.6 安全
- 強磁場 → 金屬物危險（**禁帶**所有鐵磁物入內）。
- 鐵磁植入物（部分舊式 pacemaker）禁忌；現代 MR-conditional 裝置可控條件下進入。
- SAR（specific absorption rate）：RF 沉積能量限制；長序列尤其重要。
- 噪音 > 100 dB，需耳塞。

## 第 7 章　核醫影像（Nuclear Medicine）

### 7.1 原理
注入「放射性追蹤劑（tracer）」進體內，量它發出的伽馬射線 → 估代謝 / 受體分布等**功能性**資訊。

### 7.2 SPECT
- 用 ⁹⁹ᵐTc、¹²³I 等發出單一伽瑪光子的同位素。
- **伽瑪相機**：閃爍體（NaI:Tl）+ PMT 陣列 + 準直器（鉛孔陣）→ 量光子位置。
- 旋轉得多角度投影 → 反演重建斷層影像。

### 7.3 PET
- 用 ¹⁸F、¹¹C、¹⁵O 等正子（positron）發射核種。最常用：**¹⁸F-FDG**（氟代葡萄糖）。
- 正子與電子湮滅 → 兩個 511 keV 光子**反向飛**。
- 環形偵測器（BGO / LSO 閃爍體 + PMT / SiPM）抓**符合事件（coincidence event）**：兩個光子同時擊中對面偵測器 → 知道湮滅點在連線上。
- 把所有事件投影重建（ML-EM、OSEM）→ 影像。

### 7.4 TOF-PET
偵測兩光子的**到達時間差** → 縮小事件位置範圍 → SNR ↑。
要求閃爍體快、SiPM 與 ASIC 時序精度 < 500 ps。

### 7.5 PET/CT、PET/MRI
- **PET/CT**：CT 提供解剖 + 衰減校正，PET 提供代謝功能。腫瘤學標準工具。
- **PET/MRI**：MRI 對軟組織解析度好，但與 PET 同步是工程挑戰。神經影像主用。

### 7.6 影像重建議題
- **衰減校正**：身體不同密度對光子吸收不同，要用 CT / MRI 估補正。
- **散射校正**、**隨機事件校正**。
- **正則化迭代重建（含先驗 / 解剖引導）**。

## 第 8 章　光學與顯微影像

### 8.1 螢光顯微（Fluorescence Microscopy）
- 把目標分子用螢光蛋白 / 染料標記。
- 共焦顯微（confocal）：用針孔濾掉非焦平面 → 高對比 3D 影像。
- **多光子（two-photon）**：同時吸收 2 個低能光子才激發 → 深度穿透 + 點激發。神經科學標準。

### 8.2 超解析顯微
傳統繞射極限 ~ λ/2 ≈ 200 nm。
- **STED**：用受激發射損耗環圈壓縮 PSF。
- **STORM / PALM**：單分子定位累積。
- **SIM（結構照明）**：以光柵照明後重建。
- 諾貝爾化學獎 2014 給超解析。

### 8.3 OCT（光相干斷層）
- 低相干干涉量深度反射剖面。
- 軸向解析度 ~ 5–10 µm。
- 眼科視網膜診斷主力；現代心血管腔內 OCT、皮膚癌早期篩檢。
- **SS-OCT**（swept source）：A-scan 100 kHz–MHz，整顆視網膜秒級掃描。

### 8.4 光聲成像（Photoacoustic）
脈衝雷射照組織 → 局部吸收熱漲冷縮 → 超音波 → 像素位置由 US 偵測器陣列定。結合光學對比與 US 穿透。

### 8.5 內視鏡與微創影像
- CCD / CMOS sensor + LED / 雷射光源。
- **窄頻成像（NBI）**：450 / 540 nm 凸顯黏膜血管。
- **螢光內視鏡**：ICG 注射後紅外影像看膽道、淋巴。

### 8.6 顯微數位病理（Digital Pathology）
- 高解析掃描整張玻片 → AI 模型偵測癌細胞、量化染色強度。
- 雲端會診、教學。

## 第 9 章　影像處理通用工具箱

### 9.1 預處理
- 去雜訊（高斯、雙邊、非局部、BM3D）。
- 對比增強（直方圖均化、CLAHE）。
- 偏場校正（bias field correction，MRI 必備）。

### 9.2 分割（Segmentation）
- 經典：閾值、區域成長、Watershed、Active Contour（Snake）、Level Set、Atlas-based。
- **深度學習**：U-Net 系列、DeepLab、TransUNet。
- 醫學影像常用 Dice / IoU 評估。

### 9.3 配準（Registration）
把同一病人不同時段 / 不同模式（PET ↔ CT、術前 ↔ 術中）影像對齊：
- 剛體（rigid）、仿射（affine）、形變（deformable，B-spline、Demons）。
- 互資訊（MI）為多模態目標函數常用。

### 9.4 體積渲染 / 重建
- 表面渲染（marching cubes）。
- 體渲染（ray casting）。
- VR 手術規劃、3D 列印模型。

### 9.5 量化分析（Radiomics）
- 提取大量影像特徵（形狀、紋理、頻譜）→ 機器學習預測病理。
- 與基因組合稱「影像基因組學」。

## 第 10 章　AI 在生醫影像

### 10.1 任務
- **分類**：病變有無、嚴重度分級。
- **偵測 / 分割**：腫瘤、器官、血管、病灶。
- **影像重建**：低劑量 CT、稀疏角度、快速 MRI（Compressed Sensing + DL）。
- **影像合成**：CT ↔ MRI 轉換、虛擬對比劑。
- **預後預測**：影像 + 臨床資料融合。

### 10.2 主要架構
- CNN（ResNet、U-Net、Mask R-CNN）。
- Transformer（ViT、Swin、SegFormer、MedSAM）。
- 多模態融合（影像 + 文字 + EHR）。
- 自監督預訓練（MAE、SimCLR）。

### 10.3 臨床部署的挑戰
- 跨機構、跨機型的**領域偏移**。
- 罕見類別 / 標註成本高。
- 解釋性（Grad-CAM、Integrated Gradients）。
- 法規（FDA / 歐盟 CE / 台灣 TFDA AI 醫材指引）。
- 模型生命週期管理（Real-World Performance Monitoring）。

## 與其他科目的銜接

- DSP、傅立葉：MRI 重建、超音波、FBP。
- 線性代數：迭代重建、深度學習。
- 半導體 / 感測：探頭、偵測器。
- 機器學習：本科應用大宗。
- 通訊：PACS、HL7 / FHIR、DICOM 網路傳輸。
