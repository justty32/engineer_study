## 第 10 章　類比前端（AFE, Analog Front-End）

### 10.1 標準訊號鏈
```
電極 / 感測器 → 保護電路 → 儀器放大器（IA）→
   類比濾波（高 / 低 / 陷波）→ 可程式增益（PGA）→
   ADC → 數位處理（MCU / DSP）→ 通訊（BLE / Wi-Fi）
```

### 10.2 儀器放大器（IA）
- 三 op-amp 拓撲、單晶 IC（AD8221、INA333、INA188）。
- 重點：**CMRR > 100 dB**、輸入阻抗 > $10^{10}\,\Omega$、輸入雜訊 < 1 nV/√Hz（高解析應用）。
- 接地策略：**Right-Leg Drive**（RLD）—— 把共模偵測反相回病人，把共模壓低 20–40 dB。

### 10.3 高解析 ADC
- 24-bit Sigma-Delta ADC（TI ADS1299、ADS1294）：低雜訊、整合 PGA 與內部緩衝，是 EEG / ECG / EMG 主力。
- 多通道同步取樣：保留時間 / 相位資訊。

### 10.4 雜訊預算
端到端 SNR 要從電極熱雜訊、IA、ADC 各階段累加（與通訊系統的 Friis 公式同理）。常以「µV rms 輸入端等效」為單位。
室內 50/60 Hz 干擾若超過 1 mV pp，CMRR 不足會直接淹掉 µV 級訊號。

### 10.5 動態範圍與血肉訊號的特性
- ECG 直流偏移 $\pm 300$ mV、AC 訊號 mV 級 → AFE 要既能擋偏移又能抓微弱 AC。常用 AC 耦合或數位高通。
- EEG µV 級訊號 + 動作偽影 mV 級 → 寬動態範圍 + 自適應陷波。

## 第 11 章　電氣安全與安規

### 11.1 IEC 60601-1（醫療電氣設備總則）
醫療裝置共通安規。重點：
- **病人漏電流（leakage current）** 限制：正常情形 < 100 µA、故障 < 500 µA。
- **應用部位分類**：
  - **B 型（Body）**：與病人接觸但無導電。
  - **BF 型**：浮動（floating），與地隔離。ECG、SpO₂ 標配。
  - **CF 型（Cardiac Floating）**：可直接接觸心臟，最嚴格。
- **絕緣設計**：以「means of protection（MOP）」累計，B 型 1 MOP、BF/CF 2 MOP。

### 11.2 絕緣手段
- **光耦合（optocoupler）**：訊號穿過光，極好的隔離。
- **變壓器**：電源 + 訊號隔離。
- **電容耦合**：高速、隔離電壓有限。
- **數位隔離器**：iCoupler、CMOS 製程內建變壓器或電容。
- **線性隔離放大器**：類比訊號穿越隔離界面而保持線性。

### 11.3 EMC（IEC 60601-1-2）
- 傳導性 / 輻射性發射（emission）與抗擾性（immunity）。
- 醫療裝置抗擾要求比一般工業嚴格。

### 11.4 軟體與功能安全
- **IEC 62304**：醫療軟體生命週期。
- **IEC 60601-1-10**：閉迴路控制裝置（人工胰臟、呼吸器）。
- **IEC 60601-1-12**：緊急醫療環境（救護車）。

### 11.5 風險管理
- **ISO 14971**：醫材風險管理。Hazard → Cause → Harm 樹。
- 設計階段就要做 FMEA。

## 第 12 章　穿戴式裝置

### 12.1 系統組成
- 感測：PPG、加速度計、皮膚溫度、皮膚電導、ECG 電極（手腕 / 胸帶）。
- AFE + MCU + 儲存 + 通訊（BLE 主流，部分 LTE-M / NB-IoT）。
- 電池：鋰聚合物 50–500 mAh；目標 ≥ 1 週。
- 充電：磁吸觸點或 Qi 無線。

### 12.2 設計挑戰
- **超低功耗 MCU**：µA 級平均電流。
- **感測 duty-cycling**：不需要時關掉感測器（PPG 只在量心率時開）。
- **動態 / 靜態切換**：用 IMU 偵測活動狀態調整取樣率。
- **韌體 OTA**：可遠端更新但要防錯與安全。
- **皮膚相容性**：長期接觸不過敏、不脫膠。

### 12.3 重要產品線索
- Apple Watch：ECG（30 秒、單導程）、血氧、跌倒、心律。
- Fitbit / Pixel Watch：HRV、皮膚溫度、壓力。
- 連續血糖儀（CGM）：Dexcom、Abbott Libre。
- 連續血壓（部分手錶研究中）。
- 智慧戒指：Oura，PPG + 溫度 + IMU。

### 12.4 法規路徑
- 一般「健身手錶」屬消費電子，不需醫材認證。
- 一旦宣稱「診斷 / 監測疾病」就進入醫材分級（美 FDA Class II SaMD、歐盟 MDR、台灣 TFDA）。

## 第 13 章　植入式裝置（Active Implantable Medical Device）

### 13.1 心律調節器（Pacemaker）
- 偵測心跳 → 不足時刺激。
- 鋰碘電池壽命 5–10 年。
- 雙腔（DDD）、頻率適應（rate-responsive）、無線遙測。
- **無導線（leadless）**：Medtronic Micra，整顆放心室內。

### 13.2 植入式去顫器（ICD）
- 偵測致命心律不整 → 高能量電擊。
- 加上 CRT（心臟再同步治療）功能 = CRT-D。

### 13.3 神經調節（Neuromodulation）
- **DBS（深腦刺激）**：帕金森、難治性憂鬱、強迫症。
- **脊髓刺激器（SCS）**：慢性疼痛。
- **迷走神經刺激（VNS）**：難治性癲癇。
- **人工耳蝸**：把聲音轉成電刺激繞過耳蝸。
- **視網膜植入**：早期 Argus II、現在多家研發。

### 13.4 腦機介面（BCI）植入
- Utah array、Neuralink 1024 通道、Synchron stentrode（血管內）。
- 訊號數位化後**無線傳出（perciutaneous 或感應）**。
- 巨大挑戰：長期穩定性（電極阻抗漂移、膠細胞包裹）。

### 13.5 共通工程議題
- **電池 vs 感應充電 vs 能量收集**。
- **無線通訊**：MICS（402–405 MHz）、BLE。
- **生物相容封裝**：鈦罐 + 玻璃饋通；近年柔性電子。
- **無線充電**：磁感應 / 共振。
- **安全認證**：ISO 14708 系列、FDA PMA、CE MDR Class III。

## 第 14 章　常見醫療儀器架構速覽

### 14.1 監護儀（Patient Monitor）
ECG + SpO₂ + NIBP + 體溫 + 呼吸 + IBP 一站式。多參數模組化設計。

### 14.2 心電圖機（ECG Machine）
12 導程靜息 / 動態 Holter（24–48 小時）/ 運動心電（pedaler / treadmill）。

### 14.3 EEG 機
32–256 通道，每通道一個 ADC，所有通道同步取樣。
時鐘樹要精確；多盒（headbox + cabinet）需要 daisy-chain 同步。

### 14.4 超音波（一窺）
壓電陣列 + 高壓脈衝 + 接收前端 + 數位波束成形 → 影像（詳見 [M3](M3-生醫影像.md)）。

### 14.5 呼吸器（Ventilator）
氣流 / 壓力感測 + 比例電磁閥 + 閉迴路控制。**功能安全等級最高**：感測 / 致動冗餘 + 看門狗 + 雙處理器。

### 14.6 麻醉機 / 透析機
類似呼吸器，但加上氣體混合、藥物泵、血液流動 / 透析液控制。**生命維持**等級。

## 第 15 章　通訊與資料層

### 15.1 院內標準
- **DICOM**：醫學影像存檔與傳輸。
- **HL7 / FHIR**：電子病歷與資料交換。
- **IHE profiles**：跨系統互通性。

### 15.2 無線標準
- BLE / BLE 5 + Audio。
- 802.11ax（Wi-Fi 6）醫療環境。
- 私網 4G/5G 醫院內。
- 植入式：MICS、Medical Body Area Network（MBAN）。

### 15.3 雲端與 AI 分析
- 上傳到雲做 AI 分析（心律不整偵測、糖尿病視網膜病變判讀）。
- 隱私（HIPAA、GDPR、台灣個資法）。
- Federated learning：模型在邊緣訓練、只共享參數。

## 第 16 章　穿戴 / 居家照護的工程實踐重點

1. **訊號品質指標（SQI）**：每段訊號附帶可信度標記，避免假警報。
2. **異常通報**：分級警示（提示 / 注意 / 緊急）。
3. **電池與資料協調**：壓力大的演算法（如連續 AI 推論）會搶電。
4. **使用者體驗**：醫材也要好戴、好充、好脫；不舒服的人不會用。
5. **可解釋性**：給醫師看的圖表、給病人看的解釋分層。

## 與其他科目的銜接

- 電子學、電力電子：AFE、隔離電源。
- DSP、機率：訊號處理、品質指標。
- 通訊：穿戴 / 植入無線連接。
- 計算機架構：低功耗 MCU、加速器。
- 機器學習：早期警示、診斷輔助。
