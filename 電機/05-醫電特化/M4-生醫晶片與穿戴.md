# M4 生醫晶片與穿戴式裝置（BioMEMS / Wearables）概論

> 醫電特化第 4 科。
> 名詞對照見 [中英名詞對照表 → 生醫晶片](../中英名詞對照表-3.md#生醫晶片)。

## 這科在學什麼

把半導體與 MEMS 技術用在「**體外生醫感測平台**」與「**體內植入式 / 穿戴式生醫晶片**」上。內容含 BioMEMS 製程、微流控、CMOS 生醫前端 IC、植入晶片無線供電 / 通訊、穿戴系統設計。是 M1（訊號處理）+ M2（感測 / 儀器）的「硬體實作層」。

## 第 1 章　BioMEMS 概念

### 1.1 什麼是 BioMEMS
微機電系統用在生物醫學：感測、致動、流體操控、組織介面、植入。

四大技術支柱：
- **微影 / 微加工**：把感測 / 致動結構縮到 µm 級。
- **微流控**：在 µL–nL 體積內精確操控液體。
- **CMOS 整合**：感測前端電路同晶片做出來。
- **包裝 / 生物相容**：保證在體液中能長期工作。

### 1.2 商業範例
- 連續血糖儀（CGM）。
- 數位 PCR / dPCR（液滴 / 微井陣列）。
- COVID-19 抗原試紙 / 微流控檢測卡匣。
- 微針陣列（microneedle）給藥 / 採血。
- 視網膜植入晶片（Argus II 系列）。
- DNA / RNA 定序晶片（Illumina、Oxford Nanopore）。

## 第 2 章　微加工與材料

### 2.1 主流製程
- **體型微加工**：對 Si 用 KOH / TMAH 各向異性蝕刻、DRIE 深蝕。
- **表面微加工**：在 Si 上沉積 / 蝕刻多層膜形成懸臂、薄膜。
- **軟微影**：PDMS 鑄模製作微流道。
- **3D 列印**：兩光子聚合（two-photon polymerization）可達 µm 級結構。
- **薄膜彈性電子**：把電子做在可延展基板上，貼附皮膚 / 器官。

### 2.2 常用材料
- **Si、SiO₂、Si₃N₄**：基板 + 絕緣。
- **金屬**：Au、Pt（生物相容）、Ti（封裝）。
- **聚合物**：PDMS（微流道）、Parylene（薄膜封裝）、PI（柔性）、PEDOT:PSS（神經電極）。
- **奈米材料**：石墨烯、奈米線、奈米管（高靈敏感測）。

## 第 3 章　微流控（Microfluidics）

### 3.1 流體物理在 µm 尺度
- **雷諾數低（Re < 1）**：層流為主、無湍流。混合靠擴散。
- **表面張力主導**：毛細現象重要。
- **電動驅動**：電泳、介電泳、電滲。

### 3.2 元件
- **微泵 / 微閥**：壓電、熱氣動、磁致動。
- **混合器**：被動（曲折通道、3D 結構）、主動（聲流、磁攪拌）。
- **液滴生成（droplet generator）**：T 形或流動聚焦結構產生單一分散液滴 → 數位 PCR / 數位 ELISA 基礎。
- **分離 / 濃縮**：磁珠、聲音波（acoustofluidics）、介電泳。

### 3.3 應用
- **POCT 卡匣**：把採樣 → 試劑混合 → 反應 → 偵測整合在一片塑膠卡匣裡。
- **器官晶片（Organ-on-a-chip）**：動物實驗替代方案。
- **單細胞分析**：每個液滴包一個細胞做 RNA / DNA 定序。
- **連續血糖 / 連續生化**：皮下微通道採樣。

## 第 4 章　CMOS 生醫前端晶片

### 4.1 為什麼要客製 IC
- 大通道數（EEG/ECoG 32–1000+）。
- 極低雜訊（< 1 µV rms）。
- 極低功耗（穿戴 / 植入電池或無線供電）。
- 整合感測器 + AFE + ADC + 數位 → 單晶片。

### 4.2 經典模組
- **低雜訊放大器（LNA）**：閉迴路 chopper 放大器避免 1/f 雜訊；CMRR > 120 dB。
- **可程式增益 / 可調截止頻率**：適應不同訊號類型。
- **低功耗 ADC**：SAR、Sigma-Delta；常數 µW–數百 µW / 通道。
- **數位處理區塊**：濾波、特徵擷取、壓縮、加密。
- **無線通訊**：BLE、UWB、近場感應。

### 4.3 多通道神經記錄晶片
- Intan RHD2000 系列、Neuropixels（IMEC）、Blackrock Cereplex。
- Neuropixels：384 同時記錄通道 + 960+ 可選通道，整合 ADC，矽探針 1 cm 長。
- 設計挑戰：通道間串擾、時序對齊、大量資料介面（LVDS、Serializer）。

### 4.4 心電 / 生命徵象 SoC
- 整合 ECG + PPG + IMU 介面、低功耗 MCU、藍牙、PMU、心律辨識加速器。
- Maxim MAX8614x、ADI ADuCM355、Renesas SmartBond。

### 4.5 影像感測器（生醫專用）
- **直接 X 光偵測器陣列**：a-Se 與 CMOS TFT。
- **CMOS 影像感測器**內視鏡（5×5 mm 微相機）：膠囊內視鏡（PillCam）。
- **時間相關單光子計數（TCSPC）**：螢光壽命影像、TOF-PET 前端。

## 第 5 章　穿戴式裝置系統設計（複習 + 深入）

### 5.1 SoC 平台
- 低功耗 MCU + BLE：Nordic nRF52/53、Apollo、Dialog DA1469x、STM32WB。
- 部分加上 ML 加速器（Apollo 4 內含 NPU、STM32U5 有 helium）。
- Wear OS 等級平台：Qualcomm Snapdragon W5、Apple S 系列。

### 5.2 演算法 deployment
- 模型壓縮：量化、剪枝、Knowledge Distillation。
- 框架：TFLite Micro、X-CUBE-AI、Edge Impulse、NanoEdge AI。
- 部分模型在邊緣推論（心律分類），部分傳到手機 / 雲端。

### 5.3 電源
- 電池：鋰聚合物 50–500 mAh。
- 能量收集：太陽能、熱電、人體動能（壓電 / 三電發電）— 仍是研究階段。
- 無線充電：磁感應 / 磁共振 / 超聲。

### 5.4 機構與材料
- 防水（IP67/68）、運動可靠度（ASTM）、皮膚相容、防過敏（醫材 ISO 10993）。
- 充電接點 vs 完全無線（無針孔防水更好）。

## 第 6 章　植入式生醫晶片

### 6.1 主要場景
- 心律調節器 / ICD / CRT（複習 [M2 下](M2-生醫感測與儀器-下.md)）。
- 神經調節（DBS、SCS、VNS）。
- 腦機介面（BCI）。
- 人工耳蝸、人工視網膜。
- 智慧型藥物泵（胰島素）、智慧型瓣膜壓力感測（CardioMEMS）。
- 連續血糖（CGM 半植入）。

### 6.2 共通子系統
- **生物相容封裝**：鈦罐 + 玻璃饋通；新一代用柔性聚合物（Parylene、PI）封裝。
- **生物相容電極**：Pt、PtIr、IrOx、PEDOT:PSS、石墨烯（為了低阻抗 + 長期穩定）。
- **記錄 / 刺激 ASIC**。
- **無線通訊**：MICS、ISM、UWB、近場感應；植入式天線設計受人體環境影響大。
- **無線供電**：感應耦合 (~MHz) + 整流 + LDO；或植入電池 + 無線充電（如 Medtronic Activa 系列無電池版）。
- **安全機制**：通訊加密、防 EMI 干擾、MRI 條件相容性。

### 6.3 設計重點
- **熱**：植入物表面溫升不可超過約 2°C，避免組織損傷。
- **長期可靠**：10–15 年無故障（不能說重灌換廠就直接拆）。
- **故障安全**：感測 / 致動冗餘；通訊失聯下仍能基本運作。

## 第 7 章　DNA / 蛋白質定序晶片

### 7.1 高通量定序（NGS）
- Illumina：橋式擴增 + 螢光合成 + 大型 CMOS 感測器陣列。
- 一次數十 GB–TB 級資料。

### 7.2 奈米孔定序（Nanopore）
- 單分子穿過奈米孔產生電流變化 → 推測鹼基。
- Oxford Nanopore MinION：USB 大小、即時定序。
- 適用 COVID 變異株現場快速定序。

### 7.3 數位 PCR / 數位 ELISA
- 微井 / 液滴陣列分割樣本 → 計算「有 / 無」陽性 → 絕對定量。
- 罕見變異偵測（液態切片）、ctDNA、低濃度感染源檢測。

## 第 8 章　生醫晶片的安全與規範

- **FDA**：依風險分類（Class I、II、III）；高風險 PMA、中風險 510(k) / De Novo。
- **MDR / IVDR（歐盟）**：分類差異與台灣 / 美國不同。
- **TFDA**：台灣，依國際標準參照。
- **ISO 10993**：生物相容性試驗。
- **ISO 14708**：植入醫材。
- **IEC 62304**：軟體生命週期。
- **ISO 13485**：品質管理系統。

## 第 9 章　產業 / 研究地景速覽

- **學術圈**：CMU、Stanford、MIT、Caltech、Imperial、IMEC、Tokyo、台灣的台大 / 交大 / 清華 / 陽明交大；生醫工程系與電機系共同推動。
- **公司**：Medtronic、Abbott、Boston Scientific、Johnson & Johnson、Stryker、Edwards、新創（Neuralink、Synchron、Verily、Owlet）。
- **台灣強項**：晶圓代工、封裝、感測器、台大電機 / 醫工系跨領域團隊。

## 第 10 章　跨領域 / 未來方向

- **生物相容軟性電子**：與器官界面無縫整合。
- **AI 加速整合進醫材**：Neural Engine in pacemaker / hearing aid。
- **多模態穿戴**：PPG + EEG + EMG + 呼吸一站式。
- **自驅動 / 能量收集醫材**：免電池植入。
- **數位孿生 + AI 個人化**：模擬病人對藥物 / 刺激的反應。
- **腦機介面（無線、高頻寬）**：Neuralink、Synchron 等臨床試驗中。
