## 這科在學什麼

把人體的物理 / 化學 / 生物量變成電訊號的「感測器」與處理它們的「儀器」。上半部聚焦感測元件本身的原理與分類；下半部處理電氣設計、安規、穿戴 / 植入式裝置。

## 第 1 章　生醫感測器的共同需求

### 1.1 基本指標
- **靈敏度（sensitivity）**：輸出 / 輸入。
- **解析度（resolution）**：能分辨的最小變化。
- **線性度（linearity）**：與真值的偏差。
- **動態範圍**：可量測的最大 / 最小比。
- **頻寬 / 響應時間**。
- **準確度（accuracy）vs 精密度（precision）**：偏差 vs 重複性。

### 1.2 與一般感測器的差異
- **生物相容性**：不可引起發炎 / 免疫反應；長期植入用鈦、PEEK、矽橡膠。
- **無菌與滅菌**：能承受高壓蒸氣、γ 射線、EtO。
- **電氣安全**：與病人接觸的電流要嚴格限制（IEC 60601）。
- **體內環境**：濕、鹽、活細胞會腐蝕；37°C 恆溫；血壓 / 體液壓力。

### 1.3 感測介面四類
1. **電位（potentiometric）**：量電位，幾乎無電流。例：pH 電極、ECG。
2. **電流（amperometric）**：施加固定電位、量電流。例：血氧電極、血糖酶電極。
3. **阻抗（impedance）**：量複阻抗。例：呼吸阻抗描記、體脂計、心輸出量。
4. **光（optical）**：吸收 / 螢光 / 散射 / 反射 / 偏振。例：脈搏血氧、光相干斷層 OCT、活體螢光顯微。

## 第 2 章　電位 / 電化學感測器

### 2.1 離子選擇電極（ISE）
- pH 玻璃電極：H⁺ 跨膜電位 → Nernstian 約 59.16 mV/pH。
- K⁺、Na⁺、Ca²⁺、Cl⁻：對應離子載體。
- 血液氣體分析儀（BGA）：pH + pCO₂（Severinghaus）+ pO₂（Clark）三劍客。

### 2.2 生物 ECG / EMG / EEG 電極
- **Ag/AgCl 凝膠**：可逆電化學介面，雜訊低、極化小。
- **乾電極**：橡膠 / 導電織物 / 金屬針陣列。用於穿戴裝置。
- **電容耦合電極**：透過絕緣層感應，可隔衣量測，但雜訊與動作敏感。

### 2.3 葡萄糖感測器（Clark 型）
- 酶（葡萄糖氧化酶 GOx）催化葡萄糖 + O₂ → 葡糖酸 + H₂O₂。
- 量 O₂ 消耗或 H₂O₂ 產生 → 與葡萄糖濃度成正比。
- **連續血糖儀（CGM）**：皮下植入針型，5 分鐘讀一次，10–14 天更換。
- 醛代電子轉移（second-gen）：用介體（mediator）替代 O₂。
- 第三代：直接電子轉移（DET），無介體。

## 第 3 章　光學感測器

### 3.1 脈搏血氧（Pulse Oximetry）
- 紅光（660 nm）+ 紅外（940 nm）穿過手指，量交流（脈動）/ 直流（穩態）比 R。
- 比較吸收得 SpO₂；通過經驗對照表標定。
- 缺點：低灌流、深膚色 / 指甲、運動偽影；新方向：多波長 + 機器學習。

### 3.2 PPG（Photoplethysmography）
- 與血氧用同樣光感測，但聚焦「波形」：每次心跳的血液脈動。
- 估心率、HRV、呼吸（呼吸調變 PPG）、估血壓（PTT）、估血管彈性、應力。
- 手錶 / 手環全靠這個。

### 3.3 近紅外光譜（NIRS）
- 700–1100 nm 穿透組織數 cm。
- 量血紅素飽和度（局部腦氧、肌氧）。
- fNIRS：功能性 NIRS，可量大腦活動（取代部分 fMRI 用途）。

### 3.4 螢光與光遺傳學
- 螢光蛋白（GFP）、鈣指示劑（GCaMP）標示細胞活動。
- 光遺傳學（optogenetics）：channelrhodopsin 等通道，雷射激發控制神經元。

### 3.5 OCT（光相干斷層）
- 用低相干光的干涉量深度。
- 解析度 1–10 µm，深度 1–2 mm。
- 主流眼科診斷工具（視網膜分層）；心血管腔內 OCT。

### 3.6 拉曼 / 紅外光譜
- 化學特定振動模 → 非標記識別組織成分。
- 手術中即時組織判讀（癌組織 vs 健康）。

## 第 4 章　壓力與機械感測器

### 4.1 血壓
- **直接（侵入）**：動脈線 + strain gauge 感測器。手術 / 加護病房用。
- **間接（非侵入）**：
  - 聽診法（Korotkoff）：傳統聽診。
  - 振盪法：自動電子血壓計主用。
  - 連續無創（finger cuff / PWV / PTT）：研究 / 部分商業。

### 4.2 呼吸
- **流量計**：氣流經阻力產生壓差（pneumotachometer）。
- **胸帶阻抗**：胸部阻抗隨呼吸變化。
- **PPG 呼吸性調變**：穿戴非接觸監測。

### 4.3 顱內壓（ICP）
神經外科加護必要監測，用纖維光學或 strain gauge 壓力感測器。

### 4.4 接觸力 / 觸覺
- 機械手臂 / 假肢用 piezoelectric、capacitive、optical 感測器。
- 觸覺反饋（haptic feedback）：手術機器人、義肢回饋。

## 第 5 章　慣性與運動感測器

### 5.1 加速度計、陀螺儀、磁力計（IMU）
- MEMS 製程，9 軸感測廣泛應用。
- **跌倒偵測**、**步態分析**、**復健監測**、**帕金森震顫量化**。
- 與 GNSS、Wi-Fi、藍牙融合做定位。

### 5.2 智慧手錶 / 手環的整合
- PPG + IMU + 環境光 + 溫度 + 麥克風 → 多模態。
- 訊號融合用 KF / 機器學習。

## 第 6 章　化學與生物感測器（biosensor）

### 6.1 三個構成
1. **生物識別元件**：酶、抗體、核酸（aptamer）、細胞。
2. **轉換元件**：電化學、光、SPR、QCM、SAW。
3. **訊號處理**：放大、判讀。

### 6.2 主流類別
- **酶感測器**：葡萄糖、乳酸、尿素。
- **免疫感測器**：ELISA、SPR（表面電漿共振）、暗場螢光。
- **DNA / RNA 感測器**：核酸雜交、qPCR、isothermal LAMP。
- **CRISPR 感測（SHERLOCK、DETECTR）**：低成本快速病原檢測。

### 6.3 即時診斷（POCT）
- 試紙：尿液試紙、血糖試紙、家用 COVID 抗原。
- 微流控（lab-on-a-chip）：把採樣、混合、分析整合在晶片上。
- 數位 PCR / 數位 ELISA：絕對計量。

## 第 7 章　神經電生理感測器

### 7.1 體外（非侵入）
- 頭皮 EEG。
- MEG（腦磁圖）：超導 SQUID 量極微弱腦磁場，需磁屏蔽室。
- 新興：OPM（光泵磁力計），常溫小型化。

### 7.2 體表 / 半侵入
- ECoG（皮質電圖）：把電極放在大腦皮質表面。手術中 / 癲癇定位用。

### 7.3 侵入式
- 微電極陣列（Utah array）：植入皮質，96–256 通道。
- 矽針電極（Neuropixels）：上千通道。
- 柔性電極（Neuralink、Synchron）：減少免疫反應、長期穩定。
- 應用：腦機介面、義肢控制、視覺恢復（人造視網膜）。

## 第 8 章　影像感測器（與下一科 M3 銜接）

- **超音波探頭**：壓電陣列（PZT、CMUT、PMUT）。
- **X 光偵測器**：閃爍體 + 光感測器（CsI + Si PD），或直接式（a-Se）。
- **CT 感測器**：旋轉式高速 X 光偵測器陣列。
- **MRI 線圈**：RF 接收線圈陣列。
- **PET 偵測器**：閃爍晶體 + PMT / SiPM。

詳細留給 [M3 生醫影像](M3-生醫影像.md)。

## 第 9 章　雜訊與訊號完整性

- **生理雜訊**：肌動、呼吸、心跳交叉污染。
- **電氣雜訊**：50/60 Hz、儀器熱雜訊、靜電。
- **動作偽影**：穿戴最大敵人。
- **共模拒斥**：儀器放大器設計關鍵。
- **接地與遮蔽**：避免地迴路。
