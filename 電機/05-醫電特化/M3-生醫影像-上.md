## 這科在學什麼

把人體內部「看」出來：用聲波、X 光、磁場、放射線、光，配上電子儀器與訊號處理重建出影像。
工程涵蓋：感測器、訊號鏈、影像重建演算法（很多用到 DSP / 傅立葉 / 線代）。

## 第 1 章　共通影像概念

### 1.1 影像品質指標
- **空間解析度**：能分辨多細的構造。
- **對比度**：不同組織灰階差異。
- **訊雜比（SNR）/ 對比雜訊比（CNR）**。
- **時間解析度**：對動態（心跳、血流）的捕捉能力。
- **穿透深度**：可看多深的組織。
- **偽影（artifacts）**：呼吸 / 心動 / 金屬 / 運動。
- **劑量 / 安全**：游離輻射（X、CT、PET）或非游離（US、MRI、光學）。

### 1.2 重建：從量測到影像
大多現代影像不是直接「拍」出來，而是要解一個反問題：
$$\mathbf{y}=A\mathbf{x}+\mathbf{n}$$
$\mathbf{x}$ 是影像、$A$ 是感測模型、$\mathbf{y}$ 是量測、$\mathbf{n}$ 是雜訊。
重建 = 找 $\mathbf{x}$。常見方法：
- **分析法**：濾波反投影（FBP，CT）、傅立葉重建（MRI）、迭代波束成形（US）。
- **迭代法**：ART、SIRT、ML-EM（PET）、IR（CT）。
- **深度學習**：CNN / GAN / Transformer 從量測或粗影像端到端重建。

## 第 2 章　超音波（Ultrasound, US）

### 2.1 物理
- 頻率 1–20 MHz（淺部可達 50 MHz）。
- 聲波在組織中速度 ≈ 1540 m/s。
- 不同組織交界處反射 → 回波。
- **A-mode / B-mode / M-mode / Doppler**：振幅 / 灰階 / 移動 / 速度。

### 2.2 探頭（Probe / Transducer）
- **PZT**（鋯鈦酸鉛）壓電陣列：典型 64–512 元素線陣 / 凸陣 / 相控陣。
- **CMUT / PMUT**：MEMS 製程，可整合電子；近年商品化（Butterfly iQ）。

### 2.3 訊號鏈
```
脈衝產生器 → 高壓開關（保護） → 壓電元素 → 組織
組織反射 → 元素 → 低雜訊放大（TGA / TGC）→ ADC →
波束成形 → 解調 → 對數壓縮 → 掃描轉換 → 顯示
```

### 2.4 波束成形（Beamforming）
- **發射端**：每個元素加適當延遲讓波束聚焦於某點。
- **接收端**：依各元素回波時間差延遲後相加（**Delay-and-Sum**）。
- 數位波束成形（DBF）→ 全平面影像。
- **同調平面波 + 合成**：每秒上千張畫面，看血流、剪切波彈性成像。

### 2.5 都卜勒
血液 / 組織運動 → 回波頻率偏移：
$$\Delta f=\dfrac{2f_0 v\cos\theta}{c}$$
- **顏色都卜勒**：每像素估速度，疊在 B-mode 上。
- **頻譜都卜勒**：某點時間—頻譜圖，量血流波形。
- **能量都卜勒（power Doppler）**：對流量敏感、方向資訊弱。

### 2.6 進階模式
- **剪切波彈性成像（SWE）**：用聲輻射力推組織產生剪切波 → 量速度估硬度。肝纖維化、乳房腫瘤鑑別。
- **微泡造影（CEUS）**：注入微泡，反射訊號強。腫瘤血管、心肌灌流。
- **3D / 4D US**：陣列 + 體素重建。產前檢查、心臟。
- **超快超音波（Ultrafast US）**：平面波取代多次聚焦，幾千 fps，可做超敏血流（神經影像 fUS）。

### 2.7 工程瓶頸
- 高通道數 AFE：512 通道 × 50 MHz ADC（大量資料）。
- 整合 ASIC（128 通道前端）以容入探頭內。
- 波束成形即時運算：GPU / FPGA / 邊緣 AI。
- 攜帶式（手機接探頭）：把波束成形搬到手機 SoC。

## 第 3 章　X 光攝影

### 3.1 物理
- 高速電子轟擊鎢靶 → 制動輻射（連續譜）+ 特徵輻射 → X 光（10–150 keV）。
- 衰減：$I=I_0 e^{-\mu d}$，$\mu$ 是組織線性衰減係數，骨頭高、軟組織低、空氣零。
- 雙能 X 光（DEXA）可估骨密度。

### 3.2 偵測器
- **影像增強管 + CCD**（舊式）：類比影像增強。
- **平板偵測器（FPD）**：
  - **間接式**：閃爍體（CsI）→ 可見光 → a-Si 光電二極體陣列。
  - **直接式**：a-Se 將 X 光直接轉電荷。
- **動態 FPD**：影像增強透視、血管攝影。

### 3.3 對比劑
- 碘（血管、靜脈泌尿）。
- 鋇（腸胃）。

### 3.4 應用
- 一般攝影、胸 X 光（CXR）、骨折評估。
- 透視（fluoroscopy）：即時動態。
- 血管攝影（DSA）：減去前後幀，凸顯血管。
- 乳房攝影（Mammography）：低能量、高解析、含 Tomosynthesis。

## 第 4 章　電腦斷層（CT, Computed Tomography）

### 4.1 原理
旋轉的 X 光源 + 弧形偵測器陣列繞病人轉一圈，量到許多投影 → 用**反演演算法**重建斷層影像。
影像值單位：**HU（Hounsfield Unit）**，水 = 0、空氣 = −1000、骨 ≈ +400 至 +1000。

### 4.2 影像重建
**濾波反投影（FBP）**：
1. 對每個角度的投影做**斜坡濾波（ramp filter）**。
2. 把濾波後投影**反投影**回影像平面。
3. 所有角度疊加得到影像。

數學基礎：**中心切片定理（Fourier slice theorem）** —— 投影的 1D 傅立葉轉換 = 影像 2D 傅立葉轉換沿同方向切片。

**迭代重建（IR）**：建立量測模型 + 雜訊模型，最小化代價函式（含正則項）。低劑量 CT 必備。

**深度學習重建**：U-Net、Transformer 從稀疏角度 / 低劑量原始資料直接出影像。

### 4.3 多排 / 雙源 / 螺旋 CT
- **多排（multi-slice）**：偵測器多排，一次轉拿到多層 → 加速。
- **螺旋 CT**：床移動 + 連續旋轉 → 體積掃描。
- **雙源 / 雙能 CT**：兩個 X 光管不同電壓 → 物質分離（碘 vs 鈣）。

### 4.4 劑量管理
- 自動曝光控制（mA modulation）依體型調整。
- 迭代重建讓劑量降至 1/2–1/4。
- **DLP（劑量長度乘積）**、**CTDI** 是劑量指標。
- 低劑量肺癌篩檢標準（NLST trial）。

### 4.5 進階應用
- **CT 血管攝影（CTA）**：對比劑 + 快速掃描。
- **CT 灌流（perfusion CT）**：動態多次掃描估血流參數，急性中風診斷。
- **能譜 CT**：物質分解 + 虛擬單能 + 金屬偽影抑制。

## 第 5 章　X 光 / CT 共通的工程議題

### 5.1 X 光球管
- 鎢靶旋轉冷卻、kVp 與 mA 控制、聚焦電子束。
- 高熱負荷與壽命管理。

### 5.2 高壓電源
- 100–150 kV 級高壓電源；現代用切換式（chopper + cockcroft-walton + 倍壓網路）。

### 5.3 機架（gantry）與機械
- CT 旋轉每秒 2–4 圈，承受巨大離心力。
- 機架平衡、旋轉接觸（slip ring）傳輸 GB/s 級資料。

### 5.4 影像 PACS
醫院內影像存檔通訊系統。DICOM 標準格式 + Modality Worklist + Storage / Q&R。
