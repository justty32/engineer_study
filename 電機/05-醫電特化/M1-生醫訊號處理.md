# M1 生醫訊號處理（Biomedical Signal Processing）概論

> 醫電特化第 1 科。
> 名詞對照見 [中英名詞對照表 → 生醫訊號處理](../中英名詞對照表-3.md#生醫訊號處理)。

## 這科在學什麼

把人體量到的生理訊號（心電、腦電、肌電、呼吸、血氧……）做雜訊抑制、特徵擷取、辨識診斷。
DSP 學過的觀念（濾波、FFT、自適應、小波）+ 機率統計 + 一點生理學就能做大部分事。

## 第 1 章　常見生醫訊號（速覽生理基礎）

### 1.1 心電圖（ECG / EKG）
- 心肌細胞動作電位的體表投影。
- 頻寬：0.05–150 Hz；振幅 0.5–4 mV。
- 標準 12 導程；穿戴式常用 1–3 導程。
- 主要波形：P 波（心房去極化）、QRS 群（心室去極化）、T 波（心室再極化）。
- 應用：心律不整偵測、心肌梗塞診斷、HRV（心率變異）。

### 1.2 腦電圖（EEG）
- 大腦皮質神經元同步活動的頭皮投影。
- 頻寬：0.5–100 Hz；振幅 10–200 µV（極小！）。
- 標準 10–20 導程系統。
- 頻帶：δ（< 4 Hz，深睡）、θ（4–8，淺睡 / 冥想）、α（8–13，閉眼放鬆）、β（13–30，清醒思考）、γ（> 30，高階認知）。
- 應用：睡眠分期、癲癇偵測、BCI（腦機介面）。

### 1.3 肌電圖（EMG）
- 肌肉收縮產生的電位。
- 頻寬：20–500 Hz；振幅 mV 級。
- 表面 EMG（sEMG）vs 針極 EMG。
- 應用：義肢控制、復健、運動分析。

### 1.4 其他重要訊號
- **PPG（光體積描記法）**：手錶手環的「綠光」量心跳，也可估血氧（PulseOx / SpO₂）、血壓（PWV）。
- **呼吸（呼吸流量、胸帶 impedance）**。
- **EOG（眼電圖）**：眼球運動偵測。
- **GSR / EDA（皮膚電反應）**：交感神經活動。
- **體溫、血壓、血糖（CGM）**。

## 第 2 章　量測與訊號條件（Signal Conditioning）

### 2.1 生醫電極
- **Ag/AgCl 凝膠電極**：黃金標準，但易脫水。
- **乾電極**：穿戴式長期監測。
- **侵入式（針極、植入電極）**：神經科學、植入裝置。

### 2.2 儀器放大器（Instrumentation Amp）
高 CMRR、高輸入阻抗、低雜訊。
- 標準三 op-amp 結構或單晶 IC（INA128、AD8221）。
- 共模電壓 50/60 Hz 抑制是關鍵。

### 2.3 干擾源與對策
- **電源頻率（50/60 Hz）**：陷波濾波器 + 屏蔽 + 共模驅動（right-leg drive in ECG）。
- **動作偽影（motion artifact）**：高通 + 自適應濾波。
- **EMG 干擾到 ECG**：低通 + 形態學濾波。
- **基線漂移（baseline wander）**：高通 0.5 Hz。

### 2.4 ADC 與隔離
- 高解析（16–24 bit Sigma-Delta ADC）。
- **電氣隔離**：保護病人與儀器（IEC 60601 安規）；常用光耦合、變壓器、電容耦合。

## 第 3 章　濾波與雜訊抑制

### 3.1 標準濾波器
ECG 典型：0.5–40 Hz 帶通 + 50/60 Hz 陷波。
注意：低通截止對 QRS 形狀影響大，臨床建議至少 100 Hz 上限以保留高頻細節。

### 3.2 自適應濾波（複習 DSP）
- 心電中除去呼吸 / 動作偽影：LMS 配額外感測器（加速度計）。
- 胎兒 ECG 從母體 ECG 中分離：自適應 + 多通道。

### 3.3 小波去噪
- 經驗域：DWT 分解後對小波係數軟 / 硬閾值。
- 對突波（QRS）保留好，又能去除高頻雜訊。

### 3.4 EMD（經驗模態分解）
非線性 / 非平穩訊號分解成多個 IMF（內模函數）。對 ECG、PPG、HRV 有用。

## 第 4 章　特徵擷取（Feature Extraction）

### 4.1 時間域特徵
- 心率、RR 間期、QRS 寬度、QT 間期、PR 間期。
- 峰值、過零率、能量、形狀因子。

### 4.2 頻率域特徵
- 功率譜密度、各頻帶能量比（如 EEG 的 α/β 比）。
- 主頻、頻譜質心、譜熵。

### 4.3 時頻域特徵
- 短時傅立葉、Wigner-Ville、小波尺度圖。
- 對非平穩訊號（癲癇發作、心律不整）特別有用。

### 4.4 非線性 / 複雜度
- 樣本熵、近似熵、模糊熵。
- Hurst 指數、DFA（細探漂移分析）。
- HRV 分析的標準工具。

## 第 5 章　典型應用

### 5.1 QRS 偵測（Pan–Tompkins 演算法）
經典：帶通 → 微分 → 平方 → 移動平均 → 自適應門檻 → R 峰偵測。
1985 年提出至今仍是入門標準。

### 5.2 心律不整分類
- 規則 / 不規則、寬 / 窄 QRS 分類。
- 心房顫動：RR 不規則、無 P 波（或變成 f 波）。
- 室速 / 室顫：寬 QRS 持續發作 → 急救。
- 現代：CNN / RNN 在端到端 ECG 分類已超人類醫師（AI 心律辨識）。

### 5.3 睡眠分期
EEG + EOG + EMG（多通道睡眠多項生理檢查 PSG）。
分期：Wake、N1、N2、N3（深睡）、REM。
自動分期是熱門 ML 主題。

### 5.4 癲癇偵測
- 譜功率、譜熵、相位同步、小波。
- 即時長期 EEG 監測（病房 / 穿戴）。

### 5.5 腦機介面（BCI）
- **運動想像（motor imagery）**：分類想像動作。
- **P300、SSVEP**：事件相關電位 / 穩態視覺誘發電位。
- 從訓練資料學分類器；現代 BCI 開始用深度學習。

### 5.6 血氧 / 血壓
- **SpO₂**：兩波長（紅光 660 nm + 紅外 940 nm）PPG 比值。
- **連續血壓**：PWV（脈波速度）、PTT（脈波到達時間）；Apple Watch、Huawei Watch 已有 / 開發中。

## 第 6 章　機器學習在生醫的應用

### 6.1 傳統 ML
特徵 + SVM / Random Forest / Boosting。
仍是嵌入式 / 醫療裝置（運算受限）的主力。

### 6.2 深度學習
- 1D CNN：直接吃原始 ECG / PPG。
- RNN / LSTM / Transformer：時間序列。
- 自監督預訓練：BENDR、wav2vec for EEG。
- 缺資料 / 不平衡的對策：合成資料、遷移學習、Self-supervised。

### 6.3 臨床部署的挑戰
- **資料偏差**：訓練集多為健康年輕人 → 老年人 / 病患外推差。
- **可解釋性**：醫師需要理解模型決策（SHAP、Grad-CAM）。
- **法規**：FDA、CE、TFDA 認證、SaMD（軟體即醫材）。
- **連續學習**：模型更新 vs 監管。

## 第 7 章　即時 / 嵌入式生醫處理

### 7.1 平台
- 穿戴：低功耗 MCU（STM32L、Nordic、Apollo），常含 DSP / NN 加速器。
- 居家：手機 / 樹莓派級邊緣裝置。
- 醫院：工作站 / 雲端。

### 7.2 設計挑戰
- 電池壽命 vs 取樣率 / 演算法。
- 與藍牙 / Wi-Fi / NB-IoT 整合。
- 醫療等級 EMC、生物相容性。
- 韌體 OTA + 安全更新。

## 第 8 章　訊號處理與生理的接合一例：HRV

心率變異性（HRV）：相鄰心跳 RR 間期的變化。反映自律神經平衡。
- **時間域指標**：SDNN、RMSSD、pNN50。
- **頻率域指標**：LF（0.04–0.15 Hz）、HF（0.15–0.4 Hz）、LF/HF。
- **非線性**：Poincaré plot、樣本熵。
應用：壓力評估、睡眠品質、心臟病風險、運動恢復。
這個案例同時用到 DSP（FFT、譜估測）、機率統計、非線性分析、機器學習、生理學——是生醫訊號處理的縮影。

## 與其他科目的銜接

- DSP：本科的工具箱。
- 機率與統計：分類、特徵分布。
- 控制：植入裝置（如人工胰臟、心律調節器）的閉迴路控制。
- 通訊：藍牙 / Wi-Fi 傳輸到雲端 / 手機。
- 嵌入式 / 計算機架構：穿戴 MCU 平台。
