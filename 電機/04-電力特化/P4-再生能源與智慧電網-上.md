## 第 1 章　太陽能光電（PV）

### 1.1 元件物理
- 矽 PN 接面吸光產生電子—電洞對 → 光電流。
- 單電池開路電壓 $V_{oc}\approx 0.6$–0.7 V；串聯多片成模組（36–72 cell）。
- 等效電路：電流源 ∥ 二極體 ∥ 並聯電阻，串聯電阻。
- 技術：單晶矽（主流、效率高）、多晶矽、薄膜（CdTe、CIGS）、**鈣鈦礦 / 疊層（tandem）**（研究熱、效率潛力高）。

### 1.2 I–V 與 P–V 特性
- I–V 非線性，存在唯一**最大功率點（MPP）**。
- 輸出受**日照強度**（影響 $I_{sc}$）與**溫度**（升溫降 $V_{oc}$、降效率）影響。

### 1.3 MPPT（最大功率點追蹤）
- **擾動觀察（P&O）**：簡單、最常用，MPP 附近會小幅震盪。
- **增量電導（INC）**：以 $dP/dV=0$（即 $dI/dV=-I/V$）判斷，穩態更穩。
- **模糊 / 神經網路**：複雜光照表現佳。
- **全域 MPPT**：部分陰影使 P–V 多峰，需全域搜尋避免卡在區域峰。

### 1.4 系統架構
- **集中式逆變器**：大功率單機，便宜但陰影 / 失配損失大。
- **串型（string）逆變器**：住宅 / 商業最常見。
- **微逆變器**：每片模組一台，發電最佳化、監控細，成本高。
- **DC 功率優化器**：每片 DC-DC MPPT + 集中逆變。

## 第 2 章　風力發電

### 2.1 空氣動力與功率
$$P=\tfrac12\rho A v^3 C_p(\lambda,\beta)$$
- $\rho$ 空氣密度、$A$ 掃略面積、$v$ 風速（**三次方** → 風速是關鍵）。
- $C_p$ 功率係數，**Betz 極限 = 0.593**（理論上限）。
- $\lambda$ 葉尖速比、$\beta$ 槳距角。

### 2.2 運轉區與控制
- **低於額定風速**：變速運轉維持最佳 $\lambda$ → 最大化 $C_p$（MPPT 式）。
- **高於額定風速**：**變槳（pitch）** 調 $\beta$ 限制功率、保護機組。
- 切入 / 額定 / 切出風速界定運轉範圍。

### 2.3 發電機架構
- **固定速感應機**：直接併網，已淘汰。
- **DFIG 雙饋感應機**：轉子側接約 30% 額定的部分功率變流器，成本低、可變速。
- **PMSG + 全功率變流器**：直驅永磁、免齒輪箱，可靠度高，**海上主流**。

### 2.4 海上風電
單機 10–15 MW、容量因數高；經 **VSC-HVDC** 或交流海纜送陸；基礎（單樁 / 浮式）與運維是關鍵成本。

## 第 3 章　儲能（Energy Storage）

### 3.1 電化學電池
- **鋰離子**：NMC（高能量密度）、**LFP 磷酸鐵鋰**（安全 / 壽命佳、電網與平價 EV 主流）。
- **鈉離子**：成本 / 低溫 / 安全優勢，商業化中。
- **液流電池（釩 / 鋅溴）**：功率與能量解耦、長壽命，適長時儲能。
- **固態電池**：研發中，兼顧能量密度與安全。

### 3.2 機械儲能
- **抽蓄水力**：容量最大（GW、數小時）、最成熟。
- **壓縮空氣（CAES）**、**飛輪**（短時高功率、調頻）。

### 3.3 熱與化學
- **熔鹽**：搭配聚光太陽熱（CSP）。
- **氫**：電解產氫 + 燃料電池 / 燃氣回發，可季節級大容量、低往返效率。

### 3.4 依時間尺度分類
- **功率型（秒~分）**：調頻、平滑波動（飛輪、超級電容、部分電池）。
- **能量型（小時）**：削峰填谷、容量（鋰電）。
- **長時（>10 小時~季節）**：液流、抽蓄、氫——高比例再生的必要拼圖。

### 3.5 電池系統工程
- **BMS（電池管理系統）**：電芯均衡、SOC / SOH 估測、保護（過充 / 過放 / 過溫）。
- PCS（功率轉換系統）併網、熱管理、安全（熱失控防護）。
