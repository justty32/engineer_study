# P4 再生能源與智慧電網（下）：併網、智慧電網、市場與穩定性

> 電力特化第 4 科（下篇）。上篇見 [P4（上）](P4-再生能源與智慧電網-上.md)。
> 名詞對照見 [中英名詞對照表 → 再生能源與智慧電網](../中英名詞對照表-3.md#再生能源與智慧電網)。

> 上篇談再生能源與儲能本體；本篇談它們如何併入電網、被通訊與市場管理，以及由此衍生的穩定性挑戰。

## 第 4 章　電力電子併網

### 4.1 跟網逆變器（Grid-Following, GFL）
- 用 PLL 鎖電網角度，作為「電流源」同步注入有功 / 無功。
- 絕大多數現有再生逆變器屬此型。
- **弱電網**（短路比低）下 PLL 與線路交互易失穩。

### 4.2 構網逆變器（Grid-Forming, GFM）
- 自己「定義」電壓振幅與頻率，行為類似同步機（電壓源）。
- 提供慣性、阻尼、黑啟動、可帶純逆變器電網。
- 策略：下垂控制（droop）、虛擬同步機（VSG）、虛擬慣性。
- 高比例再生與微電網的關鍵技術。

### 4.3 併網規範（Grid Code）
- 頻率 / 電壓範圍內持續運轉。
- **低電壓穿越（LVRT）/ 故障穿越（FRT）**：電網跌落時不脫網並提供無功支撐。
- **防孤島（anti-islanding）**：電網停電時停止輸出，保護維修人員。
- 諧波 / 閃變限制（IEEE 1547、IEC 61727 / 61000）。

### 4.4 多端 HVDC 與直流電網
多個 VSC 站組直流網（海上風場匯集）；**直流斷路器**是瓶頸（DC 無自然過零，滅弧難）。

## 第 5 章　智慧電網（Smart Grid）

### 5.1 三大支柱
1. **量測**：AMI 智慧電表、**PMU 同步相量（30–120 筆/秒）**、SCADA RTU。
2. **通訊**：光纖 + 4G/5G + 專網 + 電力線通訊（PLC）。
3. **計算決策**：EMS / DMS / ADMS、邊緣運算、AI 預測與最佳化。

### 5.2 控制中心系統
- **SCADA**：監視與遠端操作。
- **EMS**：輸電 / 高壓系統營運（含狀態估計、安全分析、AGC、ED）。
- **DMS / ADMS**：配電管理，整合 DER 管理（DERMS）。

### 5.3 同步相量（PMU / WAMS）
跨區同步量測電壓 / 電流相量 → 觀測振盪模態、即時穩定監視、故障 / 振盪定位。

### 5.4 通訊協定
- **IEC 61850**：變電所自動化（GOOSE 高速跳脫、Sampled Values）。
- **DNP3 / Modbus**：傳統 SCADA。
- **OPC UA**：跨層資料模型。
- 資安：IEC 62351、NERC CIP（電網是關鍵基礎設施，攻擊面大）。

## 第 6 章　需量反應與分散資源

### 6.1 需量反應（DR）
以價格或控制訊號引導用戶調整用電：尖峰 / 即時電價（價格型）、補助卸載（誘因型）；對象含智慧家庭、商用 HVAC、工業可中斷負載。

### 6.2 分散資源（DER）與虛擬電廠（VPP）
PV、儲能、EV、汽電共生、可控負載；**聚合商**把大量 DER 整合成可調度的 **VPP**，參與能量與輔助服務市場。

### 6.3 微電網（Microgrid）
DER + 儲能 + 負載組成的小型自治系統，可併網 / 孤島切換；醫院、軍事、偏遠、社區、資料中心採用漸增。需 GFM 逆變器與能量管理（EMS）協調。

## 第 7 章　電動車與電網互動

### 7.1 充電
- AC Level 1/2：1.4–22 kW（家用 / 公共慢充）。
- DC 快充：50–350 kW，重車邁向 MW 級。
- 標準：CCS、NACS（Tesla）、GB/T、CHAdeMO。

### 7.2 V2X
- **V2G**：車向電網回饋（需雙向 OBC + ISO 15118 / OCPP 通訊）。
- **V2H / V2L**：車向家庭 / 負載供電。

### 7.3 對配電的影響
大量同步充電會超載變壓器 / 饋線 → 智慧充電（錯峰調度）、本地儲能、TOU 電價、需量管理因應。

## 第 8 章　電力市場與管制（簡介）

- **批發市場**：日前（DAM）、即時（RTM）、輔助服務（調頻 / 備轉 / 無功 / 黑啟動）。
- **節點邊際電價（LMP）**：含能量 + 壅塞 + 損失。
- **容量市場**：確保長期容量充裕。
- **碳機制**：碳交易、碳邊境調整（CBAM）、再生憑證（台灣 T-REC）。
- 國際組織：歐洲 ENTSO-E、美國 ISO/RTO（CAISO、ERCOT、PJM）。

## 第 9 章　電力電子主導電網的穩定性

### 9.1 慣性下降
同步機提供旋轉慣性；高比例逆變器型電源（IBR）使系統慣性銳減 → **頻率變化率（RoCoF）** 加大、頻率最低點（nadir）更低 → 誤跳與失穩風險。

### 9.2 對策
- **合成 / 虛擬慣性**：風機動能、儲能模擬慣性。
- **構網逆變器（GFM）**：本質提供慣性與電壓支撐。
- **快速頻率響應（FFR）**：儲能毫秒級反應。

### 9.3 新型穩定議題
- 弱電網下逆變器與串補線路 → **次同步振盪（SSO）**、控制交互振盪。
- 多逆變器控制器互相「打架」。
- 廣域協調與本地控制的整合（WAMS + GFM）。

## 第 10 章　永續與政策接面

- **淨零目標**：歐盟 / 台灣 2050、中國 2060。
- 機制：碳定價、上網電價（FIT）、再生配額（RPS）、憑證。
- 工程師角色：把政策落為**技術可行 + 經濟可承受 + 系統穩定**的方案——這正是電力特化整條脈絡（P1 系統、P2 電力電子、P3 馬達 / 變流、P4 再生 / 電網）的交匯點。
