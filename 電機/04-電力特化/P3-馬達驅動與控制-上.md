## 第 1 章　常見受控馬達

### 1.1 轉矩怎麼產生（共通直覺）
所有馬達都靠**定子磁場與轉子磁場（或感應電流）交互作用**生轉矩，最大轉矩發生在兩磁場夾 90°。控制的本質就是「持續把兩磁場維持在最佳夾角」。

### 1.2 各類馬達
- **有刷 DC**：機械換向自動維持 90°，控制最簡單；電刷磨損是壽命瓶頸（玩具、低階）。
- **無刷 DC（BLDC）**：永磁轉子 + 電子換向，梯形反電動勢、六步方波驅動；無人機、風扇、電動工具。
- **永磁同步馬達（PMSM）**：正弦反電動勢、正弦電流驅動，效率 / 功率密度 / 平順度最佳——**電動車主流**。
  - 表面式（SPMSM）：磁阻對稱。
  - 內藏式（IPMSM）：有 saliency，額外磁阻轉矩、利於弱磁。
- **感應馬達（IM）**：定子磁場感應轉子電流產生轉矩，需轉差（slip）；便宜堅固、工業最大宗，變頻 + 向量控制可高效調速。
- **同步磁阻 / PMa-SynRM**：靠轉子磁阻幾何生轉矩，少 / 免稀土，EV 新寵。
- **開關磁阻（SRM）**：結構最簡、無永磁，但轉矩漣波與噪音大。
- **步進馬達**：開迴路定位，3D 列印 / CNC 小型。

## 第 2 章　馬達動態建模（本科的數學核心）

### 2.1 為什麼要 dq 模型
三相 $abc$ 量彼此耦合、時變（隨轉子轉），難以用 PI 控制。把它變換到**隨轉子同步旋轉的 dq 座標**後，穩態下電流成「直流量」，可像直流機一樣解耦控制磁通與轉矩。

### 2.2 Clarke 與 Park 變換
- **Clarke**：$abc\to\alpha\beta$（三相 → 兩相靜止正交）。
- **Park**：$\alpha\beta\to dq$（靜止 → 以轉子角 $\theta$ 旋轉的座標）。

### 2.3 PMSM 的 dq 電壓方程
$$v_d=R i_d+L_d\dfrac{di_d}{dt}-\omega_e L_q i_q$$
$$v_q=R i_q+L_q\dfrac{di_q}{dt}+\omega_e(L_d i_d+\psi_f)$$
其中 $\psi_f$ 為永磁磁鏈，$\omega_e$ 電氣角速度。**交叉項 $\pm\omega_e L i$ 是 d、q 軸耦合**，控制時要去耦補償。

### 2.4 轉矩方程
$$T_e=\dfrac{3}{2}\,p\left[\psi_f i_q+(L_d-L_q)i_d i_q\right]$$
- 第一項：**磁鐵轉矩**（與 $i_q$ 成正比）。
- 第二項：**磁阻轉矩**（IPMSM 才有，$L_d\ne L_q$）。
- $p$ 為極對數。

### 2.5 機械方程
$$J\dfrac{d\omega_m}{dt}=T_e-T_L-B\omega_m$$
$J$ 轉動慣量、$T_L$ 負載轉矩、$B$ 黏滯阻尼。$\omega_e=p\,\omega_m$。

### 2.6 感應馬達模型
類似 dq 但需轉子磁鏈動態與**轉差**；轉矩 $\propto$ 轉子磁通 × 轉矩電流分量。向量控制需估轉子磁通角（間接 FOC 用轉差關係推算）。

## 第 3 章　驅動硬體

- **三相逆變器**（[P2](P2-電力電子-上.md)）+ 必要時 dv/dt 或正弦濾波。
- **電流感測**：相電流或 DC 鏈電流；分流電阻 + 隔離放大、霍爾、磁通閘。
- **位置 / 速度感測**：增量 / 絕對編碼器、解角器（resolver）、霍爾。
- **運算平台**：MCU / DSP / FPGA（TI C2000、ST STM32 G4/H7、Infineon AURIX），需在每個 PWM 週期內完成 ADC→演算法→PWM。

## 第 4 章　控制層級與策略

### 4.1 級聯迴路
```
位置命令 → 位置PI → 速度命令 → 速度PI → 轉矩(電流)命令 → 電流PI → 電壓 → PWM
```
由內而外頻寬遞減：電流環最快（最內）、位置環最慢（最外）。

### 4.2 標量 vs 向量控制
- **V/f 標量控制**：維持 $V/f$ 定值近似定磁通，開迴路、便宜，適合泵 / 風扇等低動態。
- **向量（磁場導向）控制**：分別控磁通與轉矩，動態佳、可做精密伺服。

## 第 5 章　磁場導向控制（FOC）

### 5.1 核心思想
把電流投影到 dq：
- $i_d$：磁通方向分量。
- $i_q$：與磁通垂直 → 直接決定轉矩。
SPMSM 通常令 $i_d=0$（最大轉矩 / 電流），$i_q$ 跟隨轉矩命令。

### 5.2 完整流程（PMSM）
```
速度命令 →(PI)→ i_q*  ┐
i_d* (=0 或 MTPA)      ├→ 電流PI(d,q) + 去耦 → v_dq*
相電流 abc →Clarke→ αβ →Park(θ)→ i_dq ┘
v_dq* →反Park(θ)→ v_αβ* → SVPWM → 三相逆變器 → 馬達
轉子角 θ：編碼器/解角器 或 無感測估測（見下篇）
```

### 5.3 電流環設計
利用零極點抵消：以馬達 $L/R$ 設 PI，使電流閉迴路近似一階、頻寬可達數百 Hz~kHz；加 $\pm\omega_e L i$ 去耦項提升追蹤。

### 5.4 為什麼 FOC 這麼重要
它把交流馬達「直流機化」，使轉矩可瞬時、線性地控制——這是電動車平順加速、伺服 µm 級定位、家電靜音變頻的共同基礎。
