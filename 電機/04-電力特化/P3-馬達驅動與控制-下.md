## 第 6 章　直接轉矩控制（DTC）

### 6.1 想法
不做 dq 電流環與 PWM，直接由**轉矩誤差**與**定子磁鏈誤差**經滯環比較 → 查表選逆變器開關向量。

### 6.2 優缺點
- 優：轉矩響應極快、結構簡單、不需精確馬達參數（除定子電阻）。
- 缺：開關頻率不固定、轉矩 / 磁鏈漣波大、低速磁鏈估測（含 $R_s$ 漂移）困難。
- 折中：**SVM-DTC**（加空間向量調變）固定頻率、降漣波。

ABB 大功率工業變頻長期採用 DTC。

## 第 7 章　弱磁與 MTPA

### 7.1 弱磁（Field Weakening）
高速時反電動勢逼近 DC 母線電壓，電壓飽和限制了電流注入。令 $i_d<0$ 反向削弱磁鏈 → 降反電動勢、可續加速，但可用轉矩下降。決定 EV 的「最高車速」與高速效率。

### 7.2 運轉區間
- **定轉矩區**（基速以下）：MTPA 控制，轉矩飽和受電流限。
- **定功率區**（基速以上）：弱磁，受電壓限。

### 7.3 MTPA（每安培最大轉矩）
- SPMSM：$i_d=0$ 即最佳。
- IPMSM：因磁阻轉矩，最佳 $(i_d,i_q)$ 不在 90°，需沿 MTPA 軌跡計算 / 查表，兼顧效率與少稀土設計。

## 第 8 章　無感測控制（Sensorless）

### 8.1 動機
省去編碼器 / 解角器 → 降成本、縮體積、提升惡劣環境可靠度。

### 8.2 中高速法
- **反電動勢估測**：由電壓 / 電流模型估反 EMF → 推角度與轉速。
- **磁鏈 / 擴展 EMF 觀測器**（IPMSM）、滑動模式觀測器、龍伯格觀測器、EKF。

### 8.3 低速 / 零速法
反 EMF 趨近 0 → 改用**高頻訊號注入**：注入高頻電壓，量測電流響應的磁阻不對稱（saliency）反推角度，需 IPMSM 有凸極性。

### 8.4 啟動
零速啟動最難：常用「對齊 → 強迫換向 → 無感測接管」三段，或凸極追蹤的混合策略。

## 第 9 章　控制器設計細節

### 9.1 採樣率
- 電流環：10–20 kHz（與 PWM 同步），中斷內 < ~50 µs 完成 ADC→FOC→PWM。
- 速度環：1–4 kHz；位置環：≤ 1 kHz。頻寬層層分離（每層約差 5–10 倍）。

### 9.2 防積分飽和與限幅
- PI 積分器需 anti-windup。
- $v_{dq}$ 命令限制在可實現電壓圓內（$\le V_{dc}/\sqrt3$），超出時優先保 $v_q$（轉矩）。

### 9.3 去耦與前饋
補償 $\pm\omega_e L i$ 交叉耦合與反電動勢前饋，改善動態與解耦。

### 9.4 死區補償
逆變器死區造成輸出電壓誤差，低速 / 低調變比時轉矩漣波明顯，需建模補償（依電流極性修正）。

## 第 10 章　位置 / 速度量測

- **增量編碼器**：A/B/Z，解析度 = 線數 × 4；需算速度（差分 + 濾波 / 觀測器）。
- **絕對編碼器**：上電即知絕對位置；單圈 / 多圈。
- **解角器（resolver）**：類比、耐高溫振動，需 RDC 解碼。
- **霍爾**：低成本，BLDC 六步換相。
- **速度估測**：後向差分、M/T 法、Kalman / Luenberger 觀測器降量化雜訊。

## 第 11 章　驅動系統工程議題

### 11.1 EMI 與長纜效應
高 $dv/dt$ 切換產生寬頻干擾；長電纜的傳輸線反射可使馬達端電壓達兩倍 → 加 dv/dt 濾波器 / 端接、共模扼流。

### 11.2 軸承電蝕（Bearing Current）
共模電壓經軸—軸承寄生電容放電造成點蝕 → 絕緣軸承、接地電刷、共模濾波。

### 11.3 散熱
功率模組與馬達繞組皆需散熱；EV 用油冷 / 水冷、hairpin 繞組提升銅占比與散熱。

### 11.4 功能安全
ISO 26262（車用 ASIL）、IEC 61508、IEC 61800-5-2（如 STO 安全轉矩關閉）；要冗餘感測、診斷覆蓋率、FMEA、安全狀態。

## 第 12 章　應用速覽

- **EV 牽引**：PMSM / PMa-SynRM + 800 V SiC 逆變器 + FOC + MTPA + 弱磁 + 主動阻尼。
- **工業伺服 / 機械手臂**：6–7 軸高解析編碼器，µm 定位、高頻寬。
- **家電**：變頻冷氣壓縮機、洗衣機（PMSM + FOC）、高速吹風機（>100k rpm）。
- **風 / 水力**：DFIG / PMSG 變速發電。
- **機器人 / 無人機**：BLDC + ESC，重功率密度。

## 與其他科目的銜接

- 電機機械、電力電子、控制系統：三大基礎。
- 數位信號處理：觀測器、編碼器訊號、速度估測。
- 嵌入式 / 通訊：CAN / EtherCAT / PROFINET 與上位機。
- 機器學習：無感測啟動、預測性維護、參數自整定。
