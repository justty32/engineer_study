## 第 8 章　故障分析（Fault Analysis）

### 8.1 故障型態與機率
| 故障 | 對稱性 | 比例 | 嚴重度 |
| --- | --- | --- | --- |
| 單相對地（SLG） | 不對稱 | ~70% | 較輕～中 |
| 線對線（LL） | 不對稱 | ~15% | 中 |
| 雙線對地（DLG） | 不對稱 | ~10% | 中～重 |
| 三相（3φ） | 對稱 | ~5% | 最重（設備定容依據） |

### 8.2 用序網路算故障電流
把元件的正、負、零序網路接到故障點，再依故障型態連接三個序網路：
- **三相對稱**：只用正序網路，$I_f=\dfrac{V_f}{Z_1}$。
- **單相對地**：正、負、零序**串聯**，$I_a=\dfrac{3V_f}{Z_1+Z_2+Z_0}$。
- **線對線**：正、負序**串聯**（不含零序）。
- **雙線對地**：正序串（負序並零序）。

### 8.3 故障計算的用途
- **斷路器遮斷容量**：須大於最大故障電流（短路 MVA）。
- **保護電驛整定**：判斷與分段。
- **設備機械 / 熱應力**：母線、變壓器繞組能否承受。
- **次暫態 / 暫態 / 穩態電抗**（$X_d''<X_d'<X_d$）：決定不同時間尺度的故障電流大小。

## 第 9 章　保護電驛（Protective Relaying）

### 9.1 目標與哲學
**選擇性（只切故障段）+ 速動性 + 靈敏度 + 可靠性**。設主保護 + 後備保護雙層。

### 9.2 量測互感器
- **CT（電流互感器）**：把大電流降到 1 / 5 A 級；要注意飽和。
- **PT / CVT（電壓互感器）**：把高壓降到 110 V 級。

### 9.3 主要電驛類型
- **過電流（50 瞬時 / 51 反時限）**：最基本，配合反時限曲線做時間分級。
- **方向性過電流（67）**：判斷故障方向，環網 / 雙電源必需。
- **差動（87）**：比較保護區進出電流，差值動作；變壓器、發電機、母線的主保護，區內極靈敏、區外不動。
- **距離 / 阻抗（21）**：量測視在阻抗推估故障距離，分 Zone 1/2/3 階梯式；輸電線主保護。
- **低頻 / 低壓（81/27）**：系統性保護，觸發減載。

### 9.4 協調（Coordination）
- 時間 / 電流分級，使最靠近故障的保護先動，上游當後備。
- 距離電驛的 Zone 配合 + 通訊（pilot protection）達到全線速動。
- **IEC 61850**：變電所數位化通訊標準，GOOSE 訊息實現電驛間高速互鎖。

## 第 10 章　穩定度（Stability）

### 10.1 三類穩定度
- **轉子角穩定**：發電機保持同步的能力。
  - **暫態穩定**：大擾動（短路、跳機）後第一擺是否失步。
  - **小訊號穩定**：小擾動下的振盪模態是否衰減（區域間振盪）。
- **電壓穩定**：重載 / 無功不足時電壓能否維持，避免電壓崩潰。
- **頻率穩定**：發電 / 負載失衡後頻率動態。

### 10.2 擺動方程式（Swing Equation）
$$\dfrac{2H}{\omega_s}\dfrac{d^2\delta}{dt^2}=P_m-P_e,\qquad P_e=\dfrac{E\,V}{X}\sin\delta$$
$H$ 慣性常數（儲存動能 / 額定容量），$\delta$ 轉子角。功率—角度關係是 $\sin\delta$ 形。

### 10.3 等面積準則（Equal-Area Criterion）
單機對無窮匯流排系統，比較故障期間的**加速面積**與故障清除後的**減速面積**：
- 減速面積 ≥ 加速面積 → 暫態穩定。
- **臨界清除角 / 臨界清除時間**：保護必須在此之前切除故障，否則失步。
- 推論：**快速保護**對暫態穩定至關重要。

### 10.4 電壓穩定與 PV 曲線
負載增加 → 受端電壓沿 PV（鼻形）曲線下滑，到「鼻尖」後無解 → 電壓崩潰。無功儲備、電容 / SVC / STATCOM、避免重載長線是關鍵。

### 10.5 改善手段
- 高速勵磁 + **電力系統穩定器（PSS）** 阻尼振盪。
- **FACTS**：SVC、STATCOM（並聯無功）、TCSC（串聯補償）、UPFC（綜合潮流控制）。
- HVDC 調變阻尼、廣域量測（WAMS / PMU）回授控制。

## 第 11 章　頻率與電壓 / 無功控制

### 11.1 頻率—實功（P–f）
- **一次調頻（droop）**：調速器依轉速下垂特性 $\Delta P=-\dfrac1R\Delta f$，秒級、各機分攤。
- **二次調頻（AGC / LFC）**：中央控制把頻率與聯絡線潮流調回設定（消除穩態誤差），分鐘級。
- **慣性反應**：擾動瞬間由旋轉機動能即時頂住頻率變化率（RoCoF）。

### 11.2 電壓—無功（Q–V）
就地平衡為原則（無功不宜長距傳輸）：發電機勵磁（AVR）、電容 / 電抗器、SVC / STATCOM、變壓器 OLTC。

## 第 12 章　經濟調度與電力市場

### 12.1 經濟調度（Economic Dispatch, ED）
在滿足負載下最小化總燃料成本。最優條件：各機**增量成本（λ）相等**（含網損則乘罰因子）。用 λ 迭代、二次規劃、內點法求解。

### 12.2 機組排程（Unit Commitment, UC）
更上層：決定**哪些機組何時開 / 關**（含啟停成本、最小開 / 停機時間、爬坡率），混合整數規劃（MILP）。

### 12.3 電力市場
- 日前 / 即時市場、競價、出清。
- **節點邊際電價（LMP）**：反映能量 + 壅塞 + 損失，依位置不同。
- 輔助服務市場：調頻、備轉、無功、黑啟動。

## 第 13 章　可靠度與安全性

- **N-1 準則**：任一元件失效系統仍安全運轉；規劃與即時安全分析的底線。
- **連鎖故障**：過載 → 跳脫 → 其他線更過載 → 雪崩式停電（2003 美加、2021 德州事件）。
- **防護**：自動低頻 / 低壓減載（UFLS / UVLS）、主動孤島、系統完整性保護方案（SIPS / RAS）。
- **黑啟動**：全黑後由水力 / 燃氣等自啟機組逐步復電。

## 第 14 章　現代電網

### 14.1 智慧電網（Smart Grid）
雙向通訊與感測：AMI 智慧電表、SCADA / EMS、**PMU 同步相量（WAMS）**、需求反應（DR）、配電自動化。

### 14.2 高比例再生能源的挑戰
- 風 / 光**間歇 + 逆變器型（IBR）**，無同步機慣性 → 慣性與頻率穩定下降（RoCoF 變大）。
- 對策：**虛擬 / 合成慣性**、**Grid-forming 逆變器**、快速頻率反應（FFR）、儲能。
- 短路容量下降、保護整定與穩定性都要重新設計。

### 14.3 儲能（ESS）
鋰電、液流、抽蓄、壓縮空氣、飛輪、氫。用途：削峰填谷、調頻、備轉、平滑再生、孤島。

### 14.4 分散式能源與微電網
屋頂光電、社區儲能、EV（V2G）；微電網可併網 / 離網切換，提升韌性。

### 14.5 趨勢
數位分身、AI 負載 / 再生預測與故障診斷、HVDC 多端直流電網、直流配電（資料中心 / 船舶）。
