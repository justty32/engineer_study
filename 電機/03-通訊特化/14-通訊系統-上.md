## 這科在學什麼

通訊系統的本質是：**把訊息從一端送到另一端，對抗距離、雜訊、頻寬限制**。
第一階段的數學（傅立葉、機率）+ 第二階段的訊號 / 電磁工具，到這裡組合起來，回答幾個基本問題：
1. 為什麼要「調變」？怎麼調？
2. 雜訊對通訊性能的影響怎麼量化？
3. 接收器怎麼最佳化？

本科主要走**類比通訊**（AM、FM、PM），把通訊基礎概念建立起來；下一科「數位通訊」再用同樣框架重做數位版本。

## 第 1 章　通訊系統的方塊圖

```
訊息源 → 訊源編碼 → 通道編碼 → 調變器 ──→ [通道：雜訊、衰落、干擾] ──→
        解調器 → 通道解碼 → 訊源解碼 → 訊息接收
```

各區塊：
- **訊源編碼**：壓縮（如 MP3、JPEG）。
- **通道編碼**：加冗餘以對抗雜訊（後續資訊理論講）。
- **調變**：把基頻訊號搬到適合通道的頻段。
- **通道**：物理媒介——空氣、銅線、光纖。
- **解調與解碼**：反向操作。

類比通訊系統大多只走「調變 → 通道 → 解調」這一段。

## 第 2 章　為什麼要調變

訊號（語音、影像）本來在低頻（基頻、baseband）。直接傳遞會有：
- **天線尺寸**問題：天線需要 $\sim\lambda/4$ 長，低頻 → 巨大天線。
- **多人共用頻譜**：不調變的話大家都擠在低頻。
- **通道適配**：通道（電纜、空氣）對某些頻段傳得好。
- **雜訊抑制**：某些調變方式有抗噪優勢。

**調變（modulation）** = 用基頻訊號 $m(t)$ 去改變高頻載波（carrier）$\cos(\omega_c t)$ 的某個參數（振幅 / 頻率 / 相位）。

## 第 3 章　振幅調變（Amplitude Modulation, AM）

### 3.1 雙邊帶大載波 AM（DSB-LC）
$$s(t)=[A_c+m(t)]\cos\omega_c t$$
$A_c$ 大到 $A_c+m(t)>0$ 恆成立 → 包絡 = 訊息。

頻譜：載波 + 兩個對稱邊帶。$m(t)$ 頻寬 $W$，AM 訊號頻寬 $2W$。

**調變指數** $\mu=\dfrac{m_\text{max}}{A_c}\le 1$。$\mu>1$ → 包絡失真（過調變）。

**功率**：載波佔大部分（不傳訊息），效率最高約 33%（$\mu=1$、單音調變時）。**簡單但浪費**——好處是接收端只要包絡檢測，極便宜（電晶體收音機）。

### 3.2 雙邊帶抑制載波（DSB-SC）
$$s(t)=m(t)\cos\omega_c t$$
省掉載波，功率全用於傳訊息。但接收端要本地振盪器**相位 / 頻率同步**（同步解調 / coherent demodulation）。

### 3.3 單邊帶（SSB）
雙邊帶兩邊資訊重複，留一個就好：
$$s_\text{SSB}(t)=m(t)\cos\omega_c t\mp\hat{m}(t)\sin\omega_c t$$
$\hat{m}$ 是希爾伯特變換。
**頻寬只要 $W$**（DSB 的一半），功率亦集中。語音通訊（業餘無線電 HF 頻段）標準。

### 3.4 殘留邊帶（VSB）
類比電視訊號早期用：保留一個邊帶 + 另一邊帶的小部分（讓 DC / 低頻能透過）。

### 3.5 解調方式

| 調變 | 解調方法 |
| --- | --- |
| DSB-LC（AM） | 包絡檢測（簡單，二極體 + RC） |
| DSB-SC | 同步解調（要本地相位） |
| SSB | 同步解調 |
| VSB | 同步解調 + 適當濾波 |

**同步解調**：把訊號乘以 $\cos\omega_c t$ 再低通：
$$x(t)\cos\omega_c t\to \tfrac{1}{2}m(t)+\text{高頻分量}\xrightarrow{\text{LPF}}\tfrac{1}{2}m(t)$$

接收端的本地振盪器要與發射端相位同步——若有相位誤差 $\phi$，輸出乘上 $\cos\phi$。所以實務需要 **PLL**（鎖相迴路）做頻率 / 相位追蹤。

## 第 4 章　角度調變（Angle Modulation：FM 與 PM）

### 4.1 一般形式
$$s(t)=A_c\cos[\omega_c t+\phi(t)]$$
- **PM（相位調變）**：$\phi(t)=k_p m(t)$
- **FM（頻率調變）**：$\dfrac{d\phi}{dt}=k_f m(t)$，即 $\phi(t)=k_f\int m(\tau)\,d\tau$

兩者的瞬時頻率：$\omega_i(t)=\omega_c+\dfrac{d\phi}{dt}$。

### 4.2 窄頻 FM vs 寬頻 FM
- **窄頻**：相位偏移 $\Delta\phi$ 很小，頻譜近似 AM。
- **寬頻**：頻譜複雜，但抗噪能力強。

### 4.3 Carson 法則
FM 訊號的有效頻寬：
$$B\approx 2(\Delta f+W)$$
$\Delta f$ 是最大頻率偏移，$W$ 是 $m(t)$ 頻寬。
**調變指數** $\beta=\Delta f/W$；$\beta\ll 1$ → 窄頻、$\beta\gg 1$ → 寬頻。

例：商用 FM 廣播 $\Delta f=75$ kHz、$W=15$ kHz → $B\approx 180$ kHz。每台電台佔 200 kHz 頻段。

### 4.4 FM 的抗噪優勢
高 SNR 時，FM 接收 SNR 比 AM 高約 $3\beta^2$ 倍，$\beta$ 越大越好——但頻寬代價也越大。這是**頻寬與雜訊的權衡**。

### 4.5 FM 解調
- **鑑頻器（discriminator）**：把瞬時頻率變化轉為振幅變化，再包絡檢測。
- **鎖相迴路 PLL**：高品質 FM 解調的主流。

### 4.6 預加重 / 去加重
FM 的雜訊功率譜在高頻較大，**發射端先加重高頻、接收端去加重**，等效提高 SNR。FM 廣播標準作法。

## 第 5 章　多工（Multiplexing）

把多通道訊號擠進同一通道：
- **FDM（頻分多工）**：每路訊號占一段頻譜。類比廣播、舊式電話。
- **TDM（時分多工）**：每路訊號占一段時間。數位通訊主流。
- **CDM / CDMA（碼分多工）**：用正交編碼共享頻譜。3G 標誌。
- **OFDM（正交分頻多工）**：把高速資料分到多個子載波，每個低速。4G、5G、Wi-Fi、DVB-T 都用。

OFDM 的數學就是 FFT / IFFT——數位通訊的精華應用。

## 第 6 章　超外差接收機（Superheterodyne Receiver）

幾乎所有無線接收機的基本架構：
```
天線 → RF 帶通 → 混頻器（× LO）→ IF 帶通 → IF 放大 → 解調 → 基頻處理
```
- **混頻**到固定的 IF（中頻）：好處是後級濾波 / 放大都在固定頻段做，便宜又高選擇性。
- LO 與 RF 頻差固定為 IF。
- **影像頻率（image）** 問題：與 RF 對稱的另一頻段也會被搬到 IF；需 RF 預選器壓掉。

## 第 7 章　脈衝調變（Pulse Modulation）—— 邁向數位

把連續訊號離散化：
- **PAM（脈衝幅度調變）**：振幅 = 樣本值。
- **PWM（脈衝寬度調變）**：寬度 = 樣本值。電源、馬達驅動也用。
- **PPM（脈衝位置調變）**：位置 = 樣本值。
- **PCM（脈衝碼調變）**：先取樣 → 量化 → 編碼成位元。**這是數位通訊的入口**。
