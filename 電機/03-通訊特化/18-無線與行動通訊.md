# 無線與行動通訊（Wireless and Mobile Communications）概論

> 通訊特化第 5 科。
> 名詞對照見 [中英名詞對照表 → 無線與行動通訊](../中英名詞對照表-2.md#無線與行動通訊)。

## 這科在學什麼

把通訊理論放到**無線環境**：傳播、多徑、衰落、行動性、多用戶共享頻譜，並架構成「行動通訊系統」（蜂巢網路）。整個 GSM → 5G → 6G 演進是這科的故事。

## 第 1 章　無線傳播與通道

### 1.1 三尺度的路徑損失
1. **大尺度路徑損失（path loss）**：距離 $d$ 平均接收功率。自由空間 $L\propto d^2$；實際環境 $L\propto d^n$，$n=2\sim 5$。
2. **陰影衰落（shadowing）**：障礙物造成的對數常態變化（log-normal），標準差 $\sigma_\text{sh}$ 約 6–10 dB。
3. **小尺度衰落（multipath fading）**：多徑加總的瞬時起伏。

### 1.2 多徑通道參數
- **延遲擴展（delay spread）** $\tau_d$：最早與最晚徑的時間差。
- **相干頻寬** $B_c\sim 1/\tau_d$：通道頻譜「平坦」的範圍。
- **都卜勒擴展** $f_d=v/\lambda$：移動造成的頻譜寬化。
- **相干時間** $T_c\sim 1/f_d$：通道幾乎不變的時間。

四種衰落分類：
| 對訊號 | 對應條件 |
| --- | --- |
| 平坦衰落 | $B_s<B_c$ |
| 頻率選擇 | $B_s>B_c$ |
| 慢衰落 | $T_s<T_c$ |
| 快衰落 | $T_s>T_c$ |

### 1.3 統計模型
- **Rayleigh**：無視距，振幅 $\sim$ Rayleigh，相位 $\sim$ 均勻。
- **Rician**：含主視距 + 散射。
- **Nakagami-m**：可調 m 涵蓋兩者。
- 二階統計：自相關、功率譜（Jakes 模型用於行動通訊）。

### 1.4 室內 / 戶外經驗模型
- Okumura–Hata、COST 231（市區）。
- ITU-R 室內模型。
- 3GPP 通道模型（用於系統模擬）。

## 第 2 章　對抗衰落的技術

### 2.1 分集（Diversity）
重點：多份獨立副本平均下來變穩。
- **時間分集**：交織 + FEC。
- **頻率分集**：多載波（OFDM 配合 FEC）。
- **空間分集**：MIMO 多天線。
- **極化分集**：垂直 / 水平偏振。
- **多徑分集**：Rake 接收機（CDMA 系統用）。

### 2.2 等化（複習）
頻率選擇通道下，做線性 / DFE / MLSE 等化。OFDM 用循環字首 + 每子載波單抽頭等化。

### 2.3 編碼 + 交織
把長 burst error 打散，搭配 FEC 糾正。
- BLER / FER 是無線常用指標。
- HARQ：把編碼與重傳結合。

### 2.4 鏈路自適應
依通道品質（CQI 報告）動態調：
- 調變階數（BPSK ↔ 256-QAM）
- 編碼率
- MIMO 模式
- 排程資源塊

## 第 3 章　多重接取（Multiple Access）

幾種共享頻譜的方式：

| 方式 | 描述 | 系統例 |
| --- | --- | --- |
| **FDMA** | 每用戶一段頻段 | 1G、廣播 |
| **TDMA** | 每用戶一個時槽 | 2G GSM |
| **CDMA** | 每用戶一條碼 | 3G WCDMA / cdma2000 |
| **OFDMA** | 每用戶若干子載波 + 時槽 | 4G LTE、5G NR、Wi-Fi 6 |
| **SDMA** | 每用戶一條空間波束 | 大規模 MIMO 配合波束成形 |
| **NOMA** | 同頻同時但功率 / 編碼分離 | 5G 與 6G 候選 |

## 第 4 章　蜂巢式系統（Cellular Systems）

### 4.1 基本概念
- 把覆蓋區切成多個小區（cell），每個有基地台。
- **頻率複用（frequency reuse）**：相鄰小區用不同頻段；遠距離小區重複使用，提升頻譜效率。
- **覆蓋規劃**：六邊形理論模型；複用因子 $N=1,3,7,\dots$。

### 4.2 切換（Handover / Handoff）
行動裝置在小區之間移動時，連線轉移：
- **硬切換**：先斷後連（GSM）。
- **軟切換**：同時連接多個小區（CDMA）。
- **垂直切換**：跨不同網路（4G ↔ Wi-Fi）。

### 4.3 功率控制
- **開迴路 / 閉迴路功控**：控制每用戶發射功率，避免遠近效應（near-far problem，CDMA 特別關鍵）。

### 4.4 容量與干擾
- **同頻干擾（co-channel interference, CCI）**：來自其他小區複用同頻段的用戶。
- **鄰頻干擾（ACI）**。
- SINR 替代 SNR 成為實際性能指標。

## 第 5 章　行動通訊系統演進

### 5.1 1G（類比）
NMT、AMPS：FM 語音 + FDMA。1980 年代。

### 5.2 2G（數位語音 + 簡訊）
- **GSM**（全球主流）：TDMA + FDD + GMSK 調變 + 捲積碼。
- **IS-95**（CDMA）：cdmaOne，引入 CDMA 與軟切換。
- 衍生：GPRS、EDGE 提供低速資料。

### 5.3 3G（行動寬頻起步）
- **WCDMA / UMTS**：5 MHz 載波、Turbo 碼。
- **cdma2000 / EV-DO**。
- HSPA 提升下行至幾十 Mbps。

### 5.4 4G LTE
- **OFDMA**（下行）+ **SC-FDMA**（上行，PAPR 低）。
- MIMO 2x2 → 4x4 → 8x8。
- 全 IP 網路（沒有電路交換）。
- Turbo 碼；之後 LTE-Advanced 加上載波聚合（CA）、CoMP、heterogeneous networks。

### 5.5 5G NR
- **三大場景**：eMBB（高速）、uRLLC（低延遲高可靠）、mMTC（巨量物聯網）。
- **新頻段**：FR1（< 7 GHz）+ FR2（mmWave 24–52 GHz）。
- **彈性編碼**：LDPC（資料）+ Polar（控制）。
- **大規模 MIMO + Beamforming**：尤其 mmWave 必備。
- **網路切片（network slicing）**：軟體定義網路 + NFV，為不同應用提供不同 SLA。
- **數位化前傳（CPRI / eCPRI、O-RAN）**。

### 5.6 6G（研究中）
- THz 頻段（0.1–10 THz）
- AI 原生網路、感測通訊一體（ISAC）
- 太空—地面整合
- 智慧反射面（RIS / IRS）

## 第 6 章　短距無線

### 6.1 Wi-Fi（IEEE 802.11）
- a / b / g / n / ac / ax / be 演進。
- ax（Wi-Fi 6）：OFDMA、1024-QAM、MU-MIMO。
- be（Wi-Fi 7）：320 MHz 頻寬、4096-QAM、MLO（多鏈接同時運作）。

### 6.2 藍牙
- 跳頻 + GFSK / 8-DPSK；經典 vs BLE。
- 用於耳機、滑鼠、IoT。

### 6.3 ZigBee / Thread / Z-Wave / LoRa / NB-IoT
- 低功耗、低速率、IoT 為主。

### 6.4 UWB
- 大頻寬短距離；蘋果 AirTag、汽車數位鑰匙。

## 第 7 章　衛星與非陸地網路（NTN）

- **LEO / MEO / GEO**：低 / 中 / 同步軌道。
- **Starlink、OneWeb、Kuiper**：LEO 大星座。
- 5G / 6G 含 NTN：手機直連衛星（Direct-to-Cell）。
- 通訊參數：高都卜勒（LEO）、長傳播延遲（GEO ~250 ms）、自由空間損失。

## 第 8 章　定位（Positioning）

- **GNSS**：GPS、GLONASS、Galileo、北斗 → CDMA + 時間量測 + 多顆衛星三角定位。
- **蜂巢式定位**：CID、TDOA、E-CID、OTDOA、RTT；5G NR 高精度。
- **室內定位**：Wi-Fi RSSI / RTT、UWB、藍牙、磁場、慣性。

## 第 9 章　無線安全（一窺）

- 加密：A5/1（GSM）、AES、Snow 3G、ZUC。
- 鑑別：SIM、USIM、5G AKA。
- 完整性：HMAC、SNOW、AES-CMAC。
- 隱私：IMSI 加密、訊號擾動、零信任架構。
