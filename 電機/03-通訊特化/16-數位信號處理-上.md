## 這科在學什麼

「信號與系統（下）」把離散時間理論說完，這科把那套理論變成**設計與實作**：
- 濾波器設計（FIR / IIR）
- 譜分析（FFT、加窗）
- 自適應濾波（LMS / RLS）
- 多速率處理
- 數值與架構議題（定點誤差、平行化）

DSP 就是「能在 MCU / DSP / FPGA / GPU 上跑得動的數學」。

## 第 1 章　離散時間訊號與系統（複習要點）

- 序列 $x[n]$。
- LTI 系統 → 卷積 $y[n]=\sum_k h[k]x[n-k]$。
- 差分方程 $\sum a_k y[n-k]=\sum b_m x[n-m]$。
- DTFT / Z 轉換、$|z|<1$ 內極點穩定（因果）。

## 第 2 章　FIR 濾波器

### 2.1 性質
$$y[n]=\sum_{k=0}^{M-1}h[k]x[n-k]$$
$h[n]$ 有限長 → 本質穩定。
若 $h[n]=h[M-1-n]$（對稱）→ **嚴格線性相位**，群延遲固定 = $(M-1)/2$ 樣本——通訊與聲音處理大優勢。

### 2.2 設計方法

**(a) 窗口法**
1. 對理想頻率響應做反 DTFT 得理想 $h_d[n]$。
2. 截斷到長度 $M$。
3. 乘上窗 $w[n]$ 降低旁瓣。

| 窗 | 旁瓣 (dB) | 過渡帶寬 |
| --- | --- | --- |
| 矩形 | $-13$ | 最窄 |
| Hann | $-32$ | 中 |
| Hamming | $-43$ | 中 |
| Blackman | $-58$ | 寬 |
| Kaiser | 可調 (β) | 可調 |

**(b) 頻率採樣法**
在 $N$ 個等距頻率點指定振幅 / 相位 → IDFT。簡單但對中間頻率控制差。

**(c) Parks–McClellan（最佳等漣波）**
用 Remez 演算法在通帶 / 阻帶誤差等漣波下找最佳係數。長度最短達到指定規格的方法。

### 2.3 多相分解（Polyphase）
高速 FIR 在抽取 / 內插時，可以拆成多相子濾波器並行運算，省功又能在低時脈下做高速處理。

### 2.4 對稱性與運算節省
線性相位 FIR 可把對稱係數合併：$M$ 階濾波器只需 $M/2$ 個乘法器。

## 第 3 章　IIR 濾波器

### 3.1 結構
$$H(z)=\dfrac{\sum b_m z^{-m}}{1+\sum a_k z^{-k}}$$
含回授 → 可用較少係數達到陡峭頻率響應，但**可能不穩定**，且**相位非線性**。

### 3.2 設計：從類比原型轉換
1. 先在 $s$ 域設計（Butterworth、Chebyshev I/II、Elliptic、Bessel）。
2. 用**雙線性變換**：
   $$s=\dfrac{2}{T}\dfrac{1-z^{-1}}{1+z^{-1}}$$
   把整個左半 $s$ 平面映到單位圓內。需要**頻率預扭曲（pre-warping）**抵銷映射的非線性。

### 3.3 各家族特性

| 家族 | 通帶 | 阻帶 | 過渡帶 | 相位 |
| --- | --- | --- | --- | --- |
| Butterworth | 最大平坦 | 緩慢下降 | 寬 | 較好 |
| Chebyshev I | 等漣波 | 平滑 | 較窄 | 差 |
| Chebyshev II | 平滑 | 等漣波 | 較窄 | 差 |
| Elliptic（Cauer） | 等漣波 | 等漣波 | 最窄 | 最差 |
| Bessel | 平滑 | 緩慢 | 寬 | 最佳（接近線性） |

### 3.4 實作架構
- **直接型 I / II**：簡單但對係數量化敏感。
- **轉置型**：對某些量化情形較好。
- **級聯（cascade）二階節（biquad）**：把 IIR 拆成多個二階段串接，數值穩定性大幅提升——實務上 IIR 幾乎都這樣實作。
- **平行（parallel）**
- **格狀（lattice）**：對應反射係數，常用於語音處理（LPC）。

### 3.5 定點實作議題
- **係數量化**：把極點推離原來位置，可能不穩。
- **訊號量化**：每次乘法後截斷產生雜訊。
- **溢位（overflow）**：要用飽和算術。
- **極限環（limit cycle）**：IIR 在無輸入時可能自我振盪。
- 解法：scaling、roundoff noise 分析、bipropagation、用 Q 格式定點數規劃。

## 第 4 章　頻率變換

要從低通設計派生其他濾波器：
- 低通 → 高通、帶通、帶阻。
- $z$ 域的標準替換：
  - 高通：$z^{-1}\to -z^{-1}$
  - 帶通、帶阻：較複雜的雙曲變換。

## 第 5 章　線性相位 vs 最小相位

- **線性相位**：群延遲固定，但通常較長 → 高延遲。
- **最小相位**：所有零點在單位圓內，能量集中於前 → 低延遲，但相位失真。
- **全通（all-pass）**：$|H(e^{j\omega})|=1$，只影響相位 / 群延遲。

工程選擇：通訊 / 高保真音訊偏線性相位；即時控制 / 對話聲偏最小相位（低延遲）。

## 第 6 章　數位濾波器的硬體實作

- **DSP 處理器**：MAC（乘加）每週期一次，環形緩衝、零開銷迴圈、SIMD 指令。
- **FPGA**：客製化管線、平行 MAC、DSP slice。
- **ASIC**：終極效能 / 低功耗。
- **MCU**：低成本，但效能有限；ARM Cortex-M4F / M7 有 DSP 指令集。
- **GPU**：大量平行 FIR、卷積（影像處理／神經網路）。

## 第 7 章　常用 DSP 演算法骨架

- **MAC 迴圈**：FIR、相關器、匹配濾波器。
- **遞迴更新**：IIR、滑動平均、指數平滑。
- **轉換**：FFT、DCT、小波轉換。
- **判決 + 回授**：PLL、DPLL、CDR、Sigma-Delta 環。
- **迭代最佳化**：LMS、RLS、Newton 法（自適應 / 學習）。

這些骨架認得後，看任何 DSP 程式都可以「先猜結構再看細節」。
