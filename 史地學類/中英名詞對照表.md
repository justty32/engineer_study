# 史地學類：中英名詞對照表

本表收錄歷史學與地理學之核心專業術語。

## 1. 歷史學基本術語
| 中文 | 英文 | 說明 |
| --- | --- | --- |
| 史學方法 | Historical Method | 研究歷史的程序與技術 |
| 第一手史料 | Primary Source | 當時代留下的直接證據 |
| 第二手史料 | Secondary Source | 後人的分析與詮釋 |
| 考據學 | Philology / Textual Criticism | 對史料真偽與內容的核實 |
| 年鑑學派 | Annales School | 強調長時段、整體史的研究流派 |
| 典範轉移 | Paradigm Shift | 科學或學術觀念的重大變革 |

## 2. 自然地理學術語
| 中文 | 英文 | 說明 |
| --- | --- | --- |
| 板塊構造論 | Plate Tectonics | 解釋地殼運動的理論 |
| 地形學 | Geomorphology | 研究地表形態及其演化的學科 |
| 氣候變遷 | Climate Change | 長期氣候模式的改變 |
| 水文循環 | Hydrological Cycle | 地球上水的循環過程 |
| 喀斯特地形 | Karst Topography | 石灰岩受溶蝕後形成的地形 |

## 3. 人文地理學術語
| 中文 | 英文 | 說明 |
| --- | --- | --- |
| 紳士化 / 仕紳化 | Gentrification | 都市舊區更新與階層移入過程 |
| 核心-外圍模式 | Core-Periphery Model | 解釋經濟空間不平等的模型 |
| 人口轉型 | Demographic Transition | 人口出生率與死亡率的演變過程 |
| 都市化 | Urbanization | 人口向都市集中的過程 |
| 永續發展 | Sustainable Development | 滿足當前需求且不損害後代利益的發展 |

## 4. 地理技術術語 (GIS/RS)
| 中文 | 英文 | 說明 |
| --- | --- | --- |
| 地理資訊系統 | GIS (Geographic Information System) | 儲存、管理與分析空間數據的系統 |
| 遙測 | RS (Remote Sensing) | 遠距離偵測地表資訊的技術 |
| 全球定位系統 | GPS (Global Positioning System) | 利用衛星進行定位導航的系統 |
| 空間分析 | Spatial Analysis | 對空間現象進行量化與模型分析 |
| 網格數據 | Raster Data | 以像素組成的地理數據格式 |
| 向量數據 | Vector Data | 以點、線、面組成的地理數據格式 |
