# 史學史

史學史（Historiography）是「關於歷史研究的歷史」，探討人類對過去的認知、紀錄與詮釋方式的演變。

## 1. 中國史學史
- **先秦至兩漢**：從口傳到撰述，《史記》（紀傳體）、《漢書》（斷代史）。
- **魏晉隋唐**：史官制度的確立，《通典》（政書體）。
- **宋元明清**：考據學的興盛，《資治通鑑》（編年體）、章學誠的「六經皆史」。

## 2. 西洋史學史
- **古典時期**：希羅多德（歷史之父）、修昔底德（講求證據與邏輯）。
- **中世紀**：神學史觀，歷史作為上帝計畫的體現。
- **啟蒙與19世紀**：蘭克 (Leopold von Ranke) 提倡「如實直書」，專業歷史學的誕生。

## 3. 20世紀後的多元史觀
- **年鑑學派**：強調長時段、社會史與整體史。
- **後現代主義挑戰**：質疑歷史的客觀性，關注敘事與文本。
- **大眾史學與公共史學**：歷史在社會中的角色與應用。

## 4. 史學思想的核心爭論
- **進步論 vs 循環論**：歷史是有目的的演進，還是不斷循環？
- **英雄史觀 vs 群眾史觀**：歷史由少數偉人決定，還是廣大群眾推動？
