# 01-食品化學 (Food Chemistry)

> **定位**：食品加工的底層邏輯。理解食物在加熱、冷凍、乾燥或儲存過程中，分子層面發生了什麼變化，進而控制食品的色、香、味與保存期限。

食品在加工過程中並非只是單純的「弄熟」，而是一連串複雜的化學反應與物理變化。食品化學主要關注**水**、**三大營養素**（碳水化合物、蛋白質、脂質）以及它們如何產生**顏色與風味**的變化。

---

## 1. 水分活度 (Water Activity, $A_w$)

**直覺：** 不是食物裡有「多少水」，而是微生物能「用多少水」。

*   **總含水量 (Moisture Content)**：食品中水分的總重百分比（結合水 + 自由水）。
*   **水分活度 ($A_w$)**：食品中**自由水**（未與大分子結合，可供微生物利用或參與化學反應的水）的比例。
    *   **定義與公式**：食品表面水蒸氣壓 ($P$) 與同溫度下純水蒸氣壓 ($P_0$) 的比值。
    *   $$A_w = \frac{P}{P_0}$$ （範圍介於 0 到 1 之間）
*   **工程意義**：
    *   $A_w > 0.90$：容易滋生細菌（如鮮肉、鮮奶）。
    *   $A_w < 0.85$：多數致病菌無法生長（常見防腐門檻）。
    *   $A_w < 0.60$：幾乎所有微生物（包含黴菌、酵母菌）都無法生長（如奶粉、乾燥餅乾）。
*   **加工手段**：透過「乾燥（脫水）」或「加鹽/加糖（將自由水綁定變成結合水）」來降低 $A_w$，達到防腐效果。

## 2. 褐變反應 (Browning Reactions)

食物在加工過程中變黑或變褐，是賦予風味（如烤麵包香）或導致劣變（如蘋果切開變黑）的關鍵。主要分兩大類：

### 2.1 非酵素性褐變 (Non-enzymatic Browning)
不需要酵素參與，通常由**高溫**驅動。
1.  **梅納反應 (Maillard Reaction)**：
    *   **條件**：`還原糖` + `胺基酸（蛋白質）` + `加熱`。
    *   **結果**：產生複雜的棕色聚合物（類黑精）與幾百種揮發性風味物質。
    *   **實例**：烤肉的焦香、烤麵包的表皮、咖啡烘焙、黑蒜頭（低溫長時間）。
2.  **焦糖化反應 (Caramelization)**：
    *   **條件**：只有 `醣類（如蔗糖）` + `極高溫（>160°C）`，無蛋白質參與。
    *   **結果**：糖分子脫水聚合，產生焦糖色與微苦/甜的風味。
    *   **實例**：煮焦糖布丁的糖漿、炒糖色。

### 2.2 酵素性褐變 (Enzymatic Browning)
*   **條件**：`多酚類物質` + `多酚氧化酶 (PPO)` + `氧氣`。
*   **過程**：植物細胞受損（切開或摔傷）後，酵素將酚類氧化成醌類，進而聚合成褐色物質。
*   **實例**：蘋果、馬鈴薯切開後變黑；紅茶的發酵過程（刻意利用此反應）。
*   **防護手段**：殺青（加熱破壞酵素）、泡鹽水/檸檬酸（改變 pH 值或隔絕氧氣）、添加維生素 C（還原劑）。

## 3. 三大營養素的加工變化

在加工過程中，碳水化合物、蛋白質與脂質會發生巨大的結構改變。

### 3.1 碳水化合物 (主要探討澱粉)
澱粉在水中加熱與冷卻的過程，決定了食品的口感。
*   **糊化 (Gelatinization)**：
    *   **過程**：生澱粉加水加熱，氫鍵斷裂，水分進入澱粉顆粒使其膨潤、破裂，變成具黏性的糊狀（即「生米煮成熟飯」）。
*   **老化 (Retrogradation)**：
    *   **過程**：糊化後的澱粉在冷卻或放置過程中，澱粉分子重新排列，擠出水分，結構變硬（即「麵包放久變乾硬」、「冷飯變硬」）。冷藏溫度 (0~4°C) 最容易發生老化，冷凍或熱存可延緩。

### 3.2 蛋白質
*   **變性 (Denaturation)**：
    *   **過程**：蛋白質受 `熱`、`酸鹼`、`重金屬` 或 `機械力（攪打）` 影響，其 3D 摺疊結構（二、三、四級結構）被破壞而展開，但胺基酸序列（一級結構）未斷裂。
    *   **結果**：溶解度下降、失去生物活性（如酵素失活）。
    *   **實例**：煎蛋（蛋白質受熱凝固）、做優格（加酸使酪蛋白變性沉澱）。

### 3.3 脂質 (脂肪與油)
*   **脂質氧化 (Lipid Oxidation) / 酸敗 (Rancidity)**：
    *   **機制**：不飽和脂肪酸中的雙鍵不穩定，在 `氧氣`、`光`、`熱` 或 `金屬離子` 催化下，產生自由基連鎖反應。
    *   **結果**：裂解出醛、酮類小分子，產生難聞的「油耗味」。
    *   **防護手段**：添加抗氧化劑（如維生素 E、BHA/BHT）、真空包裝、避光保存、氫化（減少雙鍵，但可能產生反式脂肪）。

## 4. 總結與工程視角

在食品加工廠中，食品化學知識直接對應到**製程參數的設定**：
1.  **控制溫度與時間**：為了促發梅納反應（烤肉）或是為了避免維生素被破壞？
2.  **控制水分 ($A_w$)**：烘乾要到什麼程度才能確保產品在室溫下存放半年不發霉？
3.  **控制氧氣**：包裝是否需要抽真空，以防止脂肪酸敗與酵素性褐變？
