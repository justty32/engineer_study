# 材料力學（Mechanics of Materials / Solid Mechanics）

> 機械第二階段第 3 科（四大力學之一）。
> 名詞對照見 [中英名詞對照表 → 材料力學](../中英名詞對照表.md#材料力學)。

## 這科在學什麼

靜力學算「物體外部受的力與反力」，但**不問物體內部會不會壞**。材料力學接手這個問題：給定外力，物體內部各處承受多大的**應力**、變形多少，會不會降伏、斷裂或挫曲。它是機械設計、土木結構、航太強度分析的直接基礎——「這根軸要多粗才不會斷」就是這科回答的。

核心套路只有一個：**外力 → 內力 → 應力 → 應變 → 變形**，再拿應力跟材料的強度比，判斷安不安全。掌握這條鏈，整科都通。

## 第 1 章　應力與應變

### 1.1 應力（Stress）
應力＝單位面積上的內力（材料「緊張」的程度），單位 Pa（N/m²），工程常用 MPa。
- **正向應力 $\sigma=\dfrac{N}{A}$**：力垂直截面（拉為正、壓為負）。
- **剪應力 $\tau=\dfrac{V}{A}$**：力平行截面（像剪刀、鉚釘受剪）。

關鍵觀念：力一樣，**截面積越小應力越大**，所以斷裂常發生在最細、有缺口、有孔的地方（**應力集中**）。

### 1.2 應變（Strain）
應變＝變形的「相對量」，無單位。
- **正向應變 $\varepsilon=\dfrac{\delta}{L}$**：伸長量除以原長。
- **剪應變 $\gamma$**：角度的改變（rad）。

### 1.3 虎克定律與材料常數
在彈性範圍內，應力與應變成正比：
$$\sigma=E\,\varepsilon,\qquad \tau=G\,\gamma$$
- **楊氏模數 $E$**：抗拉剛性。**剪彈性模數 $G$**：抗剪剛性。
- **蒲松比 $\nu$**：縱向拉伸時橫向會縮，$\nu=-\dfrac{\varepsilon_\text{橫}}{\varepsilon_\text{縱}}$（金屬約 0.3）。
- 三者關係：$G=\dfrac{E}{2(1+\nu)}$。
- 軸向變形量：$\delta=\dfrac{NL}{AE}$（像彈簧，剛度 $k=AE/L$）。

## 第 2 章　軸向、扭轉、彎曲——三種基本載荷

材料力學的主體就是把三種典型受力各算出應力與變形。

### 2.1 軸向載荷（Axial）
- 應力 $\sigma=N/A$，變形 $\delta=NL/(AE)$。
- 靜不定問題（兩端固定的桿）要加**變形協調**條件補足平衡方程。
- 熱應力：受熱想膨脹 $\delta_T=\alpha\,\Delta T\,L$，若被約束住就產生 $\sigma=E\alpha\Delta T$（橋梁伸縮縫、管路設計的原因）。

### 2.2 扭轉（Torsion）——圓軸傳遞扭矩
傳動軸、螺絲起子的受力。圓截面：
$$\tau=\frac{T\,r}{J},\qquad \phi=\frac{TL}{GJ}$$
- $T$＝扭矩，$J$＝**極慣性矩**（圓 $J=\pi d^4/32$），$\phi$＝扭轉角。
- 剪應力**表面最大、中心為零**，所以**空心軸**幾乎一樣強卻更輕（材料用在刀口上）——這是慣性矩思想的又一體現。
- 傳遞功率 $P=T\omega$，馬達軸設計常由此反推所需軸徑。

### 2.3 彎曲（Bending）——樑（Beam）
最重要、最常考。樑受橫向載荷而彎。

**(a) 剪力與彎矩圖（SFD/BMD）**
先求支承反力，再沿樑長畫出**剪力 $V(x)$** 與**彎矩 $M(x)$** 的分布。關係：
$$\frac{dV}{dx}=-w(x),\qquad \frac{dM}{dx}=V$$
（$w$＝分布載荷）。彎矩最大處通常是最危險斷面。畫這兩張圖是樑分析的基本功。

**(b) 彎曲應力（撓曲公式）**
$$\sigma=\frac{M\,y}{I}$$
- $I$＝截面對**中性軸**的面積慣性矩（靜力學學過），$y$＝離中性軸距離。
- 應力**中性軸為零、上下表面最大**（一面拉、一面壓）。
- 所以抗彎要把材料放遠離中性軸 → **I 型樑、工字樑**的由來。
- 定義**斷面模數 $S=I/c$**，最大彎曲應力 $\sigma_\text{max}=M/S$，設計時直接比 $S$。

**(c) 橫向剪應力**
$$\tau=\frac{VQ}{I\,b}$$
樑內也有剪應力，矩形樑在**中性軸最大**（與彎曲應力相反），短而胖的樑要注意。

**(d) 撓度（Deflection）**
樑彎多少：$EI\,\dfrac{d^2v}{dx^2}=M(x)$，積分兩次（配邊界條件）得撓曲線 $v(x)$。$EI$＝**抗彎剛度**，越大越不易變形。靜不定樑用此補方程。

## 第 3 章　應力的進階觀念

### 3.1 組合載荷
真實零件常同時受軸力 + 彎矩 + 扭矩（如曲軸、起重臂）。先各自算應力，再依方向**疊加**。

### 3.2 平面應力與莫爾圓（Mohr's Circle）
同一點在不同切面方向上應力不同。給定一組 $(\sigma_x,\sigma_y,\tau_{xy})$，可求：
- **主應力（principal stress）$\sigma_1,\sigma_2$**：剪應力為零的方向，正向應力達極值——脆性材料的斷裂方向。
- **最大剪應力 $\tau_\text{max}$**：延性材料降伏的關鍵。
- **莫爾圓**：把上述關係畫成一個圓，是材料力學的招牌工具，一眼讀出任意方向的應力。

### 3.3 破壞理論（簡介，詳見機械設計）
判斷複雜應力下會不會降伏：
- **最大剪應力理論（Tresca）**、**畸變能理論（von Mises）**——延性材料常用後者。
- 脆性材料用最大正向應力理論。

## 第 4 章　挫曲（Buckling）——壓桿的突然失穩

細長的壓桿不是被「壓扁」而是突然**側向彎曲垮掉**，遠在材料降伏之前。**尤拉臨界載荷**：
$$P_\text{cr}=\frac{\pi^2 EI}{(KL)^2}$$
- $K$＝有效長度係數（看兩端如何支承：兩端鉸接 $K=1$、一端固定一端自由 $K=2$…）。
- 重點：挫曲是**剛度（$EI$）與幾何**問題，不是強度問題；柱越細長越危險，與材料多強無關。
- 設計細長壓桿（千斤頂螺桿、連桿、桁架壓桿）必查。

## 第 5 章　其他重要主題

- **壓力容器**：薄壁圓筒的環向應力 $\sigma_\theta=pr/t$、軸向應力 $\sigma_z=pr/(2t)$（環向是軸向兩倍，所以爆裂常沿縱向裂開）。
- **應力集中（stress concentration）**：孔、缺口、肩角處實際應力 = 名義應力 × 集中係數 $K_t$，疲勞裂縫的起點，設計要倒圓角。
- **能量法（卡氏定理）**：用應變能求複雜結構的變形/反力。
- **超靜定**：平衡不夠解時，補上「變形相容」方程。

## 小結

- 一條主線貫穿全科：**外力 → 內力 → 應力 → 應變 → 變形 → 比強度判安全**。
- 三種基本載荷的公式要熟：軸向 $\sigma=N/A$、扭轉 $\tau=Tr/J$、彎曲 $\sigma=My/I$。
- **面積慣性矩 $I$** 是抗彎/抗扭的核心幾何量，解釋了 I 型樑與空心軸。
- 樑分析的基本功是**剪力圖與彎矩圖**，再用 $\sigma=My/I$ 與撓度方程收尾。
- **莫爾圓**讀任意方向應力；**尤拉公式**管細長壓桿的挫曲（剛度問題，非強度）。
- 隱形殺手：**應力集中、熱應力、壓力容器環向應力**。
