# 機械振動（Mechanical Vibrations）

> 名詞對照見 ../中英名詞對照表.md

機械振動研究「機械系統在恢復力與慣性之間來回交換能量、產生週期性運動」這件事。任何有質量（慣性）又有彈性（恢復力）的東西——轉軸、樑、車身、引擎座、葉片——只要受到擾動就會振動。本科的核心不是背公式，而是建立一套統一的分析框架：先寫出運動方程（一條或一組二階常微分方程），再依「有沒有外力」「幾個自由度」「阻尼多大」分類求解，最後把數學結果翻譯成工程語言——固有頻率多少、會不會共振、要不要隔振。對電機背景的人來說好消息是：**振動的數學就是 RLC 電路與信號系統那一套二階系統，幾乎逐項對應**，差別只在物理量換成位移、速度、力。

---

## 0. 本科在本系的定位

機械振動建立在 [動力學](../01-工程力學基礎/03-動力學.md) 之上：你必須先會列剛體或質點的運動方程（$\sum F = ma$ 或拉格朗日方程），振動只是把恢復力（彈簧）與耗散力（阻尼）放進去後，看時間解長什麼樣子。往下，它直接餵養 [系統動態與控制](13-系統動態與控制.md)（控制系統要鎮壓的「廠房動態」往往就是某個振動模態）與結構設計（避免共振、設計隔振）。

**核心心法（三條）：**

1. **一切都是二階系統。** 單自由度振動的運動方程
$$m\ddot{x} + c\dot{x} + kx = F(t)$$
與 RLC 串聯電路 $L\ddot{q} + R\dot{q} + \frac{1}{C}q = v(t)$ 在數學上完全同構：$m \leftrightarrow L$、$c \leftrightarrow R$、$k \leftrightarrow 1/C$、$x \leftrightarrow q$。固有頻率、阻尼比、品質因數、共振、頻率響應的全部直覺都能搬過來。物理上的差別只在：這裡的「彈簧—質量—阻尼」是看得見的機械元件，恢復力 $kx$ 來自彈性變形，阻尼 $c\dot{x}$ 來自油膜、空氣或材料內摩擦。

2. **模態是把多自由度拆成一堆獨立單自由度的方法。** 複雜結構有很多自由度，看似耦合難解，但透過「固有頻率＋模態振型」做座標變換後，每個模態各自是一個獨立的單自由度二階系統。這跟信號系統用本徵函數（傅立葉基底）把訊號拆成獨立頻率分量是同一種思想。

3. **頻率比 $r = \omega/\omega_n$ 決定一切。** 強迫振動的放大、共振、隔振效果全由激振頻率與固有頻率之比 $r$ 決定，不是看絕對頻率。記住三個區段：$r \ll 1$（剛性區，跟著力走）、$r \approx 1$（共振區，危險）、$r \gg 1$（慣性區、隔振有效）。

## 1. 振動基本概念

### 1.1 自由度與模型

**自由度（DOF）**＝完整描述系統運動所需的獨立座標數。一個只能上下動的質量塊是單自由度（SDOF）；汽車車身（上下＋俯仰）約簡化為 2 自由度；連續的樑理論上有無限多自由度。建模的第一步永遠是「決定要保留幾個自由度」——這是準度與計算量的取捨。

系統的三個基本元件：**慣性元件**（質量 $m$、轉動慣量 $J$，儲存動能）、**彈性元件**（彈簧 $k$，儲存位能、提供恢復力）、**阻尼元件**（阻尼器 $c$，耗散能量）。振動正是動能與位能週期性互換、阻尼逐步把能量耗散掉的過程。

### 1.2 簡諧運動（SHM）

最基本的週期運動是簡諧運動 $x(t) = A\sin(\omega t + \phi)$：$A$ 為振幅，$\omega$ 為角頻率（rad/s），$\phi$ 為相位，週期 $T = 2\pi/\omega$、頻率 $f = 1/T$。速度與加速度為其微分，相位各超前 $90°$、$180°$。複數表示 $x = \mathrm{Re}(A e^{j\omega t})$ 與電機的相量（phasor）完全一樣，是後面分析頻率響應的利器。

### 1.3 振動的分類

- **自由 vs 強迫**：自由振動由初始條件激發後自行衰減；強迫振動由持續外力維持。
- **無阻尼 vs 有阻尼**：理想無阻尼振幅不衰減；實際系統都有阻尼。
- **線性 vs 非線性**：本科主要處理線性（小振幅、係數為常數）。
- **確定性 vs 隨機**：激振是確定函數，或是統計描述的隨機過程。

## 2. 單自由度自由振動

### 2.1 無阻尼自由振動與固有頻率

運動方程 $m\ddot{x} + kx = 0$，解為簡諧振動，**固有角頻率（natural frequency）**
$$\omega_n = \sqrt{\frac{k}{m}} \quad (\text{rad/s}), \qquad f_n = \frac{1}{2\pi}\sqrt{\frac{k}{m}}.$$
固有頻率只由系統本身（剛度與質量）決定，與初始條件、激振無關——這是最重要的單一參數。剛度越大、質量越小，振得越快。

### 2.2 能量法（Rayleigh 法）

對無阻尼系統，總機械能守恆：$T + V = \text{常數}$，故 $\frac{d}{dt}(T+V)=0$ 也能導出運動方程，常比畫自由體圖更快。**Rayleigh 法**更進一步：令最大動能等於最大位能 $T_{\max} = V_{\max}$，直接解出 $\omega_n$，不必寫出完整微分方程。這對含分布質量的系統（如帶質量彈簧、擺動的樑）估固有頻率特別好用。

### 2.3 黏性阻尼與阻尼比

加入黏性阻尼（阻尼力正比於速度）：$m\ddot{x} + c\dot{x} + kx = 0$。除以 $m$ 得標準型
$$\ddot{x} + 2\zeta\omega_n \dot{x} + \omega_n^2 x = 0,$$
其中**阻尼比（damping ratio）** $\zeta = \dfrac{c}{2\sqrt{km}} = \dfrac{c}{c_c}$，$c_c = 2\sqrt{km} = 2m\omega_n$ 為**臨界阻尼係數**。這個標準型與控制系統、信號系統的二階系統一字不差，$\zeta$ 就是同一個阻尼比。特徵方程 $s^2 + 2\zeta\omega_n s + \omega_n^2 = 0$ 的根決定行為。

### 2.4 過阻尼、欠阻尼、臨界阻尼

依 $\zeta$ 分三種情形（與電路暫態完全對應）：

- **欠阻尼（$0 < \zeta < 1$）**：根為共軛複數，響應為**指數衰減的振盪**
$$x(t) = X e^{-\zeta\omega_n t}\sin(\omega_d t + \phi), \qquad \omega_d = \omega_n\sqrt{1-\zeta^2},$$
$\omega_d$ 為**阻尼固有頻率**，略低於 $\omega_n$。絕大多數機械系統屬此類。
- **臨界阻尼（$\zeta = 1$）**：重根，不振盪且最快回到平衡。門的閉門器、儀表指針都設計在此附近。
- **過阻尼（$\zeta > 1$）**：兩個負實根，不振盪、回平衡較慢。

### 2.5 對數遞減率

實驗上量阻尼最方便的方法：取欠阻尼自由衰減曲線上相鄰兩個（或相隔 $n$ 個）波峰振幅之比，定義**對數遞減率（logarithmic decrement）**
$$\delta = \ln\frac{x_i}{x_{i+1}} = \frac{2\pi\zeta}{\sqrt{1-\zeta^2}} \;\approx\; 2\pi\zeta \;(\text{小阻尼}).$$
相隔 $n$ 週期時 $\delta = \frac{1}{n}\ln\frac{x_0}{x_n}$。量到 $\delta$ 即可反推 $\zeta$，再配合測得的 $\omega_d$ 得到完整模型參數。這是振動量測與系統識別的入門技巧。

## 3. 單自由度強迫振動

### 3.1 諧波激振與穩態響應

施加諧波力 $F(t) = F_0\cos\omega t$，運動方程 $m\ddot{x}+c\dot{x}+kx = F_0\cos\omega t$。穩態解仍是同頻率的簡諧運動 $x_p = X\cos(\omega t - \phi)$。用相量（複數阻抗）法最快，定義頻率比 $r = \omega/\omega_n$：
$$X = \frac{F_0/k}{\sqrt{(1-r^2)^2 + (2\zeta r)^2}}, \qquad \tan\phi = \frac{2\zeta r}{1-r^2}.$$
$F_0/k$ 是把力靜態加上去的「靜變形」，前面的分式即**動態放大因子（magnification factor）** $M = X/(F_0/k)$。

### 3.2 頻率響應與共振

把 $M$ 對 $r$ 作圖就是頻率響應曲線（與電路的頻率響應、波德圖同一回事）：

- $r \ll 1$：$M \approx 1$，位移跟著力（剛度主控），相位 $\approx 0$。
- $r \approx 1$：**共振（resonance）**，$M$ 達峰值。無阻尼時 $M \to \infty$；有阻尼時峰值 $M_{\max} \approx \frac{1}{2\zeta}$（小阻尼），出現在 $r = \sqrt{1-2\zeta^2}$ 附近。相位急轉至 $90°$。
- $r \gg 1$：$M \to 0$（慣性主控，質量來不及跟），相位 $\to 180°$。

$\frac{1}{2\zeta}$ 正是電機裡的**品質因數 $Q$**：阻尼越小，共振峰越尖越高、越危險。工程上要嘛遠離共振，要嘛加阻尼壓低峰值。

### 3.3 力傳遞率與隔振

當機器把振動力傳到地基（或地基把振動傳給設備）時，關心**傳遞率（transmissibility）** $TR$＝傳出力與激振力之比：
$$TR = \sqrt{\frac{1 + (2\zeta r)^2}{(1-r^2)^2 + (2\zeta r)^2}}.$$
關鍵結論：**只有當 $r > \sqrt{2}$ 時 $TR < 1$，隔振才有效**。也就是隔振器要設計得「軟」（固有頻率遠低於激振頻率）。$r < \sqrt{2}$ 時反而放大。此外在 $r > \sqrt{2}$ 區，阻尼越大 $TR$ 反而越高——所以隔振阻尼是兩難：要壓低共振峰需要阻尼，但高頻隔振又希望阻尼小，需折衷。

### 3.4 偏心質量激振（旋轉不平衡）

旋轉機械（馬達、風扇、泵）最常見的振源是**旋轉不平衡**：偏心質量 $m_0$、偏心距 $e$、轉速 $\omega$ 產生離心力 $m_0 e\omega^2$（正比於 $\omega^2$）。穩態位移振幅
$$X = \frac{m_0 e}{m}\cdot\frac{r^2}{\sqrt{(1-r^2)^2 + (2\zeta r)^2}}.$$
與諧波力不同的是分子多了 $r^2$：低速時振幅趨近 $0$，共振附近最大，**高速時 $X \to m_0 e/m$（趨於定值）**，因為激振力雖隨 $\omega^2$ 漲、慣性抵抗也隨 $\omega^2$ 漲，兩者抵消。

### 3.5 支承運動激振（基礎激振）

地基本身在動 $y(t) = Y\cos\omega t$（地震、車輛過顛簸路面），求質量的絕對運動或相對運動。質量絕對位移與基礎位移之比恰為前述的傳遞率 $TR$。汽車懸吊、地震隔震建築都是此類問題：希望 $r > \sqrt{2}$ 讓車身/樓板感受到的振動小於地面輸入。

## 4. 阻尼模型

實際阻尼機制多樣，工程上用幾種等效模型描述：

- **黏性阻尼（viscous）**：阻尼力 $\propto$ 速度，$F_d = c\dot{x}$。數學最易處理（線性），油壓阻尼器、流體膜近似屬此，前述全部公式都基於它。
- **庫倫阻尼（Coulomb / 乾摩擦）**：阻尼力為定值、方向與速度相反，$F_d = \mu N \,\text{sgn}(\dot{x})$。非線性，自由振動的振幅呈**線性遞減**（不是指數），且最終可能停在非零位置。
- **結構/遲滯阻尼（structural / hysteretic）**：源自材料內部摩擦，每週期耗能與頻率無關、與振幅平方成正比。常用複剛度 $k(1+j\eta)$ 表示，$\eta$ 為損耗因數（loss factor），在頻域分析特別方便。

工程上常把各種阻尼換算成「等效黏性阻尼」（令每週期耗能相等），以便沿用線性公式。

## 5. 多自由度系統

### 5.1 運動方程（矩陣形式）

$n$ 個自由度時，運動方程寫成矩陣形式
$$\mathbf{M}\ddot{\mathbf{x}} + \mathbf{C}\dot{\mathbf{x}} + \mathbf{K}\mathbf{x} = \mathbf{F}(t),$$
$\mathbf{M}$、$\mathbf{C}$、$\mathbf{K}$ 分別為質量、阻尼、剛度矩陣（對稱）。各自由度透過 $\mathbf{K}$（與 $\mathbf{M}$）中的非對角項**耦合**。這跟電機的狀態空間 $\dot{\mathbf{x}} = \mathbf{A}\mathbf{x} + \mathbf{B}\mathbf{u}$ 同源，只是這裡是二階形式。

### 5.2 固有頻率與模態振型

先看無阻尼自由振動 $\mathbf{M}\ddot{\mathbf{x}} + \mathbf{K}\mathbf{x} = \mathbf{0}$，假設同步簡諧解 $\mathbf{x} = \boldsymbol{\phi}e^{j\omega t}$，化為**廣義特徵值問題**
$$(\mathbf{K} - \omega^2 \mathbf{M})\boldsymbol{\phi} = \mathbf{0}.$$
非零解條件 $\det(\mathbf{K} - \omega^2\mathbf{M}) = 0$（特徵方程）解出 $n$ 個**固有頻率** $\omega_1 < \omega_2 < \dots < \omega_n$；每個 $\omega_i$ 對應一個**模態振型（mode shape）** $\boldsymbol{\phi}_i$，描述系統以該頻率振動時各自由度的相對位移比例與相位（同相或反相）。

### 5.3 模態分析與正交性

模態振型對 $\mathbf{M}$ 與 $\mathbf{K}$ 滿足**正交性**：$\boldsymbol{\phi}_i^T\mathbf{M}\boldsymbol{\phi}_j = 0$（$i\neq j$）。用模態矩陣 $\boldsymbol{\Phi} = [\boldsymbol{\phi}_1\,\dots\,\boldsymbol{\phi}_n]$ 做座標變換 $\mathbf{x} = \boldsymbol{\Phi}\mathbf{q}$，可把耦合的方程組**對角化**成 $n$ 個彼此獨立的單自由度方程（模態座標 $q_i$）。於是任何響應都是各模態的疊加——這正是 §0 心法第 2 條，與信號系統用本徵基底分解訊號異曲同工。實務上常只保留前幾個低頻模態（模態截斷），就能掌握主要動態。

### 5.4 振動吸振器（動力吸振器）

把一個輔助的彈簧—質量子系統 $(m_2, k_2)$ 附加到主系統上，調諧使其固有頻率等於激振頻率 $\sqrt{k_2/m_2} = \omega$，則主質量在該頻率的振幅**理論上可降到零**——能量都被吸振器吸走。這是 2 自由度系統的經典應用，用於消除單一頻率的擾動（如固定轉速機械、輸電線防舞動）。加入阻尼後變成「調諧質量阻尼器（TMD）」，可在較寬頻帶內減振（如台北 101 的風阻尼球）。

## 6. 連續系統振動（概念）

前面把系統當成有限個離散質量；真實的弦、桿、樑是**連續分布質量與彈性**，理論上有無限多自由度，運動方程是偏微分方程（PDE），解為無限多組固有頻率與模態（振型函數）。三個基本模型：

- **弦的橫向振動**：張力為恢復力，滿足一維**波動方程** $\dfrac{\partial^2 w}{\partial t^2} = c^2\dfrac{\partial^2 w}{\partial x^2}$，$c = \sqrt{T/\rho}$ 為波速。兩端固定弦的固有頻率為基頻的整數倍（諧波）——這正是樂器發聲原理，與信號系統的傅立葉級數天然對應。
- **桿的縱向振動**：彈性恢復力，方程同為波動方程，$c = \sqrt{E/\rho}$。
- **樑的橫向（彎曲）振動**：**Euler–Bernoulli 樑**，方程含四階空間導數 $EI\dfrac{\partial^4 w}{\partial x^4} + \rho A\dfrac{\partial^2 w}{\partial t^2} = 0$。固有頻率與模態由邊界條件（簡支、固定、自由）決定，且頻率間距**不**是整數倍。轉軸、橋樑、葉片的彎曲振動都用它分析。

連續系統的固有頻率與模態最後仍可用「無限多個單自由度模態疊加」來理解；工程上常用有限元素法（FEM）離散成多自由度問題求解。

## 7. 量測與工程應用

### 7.1 量測

- **感測器**：加速度規（accelerometer，最常用）、位移計、速度計。
- **激振**：衝擊鎚（impact hammer）或激振器（shaker）施加已知力。
- **頻譜分析**：對量到的時域訊號做 FFT 看頻率成分，找出共振峰與其對應頻率——這對電機背景完全是熟悉的信號處理流程。
- 由量測響應反推固有頻率、阻尼比、模態振型，即**模態測試（experimental modal analysis）**。

### 7.2 工程應對策略

1. **避開共振**：設計時讓系統固有頻率遠離工作激振頻率（轉速、葉片通過頻率等）。轉軸的**臨界轉速**就是轉速等於某彎曲固有頻率時的危險點，須快速通過或避開。
2. **隔振（isolation）**：用軟支承使 $r > \sqrt{2}$，阻斷振動傳遞（機器座、車輛懸吊、精密儀器平台、地震隔震）。
3. **減振（damping）**：加阻尼材料或阻尼器壓低共振峰值。
4. **吸振（absorption）**：加調諧吸振器/TMD 消除特定頻率振動。
5. **平衡（balancing）**：消除旋轉不平衡這個振源（轉子動平衡），治本。

## 碩士層級延伸（簡短）

- **實驗模態分析（EMA）**：由頻率響應函數（FRF）萃取模態參數，用於結構驗證與模型修正。
- **隨機振動（random vibration）**：激振為隨機過程，用功率譜密度（PSD）、均方響應描述（疲勞、車輛/航太環境）。
- **非線性振動**：大振幅、間隙、乾摩擦下的跳躍現象、極限環、混沌；用攝動法、數值積分分析。
- **轉子動力學（rotordynamics）**：旋轉機械的臨界轉速、陀螺效應、油膜不穩定（油膜渦動）。
- **聲振耦合與結構聲學**：振動向空氣輻射噪音的 NVH 問題。

## 博士研究方向（列表）

- 非線性能量收集（nonlinear energy harvesting）與寬頻振動取能
- 主動/半主動振動控制（壓電、磁流變阻尼器、主動隔振）
- 超材料與聲子晶體的振動帶隙設計
- 結構健康監測與基於振動的損傷識別
- 高維非線性系統的降階模型（reduced-order modeling）
- 不確定性量化與隨機有限元素法在振動預測上的應用
- 旋轉機械的非線性轉子—軸承—密封動力學
