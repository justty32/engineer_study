# 機械設計與機械元件（Machine Design / Machine Elements）

> 名詞對照見 ../中英名詞對照表.md

機械設計（Machine Design）研究「怎麼把一個受力的需求，變成一個尺寸、材料都確定、且不會壞的真實零件」。前面的科目各自回答一部分問題：材料力學告訴你「應力多大、強度夠不夠」（強度），機動學告訴你「桿件怎麼動、傳什麼力」（運動），工程材料與製造告訴你「這材料能不能做、做得出來嗎」（可行性）。機械設計把這三者整合起來，加上一個工程上最關鍵的觀念——**安全係數**——去回答：「軸要多粗？螺栓要幾顆、鎖多緊？彈簧要繞幾圈？軸承能撐幾小時？齒輪的齒會不會斷？」這是把分析變成可用零件的核心學問，也是機械工程師日常真正在做的事。

## 0. 本科在本系的定位

- **整合三條上游**：
  - [材料力學](../02-固體力學/04-材料力學.md)（強度）：提供應力公式 $\sigma=N/A$、$\sigma=Mc/I$、$\tau=Tr/J$，以及材料的 $\sigma_y$、$\sigma_u$。
  - [機動學](09-機動學.md)（運動）：提供零件承受的外力、速度、加速度與傳遞的力矩。
  - [工程材料](../02-固體力學/05-工程材料.md)與[製造與製程](11-製造與製程.md)（可行性）：提供材料選擇與「做得出來、做得便宜」的限制。
- **核心心法（全科都是這兩件事的延伸）**：
  1. **失效分析（failure analysis）**：先想清楚「這個零件會怎麼壞」——是被拉斷、被壓垮、被磨耗、被反覆載重疲勞斷裂，還是變形過大不能用？不同失效模式用不同準則判斷。**設計的第一步永遠是問「它會怎麼壞」。**
  2. **安全係數（factor of safety, FoS）**：真實世界充滿不確定性（材料有瑕疵、載重比預估大、製造有公差、環境會劣化）。設計時刻意讓「材料能耐的」大於「實際承受的」，兩者比值即安全係數 $n$。
- 整科邏輯鏈：**確定載重 →（材料力學）算應力 → 判斷失效模式 → 選失效準則比對材料強度 → 套安全係數 → 反推尺寸／選型**。

## 1. 設計流程與規格、安全係數與設計準則

### 1.1 設計流程
工程設計是反覆迭代的循環：

1. **辨識需求**：要解決什麼問題、誰來用、用在哪。
2. **定義規格（specification）**：把模糊需求變成可量化的條件——功能、預期壽命（循環數或時數）、最大負載、成本上限、重量、工作溫度、環境（潮濕、腐蝕、振動）。
3. **概念設計**：發想多種可行方案（用齒輪還是皮帶？焊接還是螺栓？）。
4. **分析與選型（本科重點）**：對每個元件做應力／壽命計算，把尺寸定下來或從型錄選現成件。
5. **細部設計與圖面**：公差、配合、表面處理、組裝順序。
6. **原型、測試與修改**：迭代回上面任一步。

本科聚焦在第 4 步：給定載重與材料，把每個元件的尺寸定下來，或選出合適的標準件。注意設計很少「一次到位」——多數時間在迭代與權衡（trade-off）。

### 1.2 安全係數與設計準則
安全係數定義為「強度」除以「應力」：

$$n = \frac{\text{材料強度（如 }\sigma_y\text{ 或 }\sigma_u\text{）}}{\text{設計應力}\;\sigma}$$

- 延性材料常以降伏強度 $\sigma_y$ 為準（不希望永久變形）；脆性材料以極限強度 $\sigma_u$ 為準（會直接斷）。
- $n$ 的選取憑經驗與規範：載重與材料都很確定、後果輕微時 $n\approx1.5\sim2$；不確定性大、後果嚴重（如人身安全、航太）時 $n$ 可達 $3\sim5$ 以上。
- **設計準則（design criterion）**＝把「失效模式」寫成可計算的不等式，例如「von Mises 應力 $\le \sigma_y / n$」。本科後續每種元件都是在套用某個設計準則。

## 2. 靜態失效理論（受固定載重下的破壞判斷）

材料力學的 $\sigma_y$ 來自「單軸拉伸」試驗，但實際零件常處於多軸應力（同時有彎、扭、壓）。失效理論的任務是：**把多軸應力狀態換算成一個等效的單一數值，再和單軸試驗的 $\sigma_y$ 或 $\sigma_u$ 比較。**

### 2.1 延性材料：最大畸變能（von Mises）理論
延性材料（多數鋼、鋁）的降伏是由「剪切造成的形狀改變」主導，而非體積改變。von Mises 把多軸應力濃縮成一個**等效應力（von Mises stress）** $\sigma'$，平面應力時：

$$\sigma' = \sqrt{\sigma_x^2 - \sigma_x\sigma_y + \sigma_y^2 + 3\tau_{xy}^2}$$

降伏條件：$\sigma' \ge \sigma_y$。這是延性材料**最準、最常用**的理論，設計實務的首選。

### 2.2 延性材料：最大剪應力（Tresca / Guest）理論
認為降伏由最大剪應力主導：$\tau_{\max}=\dfrac{\sigma_1-\sigma_3}{2}\ge\dfrac{\sigma_y}{2}$。比 von Mises **更保守**（預測的安全裕度較小），計算較簡單，常用於規範與快速估算。

### 2.3 脆性材料：Mohr 理論
脆性材料（鑄鐵、陶瓷）幾乎不降伏就直接斷裂，且抗壓遠強於抗拉。用**最大正向應力理論**或更精確的**庫倫–莫爾（Coulomb-Mohr）理論**：分別針對抗拉強度 $S_{ut}$ 與抗壓強度 $S_{uc}$ 設限。直覺：脆性材料怕「拉」，所以盯著最大主應力是否超過抗拉強度。

## 3. 變動載重與疲勞（Fatigue）

機械零件多在反覆載重下工作（旋轉軸、彈簧、齒輪）。即使應力遠低於 $\sigma_y$，反覆夠多次也會突然斷裂，這就是**疲勞**——機械失效中最常見、也最危險的一種（看起來好好的卻無預警斷掉）。

**為什麼會疲勞？** 反覆載重在材料微觀缺陷或表面刮痕處不斷推擠，先形成微裂縫，每一次循環讓裂縫前進一點點，累積到剩餘截面撐不住時瞬間斷裂。斷面常呈「貝殼狀紋路（海灘紋）」加上一塊粗糙的最終斷裂區，是疲勞的標誌。重點直覺：**疲勞是「裂縫慢慢長大」的過程，所以表面品質、刮痕、尖角這些「裂縫起點」極為關鍵。**

### 3.1 S–N 曲線與耐久限
拿一批試棒施加不同振幅的循環應力，記錄「斷裂前的循環數 $N$」，把「應力振幅 $S$」對「壽命 $N$（對數座標）」作圖，即 **S–N 曲線**：應力越高，壽命越短。$10^3$ 次以下稱**低週疲勞**（高應力、伴隨塑性），$10^3\sim10^6$ 次稱**高週疲勞**（本科主要對象）。

- **耐久限 / 疲勞極限（endurance limit）** $S_e$：許多鋼材存在一個門檻應力，低於它時可承受「無限次」循環（曲線變水平，約 $10^6$ 次後）。鋁等材料則無明顯耐久限，曲線持續下降。
- 鋼的粗略估計：未修正耐久限 $S_e' \approx 0.5\,S_{ut}$（當 $S_{ut}\le 1400$ MPa）。

### 3.2 應力集中與耐久限修正係數
- **應力集中（stress concentration）**：孔、溝槽、肩角等幾何不連續處，局部應力會被放大，乘上**應力集中因子** $K_t$。疲勞時用**疲勞應力集中因子** $K_f = 1 + q(K_t-1)$，其中 $q$ 為**缺口敏感度（notch sensitivity）**（$0\sim1$，脆性材料趨近 1）。
- 真實零件的耐久限不等於試棒值，要用 **Marin 修正係數**逐項打折：

$$S_e = k_a\,k_b\,k_c\,k_d\,k_e\,S_e'$$

| 係數 | 意義 | 直覺 |
| --- | --- | --- |
| $k_a$ 表面 | 表面粗糙度與處理 | 粗糙面有微裂縫，是疲勞起點 |
| $k_b$ 尺寸 | 零件大小 | 越大內含瑕疵機率越高 |
| $k_c$ 載重 | 拉壓／彎曲／扭轉 | 不同載重型態折減不同 |
| $k_d$ 溫度 | 工作溫度 | 高溫使材料劣化 |
| $k_e$ 可靠度 | 要求的存活率 | 要 99.9% 不壞就再打折 |

### 3.3 平均應力的影響：Goodman / Soderberg / Gerber
真實載重常是「平均應力 $\sigma_m$ + 交變應力 $\sigma_a$」（如重力下的旋轉軸）。$\sigma_m$ 為拉時會加速疲勞。用「破壞線圖（$\sigma_a$–$\sigma_m$ 平面）」判斷安全：

- **修正 Goodman 準則**：$\dfrac{\sigma_a}{S_e} + \dfrac{\sigma_m}{S_{ut}} = \dfrac{1}{n}$，直線、保守、最常用。
- **Soderberg 準則**：把 $S_{ut}$ 換成 $S_y$，$\dfrac{\sigma_a}{S_e}+\dfrac{\sigma_m}{S_y}=\dfrac1n$，**最保守**，並天然防止降伏。
- **Gerber 準則**：拋物線，最貼近實驗數據但較不保守。

### 3.4 累積損傷：Miner 法則
載重振幅常變化。**Palmgren-Miner 線性累積損傷**假設每段振幅 $i$（壽命 $N_i$）施加了 $n_i$ 次循環，各自「消耗壽命」$n_i/N_i$，總損傷相加達 1 即破壞：

$$\sum_i \frac{n_i}{N_i} = 1$$

優點是簡單；缺點是忽略載重「順序」的影響，且對低於耐久限的循環當作零損傷，估計可能偏樂觀或保守。

## 4. 軸的設計（Shaft Design）

軸同時承受**彎矩**（齒輪／皮帶的橫向力造成）與**扭矩**（傳遞動力），是彎扭組合載重的典型。

- 圓軸表面的彎曲應力 $\sigma=\dfrac{32M}{\pi d^3}$、扭轉剪應力 $\tau=\dfrac{16T}{\pi d^3}$，再用 von Mises 合成。
- 軸多為旋轉，彎矩造成的是**完全反向的交變應力**（每轉一圈該點由拉變壓），扭矩多為穩定（平均）應力——天生是疲勞問題。
- **ASME 軸方程**（結合 Goodman/von Mises 與安全係數，求所需直徑 $d$）的常見形式：

$$d = \left\{ \frac{32\,n}{\pi}\sqrt{\left(\frac{K_f\,M_a}{S_e}\right)^2 + \frac{3}{4}\left(\frac{T_m}{S_y}\right)^2}\,\right\}^{1/3}$$

直覺：交變彎矩 $M_a$ 拿去比耐久限 $S_e$（因為它造成疲勞），穩定扭矩 $T_m$ 拿去比降伏 $S_y$，合成後乘安全係數 $n$，開立方根反推直徑（應力與 $d^3$ 成反比，所以直徑開立方根）。設計軸時還要：在鍵槽、軸肩、油孔等應力集中處加 $K_f$；用圓角過渡避免尖角；以及檢查撓度與臨界轉速（避免共振）。

## 5. 接頭：螺栓與焊接

### 5.1 螺栓接頭與預載（Bolted Joints & Preload）
鎖螺栓不是「鎖緊就好」，關鍵是**預載 $F_i$**（鎖緊產生的初始張力）。被夾的構件像一根很硬的彈簧。當外力 $P$ 作用時，由於螺栓勁度 $k_b$ 與被夾件勁度 $k_m$ 並聯分擔，**外力只有一小部分落到螺栓上**：

$$F_b = F_i + \frac{k_b}{k_b+k_m}\,P, \qquad F_m = F_i - \frac{k_m}{k_b+k_m}\,P$$

由於通常 $k_m \gg k_b$（被夾的厚金屬比細螺栓硬得多），多數外力被「卸到被夾件壓縮量的鬆弛」上，螺栓張力增加很少——這正是高預載能大幅提升螺栓**疲勞**壽命的原因（螺栓承受的交變應力被壓得很小）。預載也必須足夠大，避免外力把接合面拉開（**接頭分離 / joint separation**）；一旦分離，外力就全部直接灌進螺栓，疲勞壽命驟降。實務上以扭力扳手控制鎖緊扭矩來間接設定預載（$T \approx K\,F_i\,d$，$K$ 為扭矩係數約 0.2）。

### 5.2 焊接接頭強度（Welded Joints）
焊道強度設計的核心是**喉部面積（throat area）**——角焊道（fillet weld）的最薄處 $t=0.707h$（$h$ 為焊腳長）。把焊道視為承受剪力的薄帶，剪應力 $\tau=\dfrac{F}{0.707\,h\,L}$，再與焊材的許可剪應力比較。

## 6. 壓縮螺旋彈簧（Helical Compression Springs）

彈簧靠線材的**扭轉**儲能。三個關鍵公式：

- **彈簧常數（spring rate）** $k=\dfrac{Gd^4}{8D^3n_a}$（$d$ 線徑、$D$ 平均圈徑、$n_a$ 有效圈數、$G$ 剪切模數）。直覺：線徑 $d$ 的**四次方**影響極大（線徑加倍硬度變 16 倍），圈徑 $D$ 越大或圈數越多則越軟。$F=k\,\delta$（虎克定律的彈簧版）。
- **線材剪應力** $\tau = K_w\,\dfrac{8FD}{\pi d^3}$，其中 **Wahl 因子** $K_w=\dfrac{4C-1}{4C-4}+\dfrac{0.615}{C}$（彈簧指數 $C=D/d$），修正曲率與直接剪力使內側應力偏高（疲勞常從內側開始）。
- 設計還要檢查實體高度、挫曲（細長彈簧會側彎）與疲勞。

## 7. 軸承

### 7.1 滾動軸承：額定壽命 $L_{10}$（ISO 281）
滾動軸承（滾珠、滾子）的失效主要是滾動面的**接觸疲勞剝落**，壽命有統計分散性。

- **基本額定壽命 $L_{10}$**：90% 的同型軸承能達到的壽命（即 10% 失效），單位百萬轉：

$$L_{10} = \left(\frac{C}{P}\right)^p \quad(\times 10^6 \text{ 轉})$$

其中 $C$ 為**基本動額定負載**（廠商提供）、$P$ 為**當量動負載**、指數 $p=3$（滾珠）或 $10/3$（滾子）。
- **當量負載**：把徑向 $F_r$ 與軸向 $F_a$ 合成 $P = X F_r + Y F_a$。
- 選型流程：算出所需壽命與負載 → 反推所需 $C$ → 查型錄選軸承。

### 7.2 滑動軸承（概念）
滑動軸承靠軸與軸承間的**油膜**承載（流體動壓潤滑）。轉動時把潤滑油「楔」入間隙形成壓力膜，使金屬不直接接觸（近乎零磨耗）。設計關鍵：適當間隙、黏度、轉速與負載，使油膜不破。重要無因次參數為 **Sommerfeld 數**（綜合黏度、轉速、負載與幾何）。

## 8. 齒輪強度

### 8.1 Lewis 彎曲方程
把單一齒視為**懸臂樑**，齒根受切線力 $W_t$ 造成的彎曲應力：

$$\sigma = \frac{W_t}{F\,m\,Y}$$

（$F$ 齒寬、$m$ 模數、$Y$ 為 **Lewis 形狀因子**，反映齒形）。直覺：齒越寬、模數越大、齒形越飽滿（$Y$ 大），齒根應力越小、越不易斷。實務還會加上動態效應的速度因子與應力集中。

### 8.2 接觸應力（概念）
齒面在嚙合線上是「兩圓柱相壓」，接觸區產生很高的**赫茲接觸應力（Hertzian contact stress）**，反覆作用導致齒面**點蝕（pitting）**。齒輪須同時通過「齒根彎曲（會斷齒）」與「齒面接觸（會點蝕）」兩道強度檢核，這也是 AGMA 規範的兩大支柱。

## 9. 聯軸器、離合器與撓性傳動

- **聯軸器（coupling）**：永久連接兩軸傳遞扭矩。剛性聯軸器要求對心良好；撓性聯軸器（如撓性盤、橡膠塊）可容忍少量偏心與角度誤差並吸收振動。
- **離合器（clutch）**：可控地接合／分離兩軸，靠摩擦面傳遞扭矩，設計重點是摩擦力矩容量與散熱（如車輛離合器）。
- **皮帶傳動（belt drive）**：靠摩擦或嚙合（同步帶）傳力。緊邊與鬆邊張力差 $T_1-T_2$ 決定傳遞的扭矩，受 $T_1/T_2 \le e^{\mu\theta}$（包覆角 $\theta$、摩擦係數 $\mu$）限制；優點是吸震、安靜、成本低、能容許軸距大。
- **鏈條傳動（chain drive）**：齒嚙合無滑動，傳遞比精確、效率高、可承大力矩，但較吵且需潤滑（如自行車、機車後傳動）。

- 機械振動：軸的彎曲勁度與軸承支撐影響臨界轉速；彈簧設計關乎共振。

## 碩士層級延伸（簡短）

- **最佳化設計（design optimization）**：在強度、重量、成本等約束下用數學規劃求最佳尺寸（軟體背景者可從數值最佳化切入，較熟悉）。
- **可靠度設計（reliability-based design）**：把載重與強度都當機率分布，以「干涉法」算失效機率，取代固定安全係數。
- **疲勞壽命預測（fatigue life prediction）**：含應變–壽命（$\varepsilon$–$N$）法、斷裂力學裂縫成長（Paris 法則）、雨流計數與隨機載重譜。
- **設計方法論（design methodology）**：公理設計、TRIZ、為製造／組裝而設計（DFM/DFA）、穩健設計（Taguchi）。

## 博士研究方向（列表）

- 多軸與變幅疲勞之損傷準則與壽命模型
- 增材製造件的設計（拓樸最佳化、點陣結構、殘留應力與疲勞）
- 機器學習輔助的零件選型、代理模型與生成式設計
- 接觸力學、磨潤（tribology）與表面工程之壽命延長
- 不確定性量化與可靠度／韌性設計
- 輕量化與多材料／複合結構接頭設計
- 數位孿生與基於監測資料的剩餘壽命預估
