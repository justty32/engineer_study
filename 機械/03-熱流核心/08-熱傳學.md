# 熱傳學（Heat Transfer）

> 機械第三階段第 3 科。
> 名詞對照見 [中英名詞對照表 → 熱傳學](../中英名詞對照表.md#熱傳學)。

## 這科在學什麼

熱力學告訴你「能量會不會傳、最多傳多少、平衡在哪」，但**不告訴你傳多快**。熱傳學補上這個缺口：**熱以什麼機制、多快的速率傳遞**。CPU 散熱、引擎冷卻、保溫杯、熱交換器、建築隔熱、電子封裝——全靠它。

熱傳只有**三種機制**，整科就是把這三種講透並組合：
- **傳導（conduction）**：靠分子/電子接力，固體內部與靜止流體。
- **對流（convection）**：靠流體流動帶走熱，與流體力學深度耦合。
- **輻射（radiation）**：靠電磁波，不需介質（太陽到地球）。

對你而言有個好用的橋樑：熱傳問題常用**「熱阻」電路類比**——溫差像電壓、熱流像電流、熱阻像電阻，串並聯一樣算。這對電機背景幾乎是免費的直覺。

## 第 1 章　傳導（Conduction）

### 1.1 傅立葉定律
$$\dot Q=-kA\frac{dT}{dx}$$
熱流正比於**溫度梯度**與面積，方向由高溫到低溫。$k$＝**熱傳導係數**（金屬大、絕熱材小、氣體最小）。

### 1.2 平壁與熱阻類比
一維穩態平壁：$\dot Q=\dfrac{kA}{L}\Delta T=\dfrac{\Delta T}{R_\text{cond}}$，其中**傳導熱阻** $R_\text{cond}=\dfrac{L}{kA}$。
- **多層牆**＝熱阻**串聯**（牆 + 保溫層 + 牆）。
- 並排不同材料＝**並聯**。
- **接觸熱阻**：兩固體接觸面不平整，多一層阻抗。
- 圓管徑向傳導：$R=\dfrac{\ln(r_2/r_1)}{2\pi k L}$（log 形式，因面積隨半徑變）。

> 完全的電路類比：$\dot Q\leftrightarrow I$、$\Delta T\leftrightarrow V$、$R_\text{th}\leftrightarrow R$。散熱設計就是「降低總熱阻」。

### 1.3 鰭片（Fin）
為增加散熱面積而伸出的金屬片（散熱器、機車汽缸）。沿鰭片解傳導方程，得到溫度沿長度遞減；用**鰭片效率**衡量「加了鰭片實際多散多少熱」。鰭片要又長又薄又用高 $k$ 材料才有效。

### 1.4 暫態傳導
物體突然遇冷/熱，溫度隨時間變化：
- **集總容量法（lumped capacitance）**：當內部傳導遠快於表面對流（**畢歐數 $Bi=hL_c/k\ll0.1$**）時，物體可視為均溫，溫度指數式趨近環境：$T(t)$ 呈 $e^{-t/\tau}$。又是一階系統，跟 RC 充放電同型。
- $Bi$ 大時內部溫度不均，要解偏微分方程（Heisler 圖或數值）。

## 第 2 章　對流（Convection）

### 2.1 牛頓冷卻定律
$$\dot Q=hA(T_s-T_\infty)$$
$h$＝**對流熱傳係數**，不是材料常數，而是**流場決定**（流速、流體性質、幾何、層流/亂流）——這就是對流與流體力學綁在一起的原因。對流熱阻 $R_\text{conv}=\dfrac{1}{hA}$。

### 2.2 強制 vs 自然對流
- **強制對流（forced）**：靠風扇/泵驅動流動（電腦風扇、水冷）。$h$ 較大。
- **自然對流（natural/free）**：靠溫差造成浮力流動（暖氣片、散熱片無風扇）。$h$ 較小。

### 2.3 用無因次數求 $h$
對流分析的核心是查/算無因次關係式：
- **紐塞數 $Nu=\dfrac{hL}{k}$**：無因次的對流強度（對流 vs 純傳導的比值），求出 $Nu$ 就得到 $h$。
- **普朗特數 $Pr=\dfrac{\nu}{\alpha}$**：動量擴散 vs 熱擴散的比值（流體性質）。
- 經驗式形如 $Nu=f(Re,Pr)$（強制對流）或 $Nu=f(Ra,Pr)$（自然對流，$Ra$＝瑞利數）。
- **熱邊界層**：與流速邊界層平行的概念，溫度從壁面值過渡到主流值的薄層。

### 2.4 沸騰與凝結
相變對流，$h$ 極大（潛熱效應），是冷凍、鍋爐、熱管的關鍵；有不同沸騰區（核沸騰最有效）。

## 第 3 章　輻射（Radiation）

### 3.1 史蒂芬–波茲曼定律
所有物體都靠電磁波輻射熱，功率隨**絕對溫度四次方**爆增：
$$\dot Q=\varepsilon\sigma A T^4$$
$\sigma$＝史蒂芬–波茲曼常數，$\varepsilon$＝**輻射率（emissivity）**（黑體 = 1，亮金屬接近 0）。$T^4$ 使輻射在高溫時主宰（火焰、太陽、白熾燈、熔爐）。

### 3.2 表面性質與灰體
- **黑體**：理想吸收/放射體；**克希荷夫定律**：good absorber = good emitter。
- 實際表面用輻射率、吸收率、反射率描述；保溫瓶鍍銀就是把 $\varepsilon$ 壓到很低。

### 3.3 表面間輻射交換
兩表面間淨輻射要考慮**視角因子（view factor）**（彼此「看得到多少」）與表面輻射率，組成輻射熱阻網路求解。

## 第 4 章　綜合：熱交換器

實務上三機制常同時發生（如冷氣散熱片：管內對流 + 管壁傳導 + 鰭片對流 + 輻射）。**整體熱傳係數 $U$** 把串聯熱阻合併：$\dfrac{1}{UA}=\sum R$。

**熱交換器（heat exchanger）**讓兩股不同溫度流體交換熱而不混合（散熱器、鍋爐、冷凝器、板式換熱器）：
- 流向：**順流、逆流、錯流**；**逆流效率最高**。
- 兩種設計分析法：
  - **對數平均溫差法（LMTD）**：已知進出口溫度求面積。
  - **效能–NTU 法（ε-NTU）**：已知面積求出口溫度。

## 小結

- 熱傳補足熱力學沒講的「**速率**」，只有三機制：傳導、對流、輻射。
- **傳導**：傅立葉定律 $\dot Q=-kA\,dT/dx$，善用**熱阻電路類比**（串並聯）；鰭片增面積；暫態看畢歐數（小→集總一階系統）。
- **對流**：牛頓冷卻 $\dot Q=hA\Delta T$，$h$ 由流場決定，靠 $Nu=f(Re,Pr)$ 求；分強制/自然。
- **輻射**：$\dot Q=\varepsilon\sigma AT^4$，高溫主宰，受輻射率與視角因子控制。
- 實務組合成**整體熱傳係數 $U$** 與**熱交換器**（LMTD / ε-NTU，逆流最佳）。
