# 工程圖學（Engineering Drawing / Graphics）

> 機械第一階段第 1 科。
> 名詞對照見 [中英名詞對照表 → 工程圖學](../中英名詞對照表.md#工程圖學)。

## 這科在學什麼

工程圖是工程師之間的**共同語言**。一個零件從設計者腦中到工廠加工，中間靠的是圖面把「形狀、尺寸、精度、材料、表面狀態」毫不含糊地傳達出去。對軟體背景的你，可以類比成：圖學是機械世界的「規格書 + 介面定義」，而且有嚴格的國際標準（CNS、ISO、ANSI/ASME），讓任何人讀同一張圖得到同一個零件。

重點不是會用尺畫線，而是建立**空間 ↔ 平面**的轉換直覺：3D 的物體如何用幾張 2D 圖完整描述，反過來看到 2D 圖能在腦中重建 3D。

## 第 1 章　投影原理

### 1.1 正投影（Orthographic Projection）
想像光線**垂直**打在物體上、投影到後方的平面，得到該方向的輪廓——這就是正投影。因為投影線互相平行且垂直投影面，所以**不會變形、可量測尺寸**，這是工程圖的基礎。

對比：
- **透視投影（perspective）**：像照片，有近大遠小，適合表現外觀但不能量尺寸。
- **斜投影 / 等角投影（isometric）**：用一張圖同時表現三個面，適合「立體示意」，但比例會失真。

### 1.2 三視圖（Three Views）
一個物體用三個正投影方向描述就夠：
- **前視圖（front view）**：主視方向，選最能表現特徵的面。
- **俯視圖（top view）**：從上往下看。
- **側視圖（side view）**：從左或右看。

三視圖之間有嚴格的對齊關係：
- 前視與俯視「**寬度**對齊」（左右）。
- 前視與側視「**高度**對齊」（上下）。
- 俯視與側視「**深度**相等」（常用 45° 線轉換）。

這個「長對正、高平齊、寬相等」是讀圖與畫圖的鐵律。

### 1.3 第一角法 vs 第三角法
同樣三視圖，視圖的**擺放位置**有兩種制度：
- **第一角法（first-angle）**：物體在觀察者與投影面之間（像物體在玻璃前）。歐洲、中國、台灣 CNS 常用。俯視圖放在前視圖**下方**會放到上方……要小心：第一角法俯視圖在前視圖**下方**，但內容是「從上看」——實務上記法是「視圖放在物體被推開的另一側」。
- **第三角法（third-angle）**：投影面在觀察者與物體之間（像物體在玻璃後）。美國、日本常用。俯視圖在前視圖**上方**。

圖框內會放一個**識別符號**（截頭圓錐的兩個視圖）標明用哪一角法，避免讀錯。

## 第 2 章　表達手法

### 2.1 剖視圖（Sectional View）
物體內部有孔、槽、肉厚時，外形輪廓用虛線表示會很亂。**剖視**是假想用一把刀把物體切開、移走前半，露出內部實體，被切到的實體斷面打上**剖面線（hatching，45° 等距斜線）**。常見：
- 全剖、半剖（一半剖一半保留外形）、局部剖、旋轉剖、階梯剖。
- 慣例：肋（rib）、輪輻、螺栓、軸這類沿軸線剖開時**不打剖面線**，以免誤判為實心塊。

### 2.2 輔助視圖與局部放大
- **輔助視圖（auxiliary view）**：物體有斜面時，正投影會失真，改用垂直該斜面的方向投影，才能看到真實形狀與尺寸。
- **局部放大圖（detail view）**：小特徵（如退刀槽）放大畫，標註比例如 5:1。

### 2.3 立體圖
- **等角圖（isometric）**：三軸夾 120°，同一比例，工程常用的立體示意。
- **分解圖（exploded view）**：組裝關係，沿裝配軸線拉開零件，標件號對應零件表（BOM）。

## 第 3 章　尺寸與精度

這一章是圖學「資訊量」最高、也最工程的部分。

### 3.1 尺寸標註（Dimensioning）
- 尺寸線、延伸線、箭頭、數字；直徑用 $\varnothing$、半徑用 R、球面用 S。
- 原則：**功能優先**（從設計基準標）、不重複標註、不過度標註（over-dimensioning 會矛盾）。
- 基準（datum）：選一個面或線當量測起點，串接式 vs 並列式標註影響誤差累積。

### 3.2 公差（Tolerance）
真實加工不可能做到「剛好」，所以給一個允許範圍。
- **尺寸公差**：例 $\varnothing 20^{+0.021}_{0}$，表示孔徑在 20.000～20.021 mm 之間合格。
- **配合（fit）**：軸與孔的尺寸關係，分三類：
  - **餘隙配合（clearance）**：孔恆大於軸，可自由轉動/滑動。
  - **過盈配合（interference）**：軸恆大於孔，要壓入，靠摩擦固定。
  - **過渡配合（transition）**：可能緊可能鬆。
  - ISO 制度用「基孔制 H / 基軸制 h + 等級數字」表示，如 $\varnothing 20\,H7/g6$。

### 3.3 幾何公差（GD&T，Geometric Dimensioning and Tolerancing）
尺寸公差管不到「形狀歪不歪、位置正不正」。幾何公差補上這塊，分四類：
- **形狀**：真直度、真平度、真圓度、圓柱度。
- **方向**：平行度、垂直度、傾斜度。
- **位置**：位置度、同心度、對稱度。
- **偏轉**：圓偏轉、總偏轉。

用**特徵控制框（feature control frame）**標註：符號＋公差值＋基準。這是現代精密製造與品檢的核心語言。

### 3.4 表面粗糙度（Surface Roughness）
用 $Ra$（算術平均粗糙度，單位 µm）等參數，標在表面符號上，告訴加工者要磨到多光滑。太光滑＝成本高，要剛好夠用。

## 第 4 章　從手繪到 CAD

- **CAD（Computer-Aided Design）**：2D 繪圖（AutoCAD）到 3D 參數化建模（SolidWorks、Creo、CATIA、Fusion 360、NX）。
- **參數化 / 特徵式建模**：模型由「草圖 + 特徵（擠出、旋轉、倒角、孔）」依序建構，改一個尺寸全模型連動更新——對軟體人來說，這就是「宣告式 + 相依圖（dependency graph）」的幾何版。
- **裝配（assembly）**：用「配合條件（mate / constraint）」把零件組起來，可做干涉檢查與運動模擬。
- **工程圖出圖**：3D 模型自動投影出 2D 三視圖、剖視、標尺寸，保證圖模一致。
- 往下游接 **CAM（製造）、CAE（分析，如有限元素）**，形成 CAD/CAM/CAE 流程。

## 小結

- 工程圖＝無歧義的零件規格，靠**正投影**保證可量測。
- **三視圖**用「長對正、高平齊、寬相等」互相鎖定；注意第一/第三角法差異。
- **剖視**處理內部；**輔助視圖**處理斜面。
- 真正的工程含量在**尺寸、公差、配合、GD&T、表面粗糙度**——它們定義零件「做到多準才算合格」。
- 現代以 **3D 參數化 CAD** 為核心，向下串接製造與分析。
