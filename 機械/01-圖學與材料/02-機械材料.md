# 機械材料（Engineering Materials / Materials Science）

> 機械第一階段第 2 科。
> 名詞對照見 [中英名詞對照表 → 機械材料](../中英名詞對照表.md#機械材料).

## 這科在學什麼

材料學回答一個核心問題：**為什麼這個材料有這種性質，以及我能不能改變它？** 機械工程師選材料時要同時考慮強度、韌性、重量、耐溫、耐蝕、成本與可加工性。這科建立「**結構 → 性質 → 製程 → 性能**」四者互相牽動的觀念：原子怎麼排列、有什麼缺陷、經過什麼熱處理，決定了它在機器裡能不能撐住。

對電機背景的你，半導體（矽晶格、摻雜、能帶）其實就是材料科學的一支；這裡換成「金屬怎麼變硬變韌」，底層思路（晶體、缺陷、相變）是相通的。

## 第 1 章　原子鍵結與晶體結構

### 1.1 四種鍵結
- **金屬鍵**：電子海，故導電導熱好、可塑性高（原子層可滑動）。
- **離子鍵 / 共價鍵**：陶瓷的主角，鍵強且方向性強，故硬、脆、耐高溫、不導電。
- **凡德瓦力（分子間）**：高分子鏈之間的弱鍵，故高分子軟、熔點低。

鍵結類型直接決定大方向性質：金屬韌、陶瓷硬脆、塑膠軟。

### 1.2 晶體結構（Crystal Structure）
金屬原子規則堆疊，常見三種晶格：
- **BCC（體心立方）**：如 α-鐵、鎢。
- **FCC（面心立方）**：如鋁、銅、γ-鐵、奧斯田鐵。堆疊較密、滑移系統多，**延展性好**。
- **HCP（六方最密）**：如鎂、鈦、鋅。滑移系統少，**較脆、方向性強**。

同一元素在不同溫度可有不同晶格（**同素異形 / allotropy**），鐵就是 BCC↔FCC 互換——這是鋼鐵熱處理能成立的根本原因。

### 1.3 晶體缺陷（Defects）
真實晶體不完美，而**缺陷正是性質可調的關鍵**：
- **點缺陷**：空孔、間隙原子、置換原子（合金化）。
- **線缺陷＝差排（dislocation）**：最重要。金屬會塑性變形（不是脆斷），正是因為差排能在晶格中「逐排滑移」，像地毯打一個皺褶推過去比整塊拖容易。
- **面缺陷＝晶界（grain boundary）**：多晶材料中晶粒之間的界面，會**阻擋差排移動**。

**強化金屬的本質＝設法阻礙差排移動**：晶粒細化（更多晶界，Hall–Petch）、固溶強化、加工硬化（差排互相纏結）、析出強化（第二相顆粒釘住差排）。

## 第 2 章　機械性質

### 2.1 拉伸試驗（Tensile Test）
把試棒拉斷，量到的**應力–應變曲線**是材料的「身分證」：
- **彈性區**：應力與應變成正比，斜率＝**楊氏模數 $E$**（剛性，越大越不易變形）。卸載會回彈。
- **降伏強度 $\sigma_y$（yield strength）**：超過此點開始**永久（塑性）變形**。設計通常不准超過降伏。
- **抗拉強度 $\sigma_{UTS}$（tensile strength）**：曲線最高點。
- **延伸率 / 斷面縮率**：衡量**延性（ductility）**——斷之前能拉多長。
- 曲線下面積 ≈ **韌性（toughness）**，吸收能量的能力。

關鍵區分：**剛性（E，難變形）≠ 強度（σ，難壞）≠ 韌性（難斷裂）≠ 硬度（難刮）**。鋼比鋁強也比鋁剛（E 約 3 倍），但設計輕量化時鋁的「比強度」可能更好。

### 2.2 硬度（Hardness）
壓痕法量「抗局部塑性變形」：布氏（HB）、洛氏（HRC）、維氏（HV）。硬度與抗拉強度大致正相關，量測快，常做品管。

### 2.3 衝擊與韌性、疲勞、潛變
- **衝擊試驗（Charpy）**：量缺口試棒被擺錘打斷吸收的能量；某些 BCC 鋼有**延性–脆性轉變溫度（DBTT）**，低溫變脆（鐵達尼號鋼板、二戰自由輪斷裂就是這原因）。
- **疲勞（fatigue）**：在遠低於降伏的應力下**反覆受力**也會壞，是機械零件最常見的失效。鋼有**疲勞極限**（低於某應力可無限次），鋁則沒有。詳見機械設計。
- **潛變（creep）**：高溫下持續受力會**緩慢持續變形**，渦輪葉片、鍋爐管設計的關鍵。

## 第 3 章　相圖與鐵碳系統

### 3.1 相與相圖（Phase Diagram）
**相（phase）**＝結構與成分均勻的區域。相圖是「成分 vs 溫度」的地圖，告訴你某合金在某溫度下有哪些相、各佔多少（**槓桿法則 lever rule** 算比例）。
- **共晶（eutectic）**：液體一次冷卻同時析出兩種固相（如焊錫 Sn-Pb）。
- **固溶體 vs 化合物**：合金元素溶進去，或形成金屬間化合物。

### 3.2 鐵–碳相圖（最重要）
鋼鐵是機械工程的主力，鐵碳相圖是材料學的核心案例：
- **肥粒鐵（ferrite, α）**：BCC 的鐵，軟、韌、含碳極少。
- **沃斯田鐵 / 奧斯田鐵（austenite, γ）**：FCC，高溫相，溶碳能力強。
- **雪明碳鐵（cementite, Fe₃C）**：硬而脆的化合物。
- **波來鐵（pearlite）**：肥粒鐵與雪明碳鐵的層狀混合，強韌兼具。
- 碳含量分界：**鋼（<2.1% C）vs 鑄鐵（>2.1% C）**。

### 3.3 熱處理（Heat Treatment）
利用鐵的同素異形與冷卻速率控制最終組織——這是金屬「同成分、不同性能」的魔法：
- **退火（annealing）**：加熱後**爐冷（慢）**，組織平衡、變軟、消除內應力，利於加工。
- **正常化（normalizing）**：空冷，較細晶、較強。
- **淬火（quenching）**：奧斯田鐵化後**急冷（水/油）**，碳來不及擴散，FCC 強制變成過飽和的 BCT＝**麻田散鐵（martensite）**，非常硬但很脆。
- **回火（tempering）**：淬火後再低溫加熱，犧牲一點硬度換回韌性。**「淬火＋回火」**是刀具、齒輪、軸件的標準組合。
- **TTT / CCT 曲線**：描述「溫度–時間–相變」，決定淬火要多快才能得到麻田散鐵。
- 表面處理：**滲碳、氮化、感應淬火**——只硬化表面（耐磨）、芯部保持韌（抗衝擊），齒輪典型做法。

## 第 4 章　材料家族總覽

### 4.1 金屬
- **碳鋼 / 合金鋼**：最通用。加 Cr、Ni、Mo 改善淬透性、強度、耐蝕。
- **不鏽鋼**：加 ≥10.5% Cr 形成緻密氧化膜抗蝕；分沃斯田系（304/316，不磁、耐蝕佳）、麻田散系（可硬化、刀具）、肥粒系。
- **鑄鐵**：碳高、可鑄、減震好；灰口鑄鐵（石墨片，脆）、球墨鑄鐵（石墨球，較韌）。
- **鋁合金**：輕（密度約鋼 1/3）、耐蝕、比強度高，航太與輕量化主力；靠析出強化（時效處理）。
- **鈦合金**：強、輕、耐高溫、生物相容（植入物、航太），但貴、難加工。
- **銅合金**：導電導熱、耐蝕（黃銅、青銅）。

### 4.2 高分子（Polymers）
長鏈分子，輕、便宜、易成形、絕緣：
- **熱塑性（thermoplastic）**：加熱可重複熔融成形（PE、PP、PVC、ABS、尼龍、PC），可回收。
- **熱固性（thermoset）**：固化後形成交聯網狀，不可再熔（環氧樹脂、酚醛），耐熱耐溶劑。
- **彈性體（elastomer）**：橡膠，大彈性變形。
- 行為與溫度強相關（**玻璃轉化溫度 $T_g$**），且有**黏彈性（viscoelasticity）**——同時像彈簧又像阻尼。

### 4.3 陶瓷（Ceramics）
離子/共價鍵：硬、耐磨、耐高溫、耐蝕、絕緣，但**脆**（抗拉差、對缺陷敏感）。用於刀具、軸承、引擎零件、生醫。

### 4.4 複合材料（Composites）
**基材（matrix）＋ 強化材（fiber/particle）**結合兩者優點：
- **纖維強化**：碳纖維/玻璃纖維 + 樹脂，沿纖維方向比強度極高，航太、運動器材。
- 性質**異向性（anisotropic）**——方向不同強度差很多，設計要鋪層（layup）。

## 第 5 章　材料劣化

- **腐蝕（corrosion）**：電化學反應（與電機的電化學相通）。原電池效應、孔蝕、應力腐蝕。防護：鍍層、陽極保護（犧牲陽極）、不鏽鋼鈍化。
- **磨耗（wear）**：表面接觸的材料流失，靠表面硬化與潤滑對抗。
- **疲勞與潛變**（前述）。

## 小結

- 一句話貫穿全科：**結構決定性質，製程改變結構**。
- 金屬能塑性變形是因為**差排**；強化＝**阻礙差排**（細晶、合金、加工硬化、析出）。
- 鐵碳相圖 + 熱處理（退火/正常化/淬火/回火）讓同一鋼材性能可大幅調整，是機械選材的核心技藝。
- 五大家族：金屬（韌）、高分子（輕軟絕緣）、陶瓷（硬脆耐溫）、複材（可裁剪的高性能）、半導體（電機那邊）。
- 設計時分清 **剛性 / 強度 / 韌性 / 硬度**，並提防**疲勞、潛變、腐蝕**這些慢性殺手。
