# 材料力學（Mechanics of Materials）

> 名詞對照見 ../中英名詞對照表.md

材料力學（又稱「固體力學」「材料強度學」，英文 Mechanics of Materials / Strength of Materials）研究**可變形固體**在外力作用下，內部會產生多大的「應力」、會「變形」多少，以及到什麼程度會「破壞」。靜力學把物體當成剛體，只問「外力平不平衡」；材料力學接著問「那這些力在材料內部怎麼分配？桿子會不會被拉斷、樑會不會被壓垮、軸會不會被扭裂？」。對機械工程師而言，這是把一個「受力的形狀」變成「安全的零件」的核心學問。

## 0. 本科在本系的定位

- **往上銜接靜力學**：靜力學教你用自由體圖（FBD）算出構件承受的外力與「內力」（軸力 $N$、剪力 $V$、彎矩 $M$、扭矩 $T$）。材料力學接手，把這些內力換算成材料內部每一點的**應力**，再判斷材料受不受得了。
- **往下支撐機械設計**：所有強度計算——軸要多粗、樑要多高、螺栓要幾顆、柱會不會挫曲——都建立在本科之上。[機械設計](../04-機構與機械設計/10-機械設計.md)就是把本科的應力公式配上安全係數與材料數據來定尺寸。
- **核心心法（記住這兩件事，全科都是它們的延伸）**：
  1. **應力 ≈ 內力 / 面積**（$\sigma = N/A$、$\tau = V/A$ 之類），衡量「材料每單位面積扛多少力」，與形狀大小無關，可直接和材料的「強度極限」比較。
  2. **變形 = 應變沿長度的累積（積分）**（$\delta = \int \varepsilon\,dx$），而應變又由應力透過材料的「勁度」決定（虎克定律 $\sigma = E\varepsilon$）。
- 整科的邏輯鏈：**外力 → 內力（靜力學）→ 應力（÷面積）→ 應變（÷模數）→ 變形（積分）→ 比對強度判斷破壞**。

## 1. 應力與應變

### 1.1 應力（Stress）
應力是「內力的密度」，單位 Pa（$=\mathrm{N/m^2}$），工程常用 MPa。把切面上的內力除以面積：
- **正向應力（normal stress）** $\sigma = \dfrac{N}{A}$：力垂直於切面（拉為正、壓為負）。
- **剪應力（shear stress）** $\tau = \dfrac{V}{A}$：力平行於切面（如剪刀剪斷、螺栓被橫向剪）。

直覺：同樣 1000 N，作用在細鐵絲上應力很大（快斷），作用在粗鋼柱上應力很小（沒事）。應力把「力」標準化成可以和材料本身強度比較的量。

### 1.2 應變（Strain）
應變是「變形的密度」，無因次：
- **正向應變** $\varepsilon = \dfrac{\delta}{L}$（伸長量除以原長）。
- **剪應變** $\gamma$：直角被剪歪掉的角度（弧度）。

### 1.3 單軸拉伸試驗與虎克定律
把材料做成標準試棒，慢慢拉，記錄「應力–應變曲線」，這是材料力學最重要的一張圖（此處以文字描述其形狀）：
1. **線彈性段**：應力與應變成正比，卸載後完全回彈。斜率就是**楊氏模數（Young's modulus）** $E$：$\sigma = E\varepsilon$（虎克定律）。$E$ 是材料的「勁度」，鋼約 200 GPa、鋁約 70 GPa。
2. **降伏點（yield point）** $\sigma_y$：超過後開始**塑性變形**（卸載不會完全回彈）。設計上通常要求工作應力遠低於 $\sigma_y$。
3. **極限強度（ultimate strength）** $\sigma_u$：曲線最高點，材料能承受的最大應力。
4. **頸縮與斷裂**：延性材料（如低碳鋼）斷前會「頸縮」明顯變細；脆性材料（鑄鐵、陶瓷、混凝土）幾乎沒有塑性段就突然斷。

### 1.4 蒲松比與剪力虎克定律
- **蒲松比（Poisson's ratio）** $\nu = -\dfrac{\varepsilon_\text{側向}}{\varepsilon_\text{軸向}}$：拉長一根桿，它會變細。金屬約 $\nu\approx0.3$，橡膠接近 0.5（幾乎不可壓縮）。
- 剪力對應的虎克定律：$\tau = G\gamma$，$G$ 為**剪切模數**。三者關係 $G = \dfrac{E}{2(1+\nu)}$。
- **熱應變**：溫度變化 $\Delta T$ 造成 $\varepsilon_T = \alpha\,\Delta T$（$\alpha$ 為熱膨脹係數）。若膨脹被約束無法自由伸長，就會產生**熱應力**。

## 2. 軸向負載（Axial Loading）

### 2.1 變形
一根受軸力的桿，伸長量由應變積分而得。等截面等內力時：
$$\delta = \frac{NL}{AE}$$
其中 $AE$ 稱「軸向勁度」。若沿長度分段（內力、截面或材料不同），就分段相加：$\delta = \sum \dfrac{N_i L_i}{A_i E_i}$；連續變化則積分 $\delta = \int_0^L \dfrac{N(x)}{A(x)E}\,dx$。這正是「變形＝應變積分」心法的最直接體現。

### 2.2 靜不定問題（Statically Indeterminate）
當支撐或構件數目超過靜力平衡方程式可解的數目（例如兩端都固定的桿），光靠 $\sum F=0$ 不夠用。解法要加上**變形相容條件（compatibility）**：例如兩端固定的桿，總伸長必須為 0。把「平衡方程 + 相容方程 + 材料的 $\delta$–$N$ 關係」聯立，即可解出各段內力。這是軟體背景者熟悉的「方程式數＝未知數」思路，只是多了一條由幾何而來的限制式。

## 3. 扭轉（Torsion）

機械的軸（傳動軸、螺絲起子、鑽桿）主要承受扭矩 $T$。

### 3.1 圓軸剪應力
圓軸受扭時，橫截面保持平面，剪應力沿半徑**線性分布**（軸心為 0，表面最大）：
$$\tau = \frac{T\rho}{J}, \qquad \tau_\text{max} = \frac{Tr}{J}$$
其中 $\rho$ 為到軸心距離，$r$ 為外半徑，$J$ 為**極慣性矩（polar moment of inertia）**：
- 實心圓軸：$J = \dfrac{\pi}{32}D^4 = \dfrac{\pi}{2}r^4$。
- 空心圓軸：$J = \dfrac{\pi}{32}(D^4 - d^4)$。

直覺：因為靠外圈材料分擔的力臂大、貢獻最大，所以**空心軸最省料**——把材料挪到外圈，去掉幾乎沒在出力的軸心，重量大減而抗扭幾乎不變。

### 3.2 扭轉角
$$\phi = \frac{TL}{GJ}$$
$GJ$ 稱「扭轉勁度」。完整關係式（圓軸）：$\dfrac{T}{J} = \dfrac{\tau}{r} = \dfrac{G\phi}{L}$。

### 3.3 傳動功率
旋轉軸傳遞的功率 $P = T\omega$（$\omega$ 為角速度，rad/s），或 $P = 2\pi n T$（$n$ 為每秒轉數）。設計傳動軸時，先由「要傳多少功率、轉速多少」反推扭矩 $T = P/\omega$，再用 $\tau_\text{max}=Tr/J$ 確認剪應力在容許範圍內，據此選軸徑。

## 4. 彎曲（Bending）

樑（beam）是承受橫向載重的構件（樓板、橋、車軸、機架）。彎曲是材料力學最重的單元。

### 4.1 剪力圖與彎矩圖（SFD / BMD）
沿樑切開做 FBD，可求得每個位置的**剪力 $V(x)$** 與**彎矩 $M(x)$**，畫成圖即剪力圖、彎矩圖。關鍵微分關係：
$$\frac{dV}{dx} = -w(x), \qquad \frac{dM}{dx} = V(x)$$
其中 $w$ 為分布載重。實用推論：彎矩在「剪力＝0」處取極值；這些極值位置就是樑最危險、最該檢查應力的斷面。

### 4.2 彎曲正應力與彎曲公式
樑彎曲時，一側被拉長、另一側被壓縮，中間有一層長度不變的**中性軸（neutral axis）**（通過截面形心）。正應力沿截面高度**線性分布**：
$$\sigma = \frac{My}{I}$$
其中 $y$ 為到中性軸的距離，$I$ 為截面對中性軸的**慣性矩（second moment of area）**。最大應力在最外緣 $y=c$：$\sigma_\text{max}=Mc/I = M/S$，$S=I/c$ 稱**截面模數**。
- 常見慣性矩：矩形 $I=\dfrac{bh^3}{12}$；圓形 $I=\dfrac{\pi D^4}{64}$。
- 直覺：$I$ 中的 $h^3$ 說明「把材料放遠離中性軸極為有效」——這就是 I 型樑（工字樑）把材料集中在上下翼緣的原因，也是同樣木板「立著放」比「平著放」抗彎強得多的理由。

### 4.3 橫向剪應力
彎曲同時伴隨剪力，截面上的剪應力分布為：
$$\tau = \frac{VQ}{Ib}$$
其中 $Q$ 為所考慮高度以外那塊面積對中性軸的**一次矩**，$b$ 為該處寬度。剪應力在中性軸處最大、上下表面為 0（與彎曲正應力相反）。對細長樑彎曲應力主導，但對短粗樑或木材（沿紋理易剪裂）剪應力不可忽略。

## 5. 樑的撓度（Beam Deflection）

很多時候零件不會壞，但「彎太多」也不能接受（例如機床主軸、導軌的剛性要求）。撓度 $v(x)$ 是樑的變形曲線（彈性曲線）。

### 5.1 彈性曲線方程式
由「彎矩造成曲率」推得：
$$EI\,\frac{d^2 v}{dx^2} = M(x)$$
$EI$ 為**抗彎勁度**。

### 5.2 積分法
把 $M(x)$ 連續積分兩次得 $v(x)$，積分常數由**邊界條件**決定（簡支端撓度為 0、固定端撓度與斜率皆為 0）。例如懸臂樑端點受集中力 $P$，端點撓度 $v_\text{max}=\dfrac{PL^3}{3EI}$；簡支樑中點受 $P$，中點撓度 $\dfrac{PL^3}{48EI}$。注意撓度對長度 $L$ 是三次方關係——加長一倍，撓度暴增八倍。

### 5.3 重疊法（Superposition）
線彈性下變形可疊加：複雜載重 = 多個簡單載重之和，各自查「標準撓度公式表」算出撓度再相加。這是工程上最常用、最省事的算法，類似電路裡的疊加原理。其他方法還有奇異函數法、力矩面積法與能量法（卡氏定理）。

## 6. 應力轉換與莫爾圓（Stress Transformation & Mohr's Circle）

同一點的應力狀態，換個切面角度看，正應力與剪應力會跟著變。應力轉換就是回答「哪個方向最危險」。

### 6.1 平面應力轉換
給定 $\sigma_x,\sigma_y,\tau_{xy}$，轉到與 $x$ 軸夾角 $\theta$ 的面上：
$$\sigma_{x'} = \frac{\sigma_x+\sigma_y}{2} + \frac{\sigma_x-\sigma_y}{2}\cos2\theta + \tau_{xy}\sin2\theta$$
$$\tau_{x'y'} = -\frac{\sigma_x-\sigma_y}{2}\sin2\theta + \tau_{xy}\cos2\theta$$

### 6.2 主應力與最大剪應力
存在一組方向使剪應力為 0、正應力取極值，稱**主應力（principal stresses）**：
$$\sigma_{1,2} = \frac{\sigma_x+\sigma_y}{2} \pm \sqrt{\left(\frac{\sigma_x-\sigma_y}{2}\right)^2 + \tau_{xy}^2}$$
最大面內剪應力 $\tau_\text{max} = \sqrt{\left(\dfrac{\sigma_x-\sigma_y}{2}\right)^2 + \tau_{xy}^2} = \dfrac{\sigma_1-\sigma_2}{2}$。

### 6.3 莫爾圓（Mohr's Circle）
把上面兩式視為參數方程，在 $(\sigma,\tau)$ 平面上其實是一個**圓**：圓心在 $\left(\dfrac{\sigma_x+\sigma_y}{2},0\right)$，半徑 $=\tau_\text{max}$。圓與 $\sigma$ 軸的兩個交點就是主應力，最高點就是最大剪應力。莫爾圓把一堆三角函數變成「轉個圓」的幾何直覺，是本科最經典的工具（應變也有對應的應變莫爾圓）。

## 7. 組合載重（Combined Loadings）

真實零件常同時承受多種載重（例如曲柄軸又彎又扭、壓力管又受內壓又受軸力）。做法：**分別**算出各載重造成的應力（軸向 $N/A$、彎曲 $My/I$、扭轉 $Tr/J$ 等），在線彈性下於同一點**疊加**得到該點的 $\sigma_x,\sigma_y,\tau_{xy}$，再用第 6 章的應力轉換求主應力，最後用第 9 章破壞準則判斷安全。這是把前面所有單元串起來的綜合題型。

## 8. 薄壁壓力容器（Thin-Walled Pressure Vessels）

鍋爐、氣瓶、管線、輪胎都是壓力容器。當壁厚 $t$ 遠小於半徑 $r$（一般 $t \le r/10$ 或 $D/10$）時，可近似壁內應力均勻分布。

對承受內壓 $p$ 的圓柱形容器：
- **環向（hoop / 周向）應力**：$\sigma_\theta = \dfrac{pr}{t}$
- **軸向（縱向）應力**：$\sigma_z = \dfrac{pr}{2t}$

環向應力是軸向的兩倍，所以圓柱壓力容器破裂時通常**沿軸向裂開**（像香腸縱裂），這是設計焊縫走向的重要依據。球形容器各方向應力相同 $\sigma = \dfrac{pr}{2t}$，最省材料，故高壓氣瓶常做成球形或膠囊形。

## 9. 挫曲與破壞準則

### 9.1 挫曲（Buckling）：Euler 公式
細長的受壓桿件（柱）不一定是被「壓碎」，而可能在應力遠低於降伏前就**突然側向彎曲失穩**，這叫挫曲。臨界載重由 Euler 公式給出：
$$P_\text{cr} = \frac{\pi^2 EI}{(KL)^2}$$
其中 $I$ 取**最小**慣性矩（從最弱的方向挫曲），$K$ 為**有效長度係數**，由兩端支撐條件決定：
- 兩端鉸接 $K=1.0$；一端固定一端自由（懸臂）$K=2.0$；兩端固定 $K=0.5$；一端固定一端鉸接 $K\approx0.7$。

重點直覺：挫曲是**幾何穩定性**問題而非強度問題——$P_\text{cr}$ 只跟 $E$（勁度）與 $I$（形狀）有關，與材料強度 $\sigma_y$ 無關。又因分母含 $L^2$，桿越長越容易挫曲。以**細長比（slenderness ratio）** $KL/k$（$k=\sqrt{I/A}$ 為迴轉半徑）區分：細長柱用 Euler，短粗柱則改以降伏控制（用 Johnson 公式等過渡）。

### 9.2 破壞準則（延性材料的降伏判斷）
單軸拉伸只要比 $\sigma_y$ 即可，但組合應力（多軸）下「何時降伏」需要破壞準則。兩個最常用的：
- **最大剪應力理論（Tresca）**：當最大剪應力達到單軸降伏時的剪應力 $\sigma_y/2$ 即降伏。判據 $\sigma_1 - \sigma_3 \ge \sigma_y$。較保守、計算簡單。
- **畸變能理論（von Mises / distortion energy）**：當「使形狀扭曲的那部分能量」達到單軸降伏的對應值時降伏。等效（von Mises）應力
$$\sigma_v = \sqrt{\frac{(\sigma_1-\sigma_2)^2+(\sigma_2-\sigma_3)^2+(\sigma_3-\sigma_1)^2}{2}} \ge \sigma_y$$
von Mises 與延性金屬實驗最吻合，是工程與有限元素分析的主流準則；Tresca 的六邊形屈服面內接於 von Mises 的橢圓，所以 Tresca 較保守、偏安全。脆性材料則另用最大正應力理論（Rankine）或莫爾破壞準則。
- **安全係數（factor of safety）** $n = \dfrac{\text{材料強度}}{\text{工作應力}}$，串接到[機械設計](../04-機構與機械設計/10-機械設計.md)。

## 與其他科目的關聯

- [靜力學](../01-工程力學基礎/02-靜力學.md)：提供內力分析（軸力、剪力、彎矩、扭矩）與截面性質（形心、慣性矩），是本科的直接前置。
- [動力學](../01-工程力學基礎/03-動力學.md)：運動構件的慣性力可化為等效載重，再用本科算應力（達朗貝爾原理）。
- [工程材料](05-工程材料.md)：提供 $E$、$\sigma_y$、$\sigma_u$、$\nu$ 等材料數據與背後的微觀機制（疲勞、潛變、脆韌轉變），補足本科只談宏觀的部分。
- [機械設計](../04-機構與機械設計/10-機械設計.md)：把本科應力公式配上安全係數、疲勞與失效理論來決定零件尺寸，是本科最直接的下游應用。
- 機械振動：樑撓度與 $EI$ 勁度是計算自然頻率的基礎。

## 碩士層級延伸（簡短）

- **彈性力學（Theory of Elasticity）**：不再靠「截面假設」，直接用偏微分方程（平衡 + 相容 + 本構）解三維應力場，材料力學的各公式都是其特例。
- **塑性力學（Plasticity）**：降伏之後的行為，含流動法則、硬化模型、極限分析。
- **斷裂力學（Fracture Mechanics）**：含裂縫構件的破壞，引入應力強度因子 $K$、能量釋放率 $G$、$J$ 積分。
- **疲勞（Fatigue）**：反覆載重下遠低於 $\sigma_u$ 即破壞，S–N 曲線與損傷累積。
- **有限元素法（FEM）**：把連體切成小元素，數值求解複雜幾何的應力場，是現代 CAE 的主力（軟體背景者可從變分／能量法切入，較易上手）。
- **複合材料力學、板殼理論、能量法（卡氏定理、虛功）、實驗應力分析（應變規、光彈）**。
