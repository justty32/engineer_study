## 為什麼這兩塊放一起

ODE 描述「隨時間演化的線性系統」（電路、機械、控制），但解 ODE 靠手算很痛苦；**拉氏轉換**把 ODE 變成代數方程，是電機人的標準解法。所以「ODE + 拉氏」是一組：問題與解法。

## 第 1 章　常微分方程（ODE）

### 1.1 定義與分類
**ODE**：含未知函數 $y(t)$ 與其導數 $y',y'',\dots$ 的方程式，只對單一變數（這裡用 $t$）求導。

分類：
- **階數（order）**：方程中最高的導數階。RC 電路是一階，RLC 是二階。
- **線性 / 非線性**：$y$ 與其導數只以一次方出現、不互相相乘 → 線性。本科主要處理線性。
- **常係數 / 變係數**：係數是否隨 $t$ 變。電機常見的是常係數。
- **齊次（homogeneous）/ 非齊次（non-homogeneous）**：右側為 0 / 非 0。對應自然響應 / 強迫響應。

### 1.2 一階線性 ODE
標準型：
$$\frac{dy}{dt}+p(t)y=q(t)$$

**解法（積分因子法）**：乘上 $\mu(t)=e^{\int p\,dt}$ 使左邊變成 $\dfrac{d}{dt}(\mu y)$，再積分。

**電機原型例**：RC 電路充電
$$RC\frac{dv}{dt}+v=v_s$$
解：$v(t)=v_s+(v_0-v_s)e^{-t/RC}$。直覺：以時間常數 $\tau=RC$ 指數逼近穩態 $v_s$。

### 1.3 二階常係數線性 ODE
$$ay''+by'+cy=f(t)$$

**齊次解（自然響應）**：解特徵方程 $a\lambda^2+b\lambda+c=0$ 得 $\lambda_1,\lambda_2$，三種情況：
- 兩相異實根：$y_h=C_1 e^{\lambda_1 t}+C_2 e^{\lambda_2 t}$ — **過阻尼（overdamped）**
- 重根：$y_h=(C_1+C_2 t)e^{\lambda t}$ — **臨界阻尼（critically damped）**
- 共軛複根 $\alpha\pm j\beta$：$y_h=e^{\alpha t}(C_1\cos\beta t+C_2\sin\beta t)$ — **欠阻尼（underdamped）**

**完整解** = $y_h$ + $y_p$（特解，由 $f(t)$ 形態猜測或變數變換求得）。

**電機原型例**：RLC 串聯電路
$$L\frac{d^2 i}{dt^2}+R\frac{di}{dt}+\frac{1}{C}i=\frac{dv_s}{dt}$$
跟機械的彈簧—質量—阻尼系統完全同形——這就是「電機—機械類比」。

### 1.4 系統的「自然頻率」與「阻尼比」
把二階方程寫成標準型：
$$y''+2\zeta\omega_n y'+\omega_n^2 y = \omega_n^2 u(t)$$
- $\omega_n$：**自然頻率（natural frequency）**——若無阻尼會以這個頻率振盪。
- $\zeta$：**阻尼比（damping ratio）**——決定衰減快慢與是否振盪。

$\zeta>1$ 過阻尼、$\zeta=1$ 臨界、$0<\zeta<1$ 欠阻尼（會振盪）、$\zeta=0$ 純振盪。
控制系統設計時調整 $\zeta,\omega_n$ 就是在塑造系統響應。

### 1.5 ODE 系統與狀態空間（State Space）
高階 ODE 可改寫成一階向量 ODE：
$$\dot{\mathbf{x}}(t)=A\mathbf{x}(t)+B\mathbf{u}(t),\quad \mathbf{y}=C\mathbf{x}+D\mathbf{u}$$

解：$\mathbf{x}(t)=e^{At}\mathbf{x}(0)+\int_0^t e^{A(t-\tau)}B\mathbf{u}(\tau)\,d\tau$。

這把線代與 ODE 縫在一起：$A$ 的**特徵值**決定系統穩定性（負實部 → 穩定）。是現代控制論的核心表達。

## 第 2 章　拉氏轉換（Laplace Transform）

### 2.1 定義
$$F(s)=\mathcal{L}\{f(t)\}=\int_0^\infty f(t)e^{-st}\,dt,\quad s=\sigma+j\omega\in\mathbb{C}$$

直覺：把 $f(t)$ 拆成「指數衰減（或成長）+ 振盪」的所有可能組合 $e^{st}$ 的「成分大小」$F(s)$。是傅立葉的廣義版（含實部 $\sigma$）。

**收斂域（ROC）**：使積分收斂的 $s$ 範圍，一般是某條垂直線右側 $\mathrm{Re}(s)>\sigma_0$。

### 2.2 核心性質（電機天天用）
| 時間域 | $s$ 域 |
| --- | --- |
| $f'(t)$ | $sF(s)-f(0)$ |
| $f''(t)$ | $s^2F(s)-sf(0)-f'(0)$ |
| $\int_0^t f(\tau)\,d\tau$ | $F(s)/s$ |
| $f(t-a)u(t-a)$（時移） | $e^{-as}F(s)$ |
| $e^{at}f(t)$（頻移） | $F(s-a)$ |
| $f(t)*g(t)$（卷積） | $F(s)G(s)$ |

**最關鍵的就是「微分變乘 $s$」**——微分方程瞬間變代數方程。

### 2.3 常見轉換對
| $f(t)$, $t\ge 0$ | $F(s)$ |
| --- | --- |
| $1$（單位步階 $u(t)$） | $1/s$ |
| $t$ | $1/s^2$ |
| $e^{-at}$ | $1/(s+a)$ |
| $\sin\omega t$ | $\omega/(s^2+\omega^2)$ |
| $\cos\omega t$ | $s/(s^2+\omega^2)$ |
| $\delta(t)$（單位脈衝） | $1$ |

### 2.4 用拉氏解 ODE 的標準流程
1. 對 ODE 兩邊取拉氏，把 $y(t)$ 變 $Y(s)$、$y'$ 變 $sY-y(0)$ 等。
2. 解出 $Y(s)$（代數運算）。
3. **部分分式展開（partial fraction）** 把 $Y(s)$ 拆成查表能對得到的形式。
4. 反拉氏得到 $y(t)$。

例：$y''+3y'+2y=0,\ y(0)=1,\ y'(0)=0$
取拉氏：$(s^2Y-s-0)+3(sY-1)+2Y=0\Rightarrow Y(s)=\dfrac{s+3}{s^2+3s+2}=\dfrac{2}{s+1}-\dfrac{1}{s+2}$
反拉氏：$y(t)=2e^{-t}-e^{-2t}$。

### 2.5 轉移函數與系統穩定性
線性時不變系統的**轉移函數**：
$$H(s)=\frac{Y(s)}{U(s)}$$
（零初始條件下，輸出與輸入的拉氏比）

- **極點（pole）**：使 $H(s)$ 分母為零的 $s$。極點就是齊次解的指數速率。
- **零點（zero）**：使分子為零的 $s$。
- **穩定性條件**：所有極點落在**左半 $s$ 平面**（$\mathrm{Re}(s)<0$）→ 系統穩定。
- 極點越靠左，衰減越快；極點落在虛軸上 → 邊界穩定（純振盪）；落在右半平面 → 發散。

這個觀念是後續**控制系統**、**信號與系統**的軸心。

### 2.6 終值與初值定理（Final/Initial Value Theorem）
不用反拉氏就能查穩態與初始：
- 初值：$f(0^+)=\lim_{s\to\infty} sF(s)$
- 終值：$f(\infty)=\lim_{s\to 0} sF(s)$（前提：極點全在左半平面）

### 2.7 雙邊拉氏與單邊拉氏
工程上多用**單邊（unilateral）拉氏**（從 $0$ 積到 $\infty$），自然處理初始條件。**雙邊拉氏**（從 $-\infty$ 積到 $\infty$）與 $z$ 轉換、傅立葉轉換有更乾淨的對應關係，理論上較通用。

## 與電機/通訊的連結（小結）
- **電路**：寫出微分方程 → 拉氏 → 求 $V(s),I(s)$。電容變 $1/(sC)$、電感變 $sL$，整個電路在 $s$ 域就是普通的歐姆定律。
- **控制**：方塊圖 $\to$ 轉移函數 $\to$ 看極點分布做穩定性 / 暫態 / 穩態設計。
- **濾波器**：類比濾波器設計直接在 $s$ 域進行（Butterworth、Chebyshev）。
