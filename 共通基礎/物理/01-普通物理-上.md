## 為什麼電機要學力學

直接用：機械振動 ↔ RLC 電路（同形 ODE）；陀螺、加速度計、MEMS、馬達運動方程都是力學。
間接用：「力—電類比」是工程師抓直覺的核心方法之一。

## 第 1 章　運動學（Kinematics）

不問為什麼，只描述「怎麼動」。

- 位置 $\mathbf{r}(t)$，速度 $\mathbf{v}=\dot{\mathbf{r}}$，加速度 $\mathbf{a}=\ddot{\mathbf{r}}$。
- 等加速度公式：$v=v_0+at$、$x=x_0+v_0 t+\tfrac{1}{2}at^2$、$v^2=v_0^2+2a(x-x_0)$。
- **拋體運動**：水平等速 + 垂直等加速度（重力 $g\approx 9.8\,\mathrm{m/s^2}$）。
- **圓周運動**：向心加速度 $a_c=v^2/r=\omega^2 r$；角速度 $\omega$、角加速度 $\alpha$。

## 第 2 章　牛頓三大運動定律（Dynamics）

**第一定律（慣性）**：無外力則速度不變。
**第二定律**：$\mathbf{F}=m\mathbf{a}$（更一般 $\mathbf{F}=d\mathbf{p}/dt$，$\mathbf{p}=m\mathbf{v}$ 為動量）。
**第三定律**：作用力與反作用力等大反向。

工程含義：把所有作用在物體上的力畫成**自由體圖（free body diagram）**，列出 $\mathbf{F}=m\mathbf{a}$，就是力學問題的標準解法。

### 常見的力
- **重力** $mg$。
- **正向力（normal force）**：接觸面垂直分量。
- **摩擦力**：靜摩擦 $f_s\le\mu_s N$、動摩擦 $f_k=\mu_k N$。
- **彈簧力（Hooke）**：$F=-kx$。
- **張力（張力 / 線張力）**。

## 第 3 章　功、能量、能量守恆

- **功** $W=\int \mathbf{F}\cdot d\mathbf{r}$（力沿位移的累積）。
- **動能** $K=\tfrac{1}{2}mv^2$。
- **功—動能定理**：$W_\text{總}=\Delta K$。
- **位能（potential energy）**：保守力對應位能。重力 $U=mgh$、彈簧 $U=\tfrac{1}{2}kx^2$、靜電 $U=qV$。
- **機械能守恆**：只有保守力作用時 $K+U=$ 常數。
- **功率** $P=\dfrac{dW}{dt}=\mathbf{F}\cdot\mathbf{v}$。

電機類比：$P=VI$、能量 $\tfrac{1}{2}Cv^2$、$\tfrac{1}{2}Li^2$——和 $\tfrac{1}{2}kx^2$、$\tfrac{1}{2}mv^2$ 完全同形。

## 第 4 章　動量、衝量、碰撞

- **動量** $\mathbf{p}=m\mathbf{v}$；**衝量** $\mathbf{J}=\int\mathbf{F}\,dt=\Delta\mathbf{p}$。
- **動量守恆**：系統內無外力時總動量守恆。
- **碰撞**：
  - **彈性碰撞**：動量與動能都守恆。
  - **非彈性**：只有動量守恆，動能部分轉熱/形變。
  - **完全非彈性**：碰後黏在一起。

## 第 5 章　轉動力學（Rotational Dynamics）

平移與轉動的對照：

| 平移 | 轉動 |
| --- | --- |
| 質量 $m$ | 轉動慣量 $I$ |
| 力 $F$ | 力矩 $\tau$ |
| 位移 $x$ | 角位移 $\theta$ |
| 速度 $v$ | 角速度 $\omega$ |
| 動量 $p=mv$ | 角動量 $L=I\omega$ |
| 牛頓 $F=ma$ | $\tau=I\alpha$ |
| 動能 $\tfrac{1}{2}mv^2$ | $\tfrac{1}{2}I\omega^2$ |

- **轉動慣量** $I=\int r^2\,dm$，描述「物體有多難轉」。
- **角動量守恆**：陀螺、行星運動、花式溜冰旋轉收手變快。

## 第 6 章　簡諧運動與振盪（Simple Harmonic Motion, SHM）

電機最該抓住的一章——因為跟 RLC 電路完全同形。

**彈簧—質量系統**：$m\ddot{x}+kx=0$
解：$x(t)=A\cos(\omega_0 t+\phi)$，$\omega_0=\sqrt{k/m}$。

**加阻尼**：$m\ddot{x}+c\dot{x}+kx=0$
- 過阻尼、臨界、欠阻尼——和 RLC 電路同樣三種響應。

**加驅動**：$m\ddot{x}+c\dot{x}+kx=F_0\cos\omega t$
- 出現**共振（resonance）**：當 $\omega$ 接近 $\omega_0$，振幅放大。

**力—電類比**：
| 機械 | 電氣 |
| --- | --- |
| 質量 $m$ | 電感 $L$ |
| 阻尼 $c$ | 電阻 $R$ |
| 彈性 $k$ | $1/C$ |
| 位移 $x$ | 電荷 $q$ |
| 速度 $\dot x$ | 電流 $i$ |
| 力 $F$ | 電壓 $v$ |

把這張表記住，往後看 RLC 就有物理直覺。

## 第 7 章　流體力學（簡介）

- **靜壓** $p=p_0+\rho g h$。
- **連續方程**：$\rho A v$ 守恆（不可壓縮就是 $Av$ 守恆）。
- **白努利方程**：$p+\tfrac{1}{2}\rho v^2+\rho gh=$ 常數。

電機交集少，了解原理即可。MEMS 感測器、冷卻設計會用到。

## 第 8 章　熱學基礎

- **溫度、熱量、比熱**：$Q=mc\Delta T$。
- **熱力學三定律**：能量守恆、熵增、絕對零度不可達。
- **理想氣體** $PV=nRT$。

電機關聯：晶片散熱、半導體中載子的熱激發、雜訊（熱雜訊功率 $kTB$）。
