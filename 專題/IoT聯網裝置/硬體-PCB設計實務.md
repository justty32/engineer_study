# 硬體：PCB 設計實務

> 重點放在「實際畫一塊含通訊模組的板子」會踩到的東西。RF 與天線相關理論連到 [A3 傳輸線與波導](../../電機/08-天線特化/A3-傳輸線與波導.md)、[A1 天線理論](../../電機/08-天線特化/A1-天線理論.md)。

## 第 1 章　設計流程

需求 → 選料 → **原理圖（schematic）** → **佈局（placement）** → **走線（routing）** → 敷銅 / DRC 檢查 → 製造輸出（Gerber / ODB++、BOM、Pick&Place 座標） → 打樣 + 焊接 → 上電驗證 → 改版。

每一步都比「會用軟體」重要的是**判斷**：阻抗、回流、去耦、保護有沒有顧到。

## 第 2 章　原理圖（Schematic）

- 分功能區塊分頁（電源 / MCU / 模組 / 感測器 / 接口），網路標籤（net label）連接。
- 去耦電容**畫在對應 IC 旁**，提醒佈局時就近擺。
- 每個 IC 標好封裝、料號；關鍵腳位（RESET、BOOT、EN）標清楚。
- 留 **測試點 / SWD 排針 / 量測點**。
- 跑 **ERC（電氣規則檢查）**：抓漏接、電源短路、未連腳。

## 第 3 章　元件與封裝

- 被動件尺寸：0402 / 0603（手焊上限約 0402）、0805。
- IC 封裝：QFN/QFP（好焊）、BGA（密度高、需 X-ray 檢查、難手焊）。
- **手焊打樣 vs 機器量產**：打樣階段盡量避免 BGA、超細間距。
- 選料看：可取得性（避免缺貨/停產）、溫度等級、認證（模組選**已預認證**者，見 [量產與認證](量產-測試與法規認證.md)）。

## 第 4 章　疊層與地（最影響成敗）

- **2 層**：便宜，但難有完整地平面，高速 / RF 不友善。
- **4 層**典型：`訊號 — 地 — 電源 — 訊號`。第二層**完整地平面**是關鍵：給每條訊號一條緊鄰、連續的**回流路徑（return path）**。
- 鐵律：**訊號的回流電流走正下方的地**。地平面被切割、或走線跨過平面裂縫 → 回流被迫繞遠 → 迴路面積變大 → EMI 與串擾爆增。
- 多數情況**保持地平面完整**，別亂分割數位/類比地（早年觀念已過時，誤割反而更糟）。需要隔離時用佈局分區 + **縫合過孔（via stitching）** 連接。

## 第 5 章　佈局（Placement）

順序：先放**位置受限的**（接頭、通訊模組、天線），再放主晶片，最後鋪小件。
- **功能分區**：類比 / 數位 / RF / 電源各自成區，訊號自然流動、互不干擾。
- 去耦電容、晶振**緊貼**對應 IC；晶振走線短、周圍包地、下方不走線。
- 高速 / 高電流路徑短而直；發熱件留散熱與間距。

## 第 6 章　訊號完整性（SI）

- 何時要在意：**邊緣速度快**（上升時間短）、走線長、頻率高。決定因素是**上升時間**而非時脈頻率。
- **受控阻抗走線**：高速差分（USB、Ethernet、MIPI）走 90/100 Ω 差分；單端高速走 50 Ω。由線寬、介電層厚、$\varepsilon_r$ 決定，請板廠給疊層阻抗表。
- 反射：阻抗不連續會反射 → 端接匹配、避免殘段（stub）。
- 串擾：相鄰平行走線耦合 → 拉開間距（3W 原則）、中間隔地。
- 差分對：等長、緊耦合、對內長度匹配。

## 第 7 章　RF 走線與天線（含模組最關鍵）

通訊模組到天線這一段是**最容易做爛**的地方：

- **50 Ω 受控阻抗**：用 microstrip 或 **CPWG（共面波導加地）**；線寬依疊層算，板廠確認。
- **越短越好**、避免過孔與直角（用圓弧或 45°），參考地必須**連續完整**。
- **天線匹配網路**：模組天線埠到天線間留 **π 型匹配（3 元件）** 位置，量測後調到 50 Ω。
- **天線淨空區（keep-out / clearance）**：天線正下方與周圍**不鋪地、不走線、不放金屬/電池**，否則失諧、效率掉、距離變短。
- **務必照模組廠商的 layout guideline**：間距、淨空、銅皮、接地過孔數量都有規定，照做能省掉大量 RF 調試。
- 天線選擇：
  - **PCB 走線天線**：最便宜，需調諧、佔板面、受周邊影響大。
  - **晶片天線**：小，需嚴格淨空與匹配。
  - **外接天線（IPEX/u.FL → SMA）**：最穩、最好過認證，成本與機構代價高。

## 第 8 章　EMC / EMI

從源頭抑制最有效：
- 降 **slew rate**（夠用就好的驅動強度）、必要時展頻時脈（SSC）。
- 縮小高頻電流**迴路面積**（去耦、完整地）。
- 對外線加共模扼流、濾波（π/磁珠）。
- 必要時加**屏蔽罐（shielding can）** 罩住吵雜/敏感區。
- 目的：通過輻射/傳導發射測試（見 [量產與認證](量產-測試與法規認證.md)）。

## 第 9 章　ESD / 突波保護

- 對外接口（USB、按鍵、外接線、電源）一進板就放 **TVS / ESD 二極體**，且**靠近接口、先過保護再進 IC**。
- 走線先到保護元件、再到晶片，別讓 ESD 路徑經過敏感腳。
- 機構接地、外殼洩放路徑一併考量。

## 第 10 章　製造性（DFM）與測試

- DFM：遵守板廠最小線寬/間距、過孔孔徑/環寬、阻焊開窗、鋼網（stencil）設計、極性標記、IPC 等級。
- **回流焊溫度曲線（reflow profile）**：依錫膏與元件耐溫設定，避免冷焊/立碑。
- 可測試性：留測試點、SWD/JTAG、飛針可達；產線用 **ICT / FCT**（見 [量產與認證](量產-測試與法規認證.md)）。
- 輸出檔：Gerber/ODB++、鑽孔、BOM、Pick&Place、疊層與阻抗要求一併給板廠/工廠。
