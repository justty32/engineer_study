# 大氣科學 中英名詞對照表

本表收錄大氣科學各學科之核心專有名詞，依學科分類。
通用數學與物理名詞請參閱 [共通基礎/](../共通基礎/)。

## 01-大氣熱力學與輻射
| 中文名稱 | 英文名稱 | 備註 |
| :--- | :--- | :--- |
| 對流層 | Troposphere | 大氣最底層，天氣現象發生處 |
| 平流層 | Stratosphere | 含有臭氧層，氣溫隨高度遞增 |
| 虛溫 | Virtual Temperature | 考慮水氣對空氣密度影響的修正溫度 |
| 靜力平衡 | Hydrostatic Equilibrium | 氣壓梯度力與重力的垂直平衡 |
| 位溫 | Potential Temperature ($\theta$) | 氣塊絕熱運動至 1000 hPa 時的溫度 |
| 絕熱過程 | Adiabatic Process | 系統與外界無熱量交換的過程 |
| 輻射傳導 | Radiative Transfer | 能量以電磁波形式傳播 |
| 史蒂芬－波茲曼定律 | Stefan-Boltzmann Law | $E = \sigma T^4$ |
| 維恩位移定律 | Wien's Displacement Law | $\lambda_{\max} \propto 1/T$ |
| 反照率 | Albedo | 行星反射太陽輻射的比例（地球約 0.3） |
| 溫室效應 | Greenhouse Effect | 大氣吸收地表長波輻射產生的增溫 |
